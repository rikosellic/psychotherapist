## 第六章 通用技能：介绍与行为分析

**章节主线**：本章是 DBT 技能训练第二部分教学笔记的开篇，分为两大部分——（1）介绍技能训练（目标、准则、假设、生物社会理论），（2）行为分析方法（链锁分析、遗漏环节分析）。通用技能贯穿后续四大模块，是所有技能选择和泛化的基础。

### 章前导言：本章结构与使用说明

- **本章定位**：新团体第一次课程进行介绍，之后每个新模块开始前简短复习；有新成员加入时由老成员协助介绍。
- **教材标记**：教学笔记中 √ = 常用必讲教材；讲义中 ★ = 标准核心讲义，不可跳过。时间紧张时跳过未标记内容。
- **介绍的核心任务**：技能训练师"营销"技能——将技能与成员个人目标联结，激发学习动机。内容可根据人群、场合调整。
- **日记卡**：下半部分（技能练习记录）在技能训练课上讨论，上半部分（问题行为记录）仅在个体治疗中讨论。强调练习所有已教技能，而非仅最新技能。
- **生物社会理论**：对 BPD 和广泛性情绪失调者特别重要；若成员已讨论过，团体中可简略。治疗其他人群需调整理论内容。
- **行为分析**：链锁分析是 DBT 广泛使用的核心技能（个体治疗、住院、DBT 团队均用）；遗漏环节分析最初为多重家庭团体中快速评估未完成作业的原因而发展。
- **链锁分析教学时机**：标准 DBT 中由个体治疗师教授；若成员无个体治疗且时间允许，可纳入技能训练课程。

## 一、技能训练的目标（通用讲义 1-1a）

- **讲义 1**（目标）：帮助成员找出想减少/增加的行为，激发练习热情。可根据群体修改目标清单。
- **讲义 1a**（自选·解决任意问题的选项）：针对"是这样没错，可是……"态度的成员。
- **练习单 1**（自选·利弊分析）：成员被执念或冷漠困住时使用；带领者在白板画 2×2 表格示范，强调四个格子都需填写。

### 四种回应痛苦的选项

1. **解决问题**：改变/离开情境 → 行中庸之道、问题解决技能
2. **改变感受**：调节情绪反应，寻找正面意义 → 情绪调节技能、正念技能
3. **容忍问题**：无法解决也无法变好时，全然接纳 → 痛苦忍受技能、正念技能
4. **留在痛苦中（或让事情更糟）**：不使用技能

### 要增加的技能速览

- **正念**（Mindfulness）：活在当下，减少评判；补充：灵性观点、慈爱、行中庸之道
- **人际效能**（Interpersonal Effectiveness）：获得所需、拒绝请求、维持自尊与关系；补充：建立/结束关系
- **情绪调节**（Emotion Regulation）：了解命名情绪、改变反应、降低易感性、管理极端情绪
- **痛苦忍受**（Distress Tolerance）：危机生存（容忍痛苦而不让事情更糟）、接纳现实；补充：戒瘾技能
- **行为分析**（补充）：链锁分析 + 遗漏环节分析

## 二、技能训练介绍与准则（通用讲义 2-4）

- **讲义 2**（概论）：可跳过，仅说明将涵盖准则、假设、生物社会理论三项。
- **讲义 3**（准则）：区分"准则"（需遵守的行为标准）vs"假设"（不能证明但同意遵守的信念）。开放式团体每次新成员加入时重述。

### 技能训练准则

1. **退出规则**：只有连续缺席四次才算正式退出，合同期内不可重返（缺席三次后出席一次又缺席三次，同样犯规）。可请假：重病、重大事件、不可避免行程；不可请假：没心情、小病、社交、害怕。合同期满后可与培训师和团体协商重新加入。
2. **彼此支持**：保密（人际问题当面处理；例外：告知个体治疗师、防自杀）、按时出席、课下练习、认可而不评判、给予有益反馈、求助时接受帮助（不允许"我要自杀"后拒绝帮助）。
3. **提前告知迟到/缺席**：让带领者不必等待；增加迟到成本；避免团体不必要担心。
4. **不诱使问题行为**：a) 不酗酒/嗑药后出席；b) 若已使用，出席时保持清醒（技能需在最有挑战的情境中练习）；c) 不讨论自伤/物质使用/暴食等具传染性的行为。
5. **不建立私密关系**：可在课外发展友谊，但人际问题必须能在课上讨论。性伴侣分到不同团体；朋友/家人团体例外。
6. **额外的准则**：带领者可讨论并纳入未列在通用讲义 3 中的其他准则，确认每位成员将额外准则写在讲义上。
7. **进阶团体**：在进阶团体中，缺席四次的退出规则可被修改——需讨论并明确"被允许的"缺席（生病、家庭紧急状况、度假、婚礼、葬礼）与"不被允许的"缺席（疲劳、心情不好、住进精神病房等）的具体标准。

### 七大假设

1. 每个人都正在尽其所能做到最好。
2. 每个人都想改进（追求快乐是共通点）。
3. \*每个人都需要做得更好、更努力、更有动力改变（\*：进展稳定时不需要）。
4. \*问题不一定是自己造成的，但仍需自己解决（\*：孩子/身心障碍者需照顾者帮助）。
5. 新行为必须在所有相关情境中学习（需在所有需要的情境中练习）。
6. 所有行为（行动、想法、情绪）都有原因，即使我们不知道。
7. 与其评判指责，不如找出并改变行为的原因。（带领者笔记：文化鼓励评判，不需评判自己的评判，放下评判才能有效改变。）

### 学习技能的承诺

- 获得承诺多多益善：学习技能需完成家庭作业，花费时间且有时困难。
- 讨论重点：是否愿意继续出席和练习；交通/准时/不早退方面的困难；对团体的恐惧和担忧（如不敢分享作业、不敢提问）。
- 即使已在单独会谈中取得承诺，团体中再次讨论仍有助于强化。

### 日记卡

- **结构**：上半记录欲减少的问题行为（仅个体治疗中讨论，0-7 评量）；下半列出 DBT 核心技能，每日圈选练习情况。
- **技能训练课上只讨论下半部分**（技能使用），朋友/家人课程及青少年多重家庭课程也只提供下半部分。
- **强调练习所有已教技能**，而非仅最新学的——防止技能退步。
- **保密**：建议用绰号或缩写代替真实姓名。

## 三、情绪失调的生物社会理论（通用讲义 5）

- **适用范围**：BPD 及广泛性情绪失调者尤其重要；治疗其他人群需调整。与无效育儿父母工作时保持敏感、同情，但不淡化理论。
- **核心论点**：情绪失调 = 生理脆弱性 × 不被认可环境（Invalidating Environment）的交互作用。

### A. 生理层面：情绪易感性与冲动性

情绪易感者特征：对刺激更敏感、更常体验情绪、情绪无端来袭、反应强烈（像被砖头砸）、持续很久。冲动者特征：难抑制行为、易惹麻烦、行为莫名其妙、被情绪干扰无法组织目标行为。

生理因素：遗传、子宫内因素、脑部受伤/疾病。

### B. 社会层面：不被认可的环境

特征：不容忍情绪表达（尤其无法被外部事件支持的内在体验）；间歇强化极端情绪表达；传达"你的情绪是错的/怪异的"；要求在不被支持下面对情绪；不回应情绪表达的需求；混淆自己与他人情绪。

无效育儿成因：没有完美的父母；父母本就在无效环境中长大；完美主义（过度强调压抑情绪）；父母有身心障碍；被压力逼垮（经济/疾病/单亲等）。

### C. 关键机制：沟通的升级

情绪的功能是沟通。不被认可时发讯者加强表达 → 收讯者更强不认可 → 循环升级至一方崩溃。

**教学示例"着火了！"**：
- 发讯者："着火了！" → 收讯者："你反应过度了！"
- 升级后两结果：① "好吧，没着火"（被压制）；② "救命！！！"→ 终于获得回应。
- **启示**：不被认可中学会夸大表达才被听见；但环境也学会了忽略。此互动观点与指责不兼容。

## 五、问题行为的链锁分析（通用讲义 7-7a）

- **讲义 7**：分析步骤与说明。
- **讲义 7a**：分解步骤（实际操作指南）。
- **练习单 2**：链锁分析练习单；**练习单 2a**：完整范例。
- 是否纳入技能训练可自行选择；无个体治疗且时间允许时建议纳入。

### 什么是链锁分析

任何行为由一系列环节串联而成。链锁分析通过"在此之前/之后发生了什么？"逐步拆解已融合的行为。目的：找出问题所在、成因、功能、解决障碍和可用资源。

### 八步结构（1-5 了解问题，6-8 改变行为）

| 步骤 | 内容 |
|------|------|
| 1. 明确问题行为 | 必须具体：做了什么/说了什么/没做什么，含时间地点对象强度持续时间。不说"我崩溃了"。 |
| 2. 诱发事件 | 行为链前直接触发的事件：外部（对方一句话、被拒）或内部（记忆、想法、身体感觉）。要具体。 |
| 3. 易感因素 | 使个体更易被触发：睡眠不足、饥饿、生病、物质使用、压力积累、创伤激活、孤独、疲惫、激素变化、近期未练习技能。不是借口，是预防线索。 |
| 4. 行为链环节 | 一环扣一环：想法/情绪/身体感觉/冲动/记忆/解释/行为/他人反应。要细，找可插入技能的节点。判定关键环节：少了它，问题行为概率是否降低？ |
| 5. 后果 | 短期（情绪缓解、获关注、逃离任务）+ 长期（关系破坏、羞耻、治疗受阻）。解释为何行为重复。 |
| 6. 替代技能 | 不同环节对应不同技能：易感因素→PLEASE/睡眠/运动；解释错误→核对事实；情绪过强→TIP/痛苦忍受；行动冲动→STOP/相反行为；人际触发→DEARMAN/GIVE/FAST；评判沉溺→正念 |
| 7. 预防计划 | 针对易感因素和高风险情境提前安排，需可执行，不能只是"下次不要这样"。 |
| 8. 修复计划 | 修复对环境和自身的损害：道歉/补偿/清理后果/重新承诺。**关键提醒**：先找出真正被伤害的是什么——用鲜花糖果弥补背叛是不够的，需要真诚道歉、收回言论、重建信任。 |

### 教学示例：对伴侣大吼

- **问题行为**：对男友大吼"你从来不管我的感受！"然后冲出甩门。
- **诱发事件**：下班回家发现男友在沙发上睡觉。
- **易感因素**：他昨晚深夜才回、答应今晚出门、期待约会。
- **行为链**：看到他在睡 → "他又在睡觉，今晚不能出门了" → "他根本就不爱我" → 火冒三丈 → 想报复 → 大吼甩门。
- **替代技能节点**："他不爱我"→ 核对事实（他只是想小睡以便共度时光）；"火冒三丈"→ STOP/TIP；"想报复"→ 相反行为（温和询问）。
- **修复计划**：以相信他爱我的方式对待；道歉并补偿那一晚；计划他喜欢的共度之夜。

### 带领者提示

- 成员初期很难辨识从 A 到 B 的步骤，需耐心。
- 不要让"完美分析"卡住——重要的是开始使用，多练习即熟练。
- 链锁分析可增进对内在想法与感受的觉察。

## 六、遗漏环节分析（通用讲义 8）

- **讲义 8**：详细步骤。
- **练习单 3**：遗漏环节分析练习单；可课上用来分析成员未完成作业的行为。
- 优点：问题可快速问答。

### 什么是遗漏环节分析

探究一连串事件以澄清**缺少了哪个有效行为**及其干扰因素。两种缺失：
1. **预期的行为**：同意/被指示/计划/希望去做的（准时上班、做作业、锻炼）。
2. **需要的行为**：特定情境中需使用的有效技能。

### 四步问题链

按顺序问答，直到问题不再有意义，然后转入问题解决：

| 问题 | 若"否" → | 若"是" → |
|------|-----------|----------|
| 1. 你知道需要/预期做什么吗？ | 问"什么妨碍你知道？"，转入问题解决 | → 问题 2 |
| 2. 你愿意去做吗？ | 问"什么妨碍你愿意？"，转入问题解决 | → 问题 3 |
| 3. 你曾想到要去做吗？ | 停止，找出能让成员想到的方法 | → 问题 4 |
| 4. 什么妨碍了你马上去做？ | 针对妨碍因素问题解决；团体中集思广益 | — |

### 遗漏环节 → 解决方案速查

| 遗漏环节 | 解决方案 |
|----------|----------|
| 不知道要做什么 | 重新说明任务 |
| 不知道怎样做 | 示范、练习、角色扮演 |
| 不愿意做 | 利弊分析、承诺策略、认可矛盾 |
| 忘记/没想到 | 设置提醒、环境提示、练习单、同伴提示 |
| 情绪阻碍 | 先用情绪调节或痛苦忍受技能 |
| 环境阻碍 | 问题解决或调整任务 |
| 任务过大 | 拆分为更小步骤并强化 |

## 七、结合遗漏环节分析与链锁分析

- **简单问题**：仅用遗漏环节分析。
- **复杂问题**（知道该做什么却被复杂因素阻碍）：从遗漏环节分析问题 4 之后转入链锁分析。避免把复杂问题简单化，也避免对简单遗漏过度分析。

### 结合分析示例：开会迟到四十五分钟

- **遗漏环节分析**：知道（知道要准时、何时开会、怎样准时）→ 愿意 → 想过。但问题 4：“一连串事件阻碍了我"→ 转入链锁分析。
- **诱发事件**：早起煮咖啡看报纸，首页有感兴趣的丑闻。
- **易感因素**：前晚很晚睡，很累，动作缓慢，对诱惑无抵抗力。
- **行为链**：看时间知道没多少时间 → "迅速穿好衣服不会迟到" → 坐下看新闻 → 母亲来电 → 接了电话 → 想挂但有罪恶感 → 讲十分钟 → 发现至少迟到十分钟 → "反正要迟到就慢慢来" → 看完新闻 → 换好衣服 → 迟到四十五分。
- **替代技能节点**："迅速穿就行" → 改成"现在就换衣服，提醒自己看报纸通常会迟到"；很累时不坐下看报；快迟到不接电话；用 DEARMAN 对母亲说晚点回电；不因小失误放弃全部（迟到十分钟好过四十五分钟）。
- **预防计划**：早睡增加睡眠；每周给母亲打一次电话；上班前不打开报纸。
- **修复计划**：向同事道歉；未来两个月提早出席；主动帮迟到的同事了解错过信息。
