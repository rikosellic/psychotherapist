# 认知行为疗法(第三版)_02_03_初访与评估

来源：`knowledge/干预/认知行为疗法(第三版)/book/认知行为疗法(第三版).md`
抽取范围：第2-3章。
用途：指导 agent 理解 CBT 治疗师如何与来访者进行初次接触和系统评估。

## 第2章：与来访者的初次互动

### 章节主线

初次联系的场景因工作环境而异，但核心目标一致：建立融洽关系、收集基本信息、开始个案概念化。从电话到等候室到治疗室，每一步都是建立治疗关系的关键时刻。

### 初次电话联系

- **保持敏感**：来访者可能花数月甚至数年才鼓起勇气打电话。初次电话不适合做彻底评估，只收集基本信息确定下一步计划。
- **入手提问**："我今天能帮你做些什么？""是什么让你今天打电话过来？"
- **时间管理**：控制通话时长（如"在下一个来访者到来之前，我有10分钟"），温柔打断："具体情况让我们在面对面会谈中详细说。"
- **提出建议选项**：(1)邀请来做深入评估；(2)转介给其他专业人员（需小心避免让对方感到被拒绝）；(3)提供信息而非治疗（自助手册、支持小组）。
- **安排第一次来访**：告知目的和内容，确保清楚时间、持续时长、到达方式。获得发送提醒函的许可（注意保密，使用无标记信封）。
- **议定费用**：APA 伦理准则要求"在可行的情况下尽可能早"讨论。介绍未付费和未赴约政策。

### 会见来访者之前

- **准备物品**：知情同意书、HIPAA 表格、评估量表、笔、手表/时钟。提前到达，整理办公室，清除其他来访者信息。
- **手机静音**：治疗师和来访者双方。来访者对手机要求的反应可提供个案概念化信息（如公然拒绝→应对权威有困难）。
- **穿戴专业**：女性避免低胸、短裙、紧身、突出妆容、叮当作响首饰；男性穿有领衬衫、齐整裤子（非牛仔裤）、正式鞋。
- **注意焦点**：注意力集中在来访者身上，而非自己的表现。自我关注会错过来访者正在说的内容。

### 在等候室与治疗室中

- **等候室**：只称呼名不称呼姓以保护隐私，介绍自己，陪伴到办公室。
- **社交礼仪**：开场使用社交礼仪，询问希望如何被称呼。介绍自己全名和称呼方式。
- **告知实习身份**：APA 伦理准则 10.01c 要求告知来访者督导师信息。
- **征询录音/录像许可**：解释原因、谁可看到、保密措施。
- **介绍会谈框架**：先"处理业务"（知情同意、保密），接着进行评估。

### 知情同意与保密

**知情同意五要素**：
1. 来访者理解获得许可的目的和意义
2. 告知潜在获益和风险（不能保证每位来访者都能变好）
3. 告知其他选择（其他疗法、药物、不作为）及不作为的可能结果
4. 来访者可随时退出，不会受歧视
5. 包括保密及保密限制的讨论

**保密原则**：
- 向来访者保证保密对治疗师和诊所人员的重要性（伦理和法律要求）
- 在治疗室外相遇：治疗师不主动接触来访者
- **保密例外（必须告知）**：(1)威胁伤害自己；(2)威胁伤害他人；(3)怀疑虐待儿童或老人；(4)法庭传唤记录
- 为第三方工作（雇主评估、法庭评估）时保密规则不同

### 个案概念化（迄今为止）

即使只有有限互动，也能收集有用信息。异常强烈的反应可能提示对特定话题敏感，应在评估中深入探究。

### 珍妮的初次电话与评估开端

- 珍妮·S.，被初级保健医生转介治疗抑郁，但不确定是否需要治疗。
- 丈夫和医生担心她，她自己觉得"只是无聊，有点迷茫"——孩子长大了，"没有人真正需要我了"。
- 治疗师以合作性经验主义回应：不强求，提供多种选择（评估后决定是否治疗，或转介生涯辅导/支持团体）。
- 第一次评估会谈：珍妮早到几分钟，穿得整齐，热诚打招呼。但在会谈中当治疗师问及朋友时，珍妮说"我不是交际花"，然后感觉被评判——"这就是我不想来的原因。我不需要有人告诉我，我做的所有事情都是错的。"治疗师及时澄清并示范 CBT 中的"读心术"概念。

## 第3章：评估过程

### 章节主线

评估对应旅行比喻的"启程阶段"——熟悉地形。评估不仅是做出诊断，更是用 CBT 术语初步解释症状（个案概念化），用于制订治疗方案。

### 留意你（治疗师）的反应

- **不当反应**：说经历是"怪异的"或"罕见的"；消极面部表情；间隔太久的反应。
- **恰当反应**：点头、说"我明白""我理解"，把困难正常化（"我在这儿见过许多有类似困惑的来访者"）。
- **囤积癖案例**：囤积物从腐烂食物到身体排泄物，无奇不有。当来访者问是否见过类似案例时，即使没见过也可以说："我没有遇到过一模一样的情况，但你和其他有囤积习惯的来访者有许多相似之处——囤积可能是因为考虑以后会用到，或担心丢掉有意义的东西会导致失去自己的一部分。不论囤积的是什么，治疗方式是相同的。"承认独特性，又通过关联让来访者感到被理解和有希望。
- 如果来访者感觉被评价，应直接澄清，示范"读心术"概念。
- 眼眶含泪是传递共情，没有问题。

### 评估的两个目标

1. **做出诊断**：确定保险责任范围、与其他专业人士交流、选择合适治疗方案。
2. **用 CBT 术语初步个案概念化**：理解来访者，而非仅仅命名症状。

### 评估工具

#### 半结构化临床会谈

- 常用工具：SCID-5-CV（DSM-5 结构化临床会谈）、ADIS-5（DSM-5 焦虑及相关障碍会谈）。
- **新手常见担忧**：会谈显得生硬、没有人情味。
- **解决**：随熟练度增加，越来越关注来访者内容而非按部就班。关键在于提出**后续问题**。保持温暖和共情。
- *例子*：ADIS 中强迫症初始问题："目前你是否被某些反复出现的、不合宜的想法、图像或冲动所困扰？"若回答"否"，可跟进："比如，你会不会有一些关于污垢或细菌的念头？"

#### 非结构化临床会谈

使用标准主题大纲：

| 主题 | 内容 |
|------|------|
| 人口统计学信息 | 年龄、民族/宗教、工作/教育、性取向/关系状态、居住情况——良好起点，不具威胁性 |
| 目前的问题 | 起始、经过阶段、频率、情境、自动思维、情绪/行为/生理反应、逃离和回避行为、过去治疗经历 |
| 其他问题 | 筛查情绪、焦虑、进食、物质使用、睡眠、躯体担忧、性/性别认同 |
| 家庭背景 | 社会经济地位、父母职业、家族精神病史、家庭关系动力——提供核心信念线索 |

#### 精神状况检查（Mental Status Examination, MSE）

对来访者外表、言语、行动和思维的描述。常用工具：MMSE（简易精神状态检查表）、MoCA（蒙特利尔认知评估量表）。

**MSE 核心维度**：一般描述（外表/行为/态度）→心境和情感→言语→感知障碍→思维（过程/内容）→感觉中枢和认知（定向力/记忆/注意力/抽象思维）→冲动控制→判断和见解→可靠性。

#### 自陈问卷

- 不能作为唯一工具，但可辅助。评估前让来访者填写有助于引出不愿口头分享的信息。
- 创建"备忘单"记录计分规则和分值意义。
- 注意会谈和问卷之间的差异——需要跟进向来访者询问澄清。

#### 行为观察

来访者是否轻声、矜持、勉强？是否会生气？是否有"禁区"话题？行为线索都是饶有深意的信息：强迫症患者拒绝握手、物质滥用来访者散发酒气、皮肤抓痕来访者穿着暴露。

#### 自我监控

记录靶行为的发生日期、时间、情境、想法和情绪反应。

- **皮肤抓挠行为案例**：来访者双腿遍布条状斑痕。通过一周自我监控发现：(1)妈妈在家尤其急着用卫生间时，行为很快结束；(2)夜晚妈妈睡着后可独占卫生间，行为持续数小时；(3)疲乏或伤心时频率增加；(4)需要"完整感"——在一个部位不停挠直到觉得"光滑无物"。还意外发现她也在拔腿部汗毛——从未意识到的行为，对治疗计划很重要。

#### 向其他专业人士请教

- 必须先获来访者允许。
- 新手常对此焦虑，但不去请教才被认为缺乏能力。
- **OCD 拒绝暴露案例**：来访者在以擅长 EX/RP 闻名的诊所治疗过但未成功，自称"从来没做过暴露"。联系前治疗师后发现：来访者完全拒绝暴露疗法。新治疗师坦诚探讨："是否愿意考虑做一些暴露？"来访者回应绝不会考虑。转介精神科医生考虑药物，并告诉她改变主意后可随时回来。通过与先前治疗师沟通，避免了重复负面体验，同时敞开未来大门。

#### 与来访者生活中其他人交流

必须先征得明确同意。使用目标清晰的问题，选择能帮助个案概念化的信息。

#### 以初步问题清单结束

珀森斯（Persons, 1989, 2008）建议创建问题清单——"包含来访者所有困扰的清单"，包括诊断问题和失业、夫妻冲突、医疗状况等。

### 评估中的常见挑战

1. **"暂停整理思路，来访者会认为我不知道该做什么"**：来访者大多认为停顿是治疗师在思考。短暂停顿没问题；长停顿可告知需要休息。也可以分两次完成评估。
2. **"遗漏了重要信息"**：可打电话给来访者澄清，来访者通常将此解读为关心。但过于频繁需跟督导师讨论。个案概念化是持续过程，不是一次完成的。
3. **"我会把评估搞砸"**：所有治疗师都犯错，督导师就是来纠错的。错误提供学习机会。

### 珍妮的评估会谈

- **人口统计学信息**：45岁已婚女性，两个女儿（高中一年级和初中），全职妈妈，双亲酗酒且离异，上过两年大学后做秘书，婚后辞职。
- **ADIS-5 评估结果**：符合重性抑郁障碍（中度，反复发作）、社交焦虑障碍和中度酒精使用障碍诊断标准。珍妮呈现有趣的否认倾向——需跟进询问才会承认症状。
- **行为观察**：对酒精问题防御性强（"那是我的父母，不是我"）。
- **问题清单**：抑郁、社交焦虑、酒精使用、身份认同问题（"我还有什么用？"）、家庭关系问题。珍妮最初不认为喝酒是问题，治疗师温和地坚持将其放在清单上。
- **自陈量表**：BDI-II 28分（中度抑郁），SPS 38分（严重社交焦虑），SIAS 52分（严重社交焦虑）。
