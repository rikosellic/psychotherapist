## 第 2 章：Psychoanalytic Approaches

### 章节主线

- 本章不是只讲 Freud，而是从古典精神分析一路走到当代心理动力学治疗。
- 章节核心线索：
  - Freud 如何从催眠、hysteria 和 talking cure 发展出精神分析。
  - 古典理论如何用驱力、无意识、发展阶段、人格结构解释症状。
  - 后续理论如何转向自我功能、客体关系、自体、关系场和依附。
  - 临床技术如何围绕自由联想、解释、防御、移情、反移情、梦和关系模式展开。
- 本章适合作为 agent 的动力学框架：把问题写成“早期关系如何内化、无意识如何重复、防御如何保护又限制、治疗关系如何成为改变现场”。

### 精神分析与心理动力学治疗的基本定义

**古典精神分析**

- Freud 的原始模型是一人内在心理模型，治疗师被设想为空白屏幕，主要任务是倾听来访者无意识冲突和重复的适应不良模式。
- 古典分析常让来访者躺在沙发上，治疗师坐在身后，尽量减少治疗师表情和反应对来访者联想的影响。
- 目标是通过洞察使无意识动力进入意识，从而减少症状和关系重复。

**现代精神分析**

- 现代精神分析包括 ego psychology、object relations、self psychology、relational / two-person psychoanalysis。
- 治疗不再被理解为治疗师客观解剖来访者内心，而是两个人共同构成的关系场。
- 来访者的移情、治疗师的反移情、治疗关系中的 enactment 都是理解模式的材料。

**心理动力学治疗**

- psychodynamic psychotherapy 是更宽泛的临床术语。
- 它强调无意识行为模式、早期关系、重复性人际主题和洞察。
- 它可以比古典分析更短程、更聚焦、更支持性，也可以整合依附、关系和现代评估工具。

### Freud 的历史脉络

**从 hysteria 到 talking cure**

- Freud 起初是神经科医生，接触大量被诊断为 hysteria 的患者，症状包括麻木、瘫痪、震颤等无法解释的身体表现。
- 他受到 Charcot 催眠研究影响，后来又与 Breuer 合作，受 Anna O. 个案的 talking cure 启发。
- Freud 逐渐放弃催眠，转向让来访者谈论情绪化的童年经验。

**seduction hypothesis 的意义**

- Freud 曾提出儿童早期性创伤被压抑后会导致 hysteria。
- 这一假说被冷淡对待；他后来撤回，转向 Oedipus complex 和幻想解释。
- 书中没有给出单一结论，而是把它作为临床警示：治疗师必须区分来访者事实、记忆、幻想、暗示、治疗师预设和投射。
- 对 agent 来说，处理创伤叙述时不能草率写“这是幻想”，也不能未经评估就把所有叙述当作客观事实。动力学解释要保持假设性。

### 古典精神分析的四个理论模型

**动力模型：冲动、能量与心理决定论**

- Freud 认为心理能量来自 eros 和 thanatos；临床上可转译为生命/亲密/满足需要与攻击/破坏/防御冲动。
- 心理决定论认为心理事件不是随机的，遗忘、迟到、梦、口误、过度反应和关系重复都可能有心理动机。
- 每个冲动可从四个角度理解：来源、目标、对象、强度。

- **案例：Katie 的食物囤积**

- Katie 婴幼儿时期经常被剥夺食物，饥饿和痛苦反复出现。
- 她必须强烈哭喊才可能得到喂养，这种“需要-强烈表达-迟来满足”的模式被内化。
- 成年后，她轻度囤积食物；现实中的食物短缺很小，但一旦可得性被打断，她的痛苦极强，并影响亲密关系。
- 这个案例说明动力模型的用法：当前行为不是孤立症状，而是早年需要满足模式在成人关系中的重复。
- 临床重点不是指责囤积，而是帮助她理解“食物-安全-关系可得性”的内部工作模型，并发展更灵活的安全感调节方式。

**地形模型：无意识、前意识、意识**

- 无意识像拥挤的入口大厅，冲动在其中活动。
- 前意识像守门人，筛选哪些内容可以进入意识。
- 意识只是较小的会客室。
- 治疗目标是 make the unconscious conscious，让原本间接影响行为的动力变得可被觉察和调节。

**发展阶段模型**

- Freud 的心理性发展阶段包括口唇期、肛门期、性器期、潜伏期、生殖期。
- 现代使用时，不必机械性化每个症状，更重要的是保留一个原则：早期需要如何被满足或挫败，会影响成人的调节方式和关系期待。
- 当 agent 使用这一模型时，建议写成“早期照顾经验如何塑造当前需求满足和关系反应”，而不是直接写“固着在某阶段”。

**结构模型：id、ego、superego**

- id 代表欲望和冲动，遵循快乐原则。
- ego 负责现实检验、计划、延迟满足和调解冲突。
- superego 包括良心和自我理想，来自内化的禁令、惩罚和期待。
- 症状常可理解为欲望、恐惧、现实和内化标准之间的妥协。

### 防御机制

**防御的基本理解**

- 防御机制是自我为了减少焦虑、羞耻、内疚或冲突而自动启动的心理操作。
- 防御不是“坏东西”。它保护来访者免于被情绪淹没，但僵化防御会限制现实检验、亲密关系和行动选择。
- 临床上应先理解防御保护了什么，再讨论它的代价。

**常见防御**

- **压抑**：无法意识到某些冲突性材料，但它们通过症状或关系反应出现。
- **否认**：拒绝承认现实，如关系已破裂却坚持“没事”。
- **投射**：把自己的愤怒、欲望或敌意归给他人。
- **反向形成**：表现出与真实冲动相反的态度，如过度冷漠掩盖依赖。
- **移置**：把对危险对象的情绪转向较安全对象。
- **合理化**：用看似合理的理由掩盖真实动机。
- **退行**：压力下回到更早、更幼稚的应对方式。
- **升华**：把冲动转化为可接受、创造性或社会有益的活动。

### 精神分析理论的发展

**自我心理学**

- Anna Freud 和后续自我心理学把焦点从 id 冲动转向自我功能、防御和适应。
- Erikson 将发展理论拓展为心理社会八阶段，减少了古典理论的纯性心理色彩。

**客体关系**

- 客体关系理论把动机重点转向人与重要他人的内化关系。
- 来访者携带 self-other 表征进入成人关系，旧关系模式会在新关系中反复上演。
- 治疗中的治疗师可作为新的关系对象，帮助来访者体验更稳定、更可承受的关系。

**自体心理学**

- Kohut 关注自体凝聚、自尊、镜映、理想化和共情失败。
- 来访者的问题可能来自早期自体客体需要未被充分回应。
- 治疗师通过共情、适度回应和“最佳失败”帮助修复自体。

**限时心理动力学**

- 短程/限时动力学治疗通常更聚焦一个核心关系主题。
- 它通过治疗关系和明确焦点加速模式显现。

**关系精神分析**

- 关系精神分析把治疗师视为参与者-观察者。
- 治疗师无法完全客观；其反应和主观性也是治疗场的一部分。
- 解释不再是权威宣判，而是供来访者共同检验的替代视角。

**依附知情心理治疗**

- Bowlby 和 Ainsworth 将真实照顾互动放到人格形成中心。
- secure、anxious-resistant、anxious-avoidant、disorganized 等依附模式可帮助理解来访者如何调节亲密与距离。
- 书中同时提醒：attachment 概念很流行，但不能把美国中产实验范式当作全球标准。

### 评估

**临床访谈与投射测验**

- 精神分析传统常使用临床访谈和投射测验，如 Rorschach、TAT、词语联想、人形绘画。
- 投射测验的逻辑是：模糊刺激会激活内在冲突和关系模型。
- 但其科学地位有争议，不宜作为诊断、儿童监护权或行为预测的主要依据。

**John 的实习评估课**

- John 在精神分析实习中，督导花大量时间讨论 Rorschach 和人形绘画，只用很少时间看 WAIS 和 MMPI。
- 一位督导曾依据 Rorschach 第一张卡反应推测某女性有性虐待史，这让实习生印象深刻。
- 这个故事的重点不是证明投射测验可靠，而是展示精神分析传统如何重视象征材料，同时也提醒 agent：投射材料最多作为探索线索，不能当作硬证据。

### 关键技术与治疗方法

**自由联想**

- 基本规则是：说出脑中浮现的任何内容。
- 操作上要减少外部干扰、减少身体不适、减少意识计划。
- 如果来访者总带着严密清单来会谈，动力学治疗师可能会把这理解为控制和防御的线索，而不是简单夸奖其准备充分。
- 临床笔记应记录反复主题、停顿、尴尬、绕开的话题、突然转移的情绪，而不是把每句话都象征化。

**解释（Interpretation）**

- 解释是把无意识衍生物、症状、防御、关系反应或梦转译成可意识化的意义。
- 解释需要前置条件：工作联盟、角色诱导、合适时机、具体材料支持。
- 解释不是治疗师越快越好；过早解释会增加防御。

**较差的解释案例**

- 来访者说自己过去没什么可谈，哥哥死了，谈也没用。
- 治疗师立即说她“竖起一堵墙”，并重复面质。
- 问题在于：缺少共情、时机过早、没有先理解谈哥哥为何困难。
- 这个案例说明：动力学技术不是“看穿来访者”，而是等待材料接近意识表面。

**较好的解释案例：沉默与“愚蠢”**

- 来访者沉默，治疗师没有立刻解释，而是询问沉默时的感受。
- 来访者说自己觉得愚蠢、担心治疗师失望。
- 治疗师追问“愚蠢”让她想到什么，来访者联想到父母常说她想法愚蠢。
- 解释变成：沉默可能是在保护自己，避免治疗师也认为她的私人想法愚蠢。
- 这个案例把防御、移情和早期内化批评连接起来，技术顺序是：体验澄清 -> 词语联想 -> 关系假设 -> 来访者确认。

**移情（Transference）**

- 移情是来访者把早期重要关系中的感受、期待和行为模式带入治疗关系。
- 它的特征是“不合比例”：来访者对治疗师的反应超出当前现实可解释范围。
- 识别移情时，要观察来访者是否把治疗师体验成评判者、控制者、冷漠者、拯救者、抛弃者或理想父母。

**John 的“机器人”案例**

- 来访者强烈指责 John 是最不敏感的男人，说他像机器人。
- 她过去的男性关系模式是：男性先不回应，最后爆发愤怒。
- 她把这种期待带到治疗关系中，等待 John 也失控。
- John 非攻击性回应可能提供了 corrective emotional experience。
- 这个案例说明：移情不是“来访者误会治疗师”这么简单，而是旧关系地图在新关系中的激活。

**反移情（Countertransference）**

- 古典定义把反移情看作治疗师未解决冲突被来访者激活。
- 当代视角更宽：治疗师对来访者的所有反应都可成为临床资料。
- 重要指标是强度、频率和持续时间。
- **有益功能**：(1) 帮助治疗师更深理解自己，提高诊断准确性；(2) 强烈不寻常反应可能反映来访者的问题
- 在“机器人”案例中，John 发现自己真的开始情绪退缩；这既反映自己的脆弱点，也揭示来访者如何在人际中诱发他人退缩。
- 反移情应通过自我觉察、督导和假设检验使用，不能变成治疗师随意表达情绪。

**洞见三角（Triangles of Insight）**

- **冲突三角**把问题看成：愿望/目标 -> 想象威胁/焦虑 -> 防御性妥协。
- 例子：来访者想从父亲那里获得更多独立，却担心父亲受伤或愤怒，于是说“他就是那样，没什么可做”，表面缩小问题，内在仍焦虑内疚。
- **移情三角**把材料连接为：过去关系 -> 当前外部关系 -> 治疗关系。
- 它适合描述重复人际模式，而不是给来访者贴固定人格标签。

**梦的工作**

- Freud 把梦视为通往无意识的道路，但现代动力学治疗不再依赖权威式象征解释。
- 梦的工作应让来访者自由联想到梦中元素，治疗师与来访者合作探索情绪、人物、关系和现实连接。
- 不能直接把梦解释为固定符号，例如树就等于性象征；这会把治疗师的预设压到来访者身上。

**核心个案：Andrew**

**问题描述**

- Andrew 感到妻子 Jane 控制自己。
- 他难以表达个人需要，常常顺从，然后又体验到被侵入、被困。
- 伴随焦虑、抑郁和对婚姻无望感。

**理论连接**

- Wachtel 用依附知情的心理动力学框架理解 Andrew。
- 重点不是简单说“妻子控制他”，而是看 Andrew 如何在亲密中既渴望联系又害怕被过度侵入。
- “overtracking”是核心词：照顾者过度追踪儿童情绪和需要，会让儿童为了保留自主空间而回避接触。

**会谈中的关键观察**

- 当 Wachtel 用 overtracking 命名 Andrew 的体验时，Andrew 立刻感到被精准理解。
- 但随后他转头看向别处，像是在测试治疗师是否允许他控制接触强度。
- Wachtel 没有急着解释，而是尊重 Andrew 对亲密距离的敏感。

**技术要点**

- 对 Andrew，过早解释可能被体验为治疗师也在 overtrack。
- Wachtel 先提供情感空间，间接传递“你可以被理解，也可以保留自主距离”。
- 后续建议也很间接：他也许可以和妻子说话时不必一直凝视对方，可以边看邮件边确认妻子仍在听。

**可复用启发**

- 动力学治疗不只是解释过去，也要创造新的关系经验。
- 对回避型或怕被侵入的来访者，治疗师的“不过度理解、不过度靠近”本身就是干预。
- 个案概念化要同时写早期关系、当前婚姻、治疗关系中的即时互动。

### 证据、文化、性别与灵性

**证据基础**

- 长程精神分析难以研究，成本高，目标也不只是症状减轻。
- 近期元分析和综述支持短程心理动力学治疗对抑郁、焦虑、躯体症状、进食障碍、物质相关障碍、边缘人格障碍有一定疗效。
- Driessen 等研究显示短程心理动力学治疗对抑郁有改善，且随访可能继续改善。
- 研究解释要注意 allegiance effect：研究者理论忠诚可能影响结论。

**文化敏感**

- 很多动力学研究没有报告种族/族裔信息。
- Andrew 案例也没有充分讨论文化、男性气质和婚姻期待。
- 书中特别指出：Andrew 作为 21 世纪已婚男性，必然受到“男人不应被妻子控制”等社会讯息影响，不能把一切都追溯到母亲。

**性别与性取向**

- 古典精神分析有明显父权和 phallocentric 历史，曾把女性和同性恋病理化。
- 当代精神分析正在转向更包容的性别和性取向理解。
- 使用动力学框架时，要避免 mother-blaming，也要避免把性少数身份解释为发展失败。

**灵性**

- Freud 对宗教高度负面，认为宗教投射、退行、非理性。
- 当代动力学取向更倾向把宗教和灵性经验当作有意义的关系表征与自我经验来理解。
- “God representation”可被探索为自我与神圣对象之间的关系，而不应自动被贬低。

## 第 3 章：Individual Psychology and Adlerian Therapy

### 章节主线

- 阿德勒理论的核心不是“自卑”这一个词，而是整体性、目的性、社会兴趣、生活风格、鼓励和再定向。
- 与 Freud 相比，Adler 更关注社会、家庭、教育、平等、女性处境和当下/未来目标。
- 本章适合 agent 用来处理：完美主义、气馁、自卑、青少年行为问题、家庭角色、归属困难、生活任务回避、身份模糊。

### Adler 的历史位置

**为什么 Adler 重要**

- 作者认为 Adler 的思想像“来自未来的人”：他早早提出了认知、社会兴趣、平等关系、女性主义、家庭角色、目标导向等后来被许多流派吸收的概念。
- Adler 不是 Freud 的学生，而是同时代思想家。
- 他服务的对象更多是工人阶级，而不是精英，这可能是他在后世被低估的原因之一。

**童年与理论来源**

- Adler 童年多病，患佝偻病，被车撞过，弟弟死在身旁，四岁肺炎时医生认为他可能活不了。
- 他在学校曾被建议退学做鞋匠，但父亲鼓励他，后来他成为班上数学最好学生。
- 这些经历与他后来的补偿、鼓励、克服不足、追求完善等概念高度呼应。

**与 Freud 的分离**

- Adler 曾任 Vienna Psychoanalytic Society 主席。
- 1911 年他提出 masculine protest，强调女性处于低地位是社会强迫和文化结果，不是生物低劣。
- 这一点与 Freud 的生物化和性心理解释发生冲突，最终分裂。

### 核心理论

**整体性与目的性**

- individual psychology 中 individual 指不可分割。
- 人的行为、情绪、身体、认知、家庭、文化、目标要作为整体理解。
- 阿德勒认为行为有目的，即使问题行为也在服务某个主观目标：获得归属、避免失败、保护自尊、争取权力、表达愤怒、逃避任务。

**社会兴趣 Gemeinschaftsgefühl**

- 社会兴趣是与他人和社区连接、合作、贡献的能力。
- 它是阿德勒心理健康的核心指标。
- 如果人只追求优越而缺乏社会兴趣，就可能变得自我中心、控制、竞争或虚荣。

**追求优越/完善**

- superiority 容易误解为支配他人；原意更接近追求完整、卓越和自我完善。
- 健康形式是克服不足，并带着社会兴趣贡献他人。
- 病理形式是过度自我关注、完美主义、控制、贬低他人或逃避失败。
- 书中用自伤举例：短期看是损害，主观上却可能带来注意、让他人小心、获得控制感或痛苦缓解。阿德勒式概念化要看到问题行为的“主观收益”。

**现象学与私人逻辑**

- 人根据主观知觉行动，而不是根据客观事实本身行动。
- private logic 是个体解释自我、他人和世界的独特规则。
- 例子：只有完美才有价值；如果我不尝试，就不会真正失败；别人迟早会忽视我；我要让位给更会吸引注意的人。

**软决定论**

- 生物、环境、家庭、文化都会影响行为，但不是单一因果决定。
- 人在有限条件中仍有选择。
- 阿德勒治疗不主张责备，而主张教育、鼓励和重新学习。

**生活风格**

- style of life 是个体在童年形成的认知地图，包含对自我、他人、世界、未来和理想自我的看法。
- 它通常在 5-6 岁左右形成，之后保持相对稳定。
- 生活风格既有意识部分，也有不在觉察中的部分。

**动物园三儿童例子**

- 三个孩子第一次看到狮子。
- 一个躲在母亲身后说想回家；一个脸色苍白却说不怕；一个瞪着狮子问能不能朝它吐口水。
- 三人都体验到自卑或威胁，但按照各自生活风格表达。
- 这个例子适合 agent 用来说明：同一外部事件不会自动产生同一反应，关键是主观解释和生活风格。

### 生活任务与病理理解

**六大生活任务**

- 工作/职业：能否合作、分工、友好处理责任。
- 社会关系：能否归属、连接、同理、合作。
- 爱与婚姻：能否平等、互相丰富对方生活、让彼此感到有价值。
- 自我：如何照顾身体、心理、社会自我，如何评价自己。
- 灵性：如何处理意义、价值、共同体和超越性。
- 育儿与家庭：如何在家庭中建立鼓励、合作和民主关系。

**气馁是核心病理**

- 来访者不是“坏”或“懒”，而是缺乏面对生活任务的勇气。
- 症状和问题行为常是气馁后的低效策略。

**自卑的三个层次**

- 客观自卑：现实中可测量的不足或劣势。
- 自卑感：主观感觉自己低于他人。
- 自卑情结：自卑感造成显著功能损害。

**基本错误**

- 过度概括：世界不公平，永远都是我承担。
- 错误或不可能目标：只有当老板才有价值；别人必须照顾我。
- 错误理解生活要求：世界都反对我；世界应任我取用。
- 否认自我基本价值：我完全没价值，没人会爱我。
- 错误价值观：必须赢，哪怕伤害他人。

### 关键技术与治疗方法

**建立治疗关系**

- 平等、友好、合作。
- 治疗师像友好的教师或顾问，而不是高高在上的解释者。
- 开场可问“你希望我了解你什么”，先认识人，而不是先固定问题。

**生活风格评估与分析**

- 使用家庭系统排列访谈、那个问题、早期回忆、心理出生序、生活任务评估。
- 目标是理解来访者如何获得归属、如何避免失败、如何保护自尊、如何形成基本错误。

**解释与洞察**

- 阿德勒也重视洞察，但他的无意识不是 Freud 式沸腾冲动，而是“暂时不在觉察中”。
- 教育、解释、面质都可帮助来访者看见生活风格和基本错误。

**再定向**

- 洞察必须进入行动。
- 技术包括未来自传、acting as if、reflecting as if、按钮技术、汤里吐痰、创造新形象、捕捉自己、任务设定、悖论、建议与游戏治疗。

**评估与核心技术**

**初始生活风格解释**

- 书中建议：首次会谈结束时，治疗师应该能分享一些“学到的关于来访者的重要东西”。
- 这种解释必须是合作式猜测。
- 例子：治疗师注意到来访者花很多能量避免冲突，指出这可能很耗竭，并邀请下次继续探索。
- 这个技术的作用不是下诊断，而是建立共同语言，让来访者愿意继续看自己的模式。

**家庭系统排列访谈（Family Constellation Interview）**

- FCI 类似 genogram，但更重视主观家庭位置。
- 关注家庭成员特征、谁与谁亲近、谁冲突、谁掌权、谁被偏爱、谁被忽视、来访者如何定义自己和兄弟姐妹。
- 心理出生顺序比实际出生顺序重要；Adler 明确反对把出生顺序当占星术。

**心理出生顺序**（非时间顺序）

- 长子可能先被宠、后被取代，发展出负责、权威或控制。
- 次子面对年长竞争者，可能竞争、反叛、寻找独特身份。
- 中间子可能平和，也可能感到被忽视和难找位置。
- 幼子未被取代，可能期待被特殊照顾，理想高但跟进不足。
- 独生子习惯成人关注，可能在同伴互惠上困难。
- 这些只是线索，不是公式；关键是来访者如何解释自己的位置。

**那个问题**

- 核心问法：如果你已经好了，生活会有什么不同？
- 它用于发现症状的目的、二级获益和被回避的生活任务。
- 例子：焦虑来访者说“我会工作更好”，可能提示焦虑让他避免工作任务；抑郁来访者说“我会对别人更好”，可能提示抑郁掩盖攻击或贬低倾向。

**早期回忆**

- 早期回忆不是历史考证，而是生活风格的窗口。
- 重点是主题、人物位置、情绪、主动/被动、孤立/连接、感知细节、权威与性别刻板线索。
- 可收集 3-8 个具体记忆，并与家庭系统排列材料相互印证。

**Sarah 的早期回忆**

- Sarah 记得和母亲坐红色雪橇下山，穿着暖和，和母亲一起笑，能感觉雪落在脸上的触感。
- 问题描述：这不是病理个案，而是用于示范早期回忆如何呈现生活风格。
- 理论连接：记忆中有亲密、合作、主动参与、感官愉悦和积极情绪。
- 临床启发：Sarah 的回忆提示较高社会兴趣、乐观取向、自我效能和支持性关系经验。

**Keith 的早期回忆**

- Keith 因焦虑来咨询，记得 6 岁时和父亲在停车场，车发动不了，父亲吼叫、砸方向盘、眼睛突出，他害怕又悲伤。
- 问题描述：Keith 在记忆中被动坐在车里，无法逃离，也无法改变父亲的失控。
- 理论连接：该回忆呈现一种焦虑、无助、被困、权威失控的世界图像。
- 临床启发：治疗计划应更支持性，帮助他建立安全感和主动性，而不是过早挑战。

**再定向技术**

**未来自传**

- 让来访者写从现在到未来的人生故事，用于暴露 guiding self-ideal。
- 对完美主义来访者，可布置一个反完美主义版本：写一个会犯错但仍可接受、有爱、能生活的未来。
- 这不是梦想清单，而是把生活风格中的未来目标拿出来检视和重写。

**Acting as if**

- 让来访者像已经拥有某种特质一样短期行动。
- 例如像自信的人、能表达需要的人、知道自己想要什么的人那样行动。
- 重点是把新行为具体化：怎么说话、怎么站、怎么选择、怎么回应他人。
- 它把改变变成“试穿角色”，降低阻抗。

**Reflecting as if**

- Watts 把 acting as if 改成三步：反思、计划、实施。
- 先问如果已经成为想成为的人，行为会有什么不同。
- 再列出具体行为并按难度排序。
- 最后从最容易的一两个行为开始作为作业。
- 它适合低自我效能来访者，因为先成功的小行动会带来鼓励。

**按钮技术**

- 基础假设是认知和想象会影响情绪。
- 来访者先回忆愉快记忆，再回忆痛苦记忆，再回到愉快记忆，体验情绪随注意内容变化。
- Mosak 用它帮助抑郁来访者理解“抑郁思想会按下抑郁按钮”。
- Sommers-Flanagan 夫妇把它改编为儿童和青少年的三步情绪改变技巧。
- 使用时不能否定真实痛苦；它是情绪教育和选择感训练。

**在汤里吐痰**

- 这是面质性技术，用来破坏问题行为的隐性收益。
- 来访者可能仍继续旧行为，但旧行为“不再那么香”。
- 适用前提是已有稳定联盟、治疗师识别了基本错误、来访者能承受温和面质。

**Mark 案例**

- Mark 15 岁，带爆炸物到学校，认为大家过度反应。
- 问题描述：他使用“我不会被抓”的犯罪思维保持无责，生活风格像是“我可以为所欲为且不负责”。
- 技术尝试：John 猜中其想法，试图指出他事实上被抓了，让“我不会被抓”不再好用。
- 来访者反应：Mark 抵抗，说自己不是被抓，而是被告密。
- 技术调整：John 没有继续强压面质，而是改用创造新形象。
- 新形象：把“我不会被抓”变成红色停止灯，提醒危险；再加入新自我陈述“我是会做出好决定的人”；用棒球卡作为正向强化。
- 临床启发：阿德勒技术不是僵硬执行。汤里吐痰失败时，可以转向视觉化、自我陈述和行为强化。

**创造新形象**

- 帮助来访者发展新的自我图像，替代旧的负性或问题图像。
- 可通过视觉化、绘画、语言脚本和行为强化完成。
- Mark 的红色停止灯和“会做出好决定的人”是典型例子。

**捕捉自己**

- 教来访者在旧模式刚启动时识别它。
- 书中篮球运动员因训练中暴怒面临离队；治疗师让他先注意完美主义自我批评何时出现，何时从自责转向责备他人。
- 初期目标不是马上改变，而是抓到模式。
- 这是后续行为自我监测和认知自我监测的前身。

**任务设定与间接建议**

- Adler 对抑郁来访者使用两阶段社会兴趣任务。
- 第一阶段：只做让你觉得可以接受的事；如果没有，就至少不要勉强做厌恶的事。
- 第二阶段：时常思考如何给别人带来一点快乐。
- 如果来访者说自己没有快乐给别人，Adler 进一步降低门槛：先只想，不必做。
- 这个技术同时包含任务设定、汤里吐痰、悖论、间接建议和社会兴趣训练。

**悖论策略**

- Adler 是早期使用悖论指令的人之一。
- 悖论可通过规定症状或夸张问题模式让来访者获得控制感。
- 但它是高风险技术，不适合自伤、暴力、成瘾等危险行为。

**建议、暗示与方向**

- 阿德勒治疗师可以直接建议，但应放在友好、合作、鼓励性的关系中。
- Corsini 的 “I’ll Betcha” 用小额打赌推动来访者尝试具体行为，最后由来访者判断是否有效。
- 建议不是命令；它属于再定向阶段，前提是洞察和动机已经出现。

**阿德勒游戏治疗**

- Terry Kottman 发展了 Adlerian play therapy。
- 儿童被看作独特、有创造力、追求连接和归属、拥有虚构目标、需要社会兴趣。
- 四阶段仍是关系、生活风格调查、洞察、再定向/再教育。
- Crucial C’s 包括 connect、capable/competent、count、courage。
- 儿童个案通常还需要父母和教师咨询。

**Sean 案例：完美主义、愤怒与身份**

**问题描述**

- Sean 17 岁，Job Corps 学生，因反复愤怒爆发被转介。
- 他自述 half black and half white。
- John 先询问喜欢什么，Sean 说足球、篮球、画画、写歌词；再问讨厌什么，他说也是这些，因为做不好就会生气。

**理论连接**

- Sean 的愤怒不是单纯冲动控制问题，而是生活风格中的完美主义和“不能低于别人”。
- 他认为自己应该做对、应该最好；一旦失误，就感到强烈压力和愤怒。

**技术要点**

- John 迅速给出解释：问题与完美主义有关。
- 更理想的阿德勒做法应先请求许可，但对青少年来说，快速具体的问题形成有时能建立合作焦点。
- John 后续用认知反映和情绪共情，让 Sean 自己说出“我从小就是这样，不想输，不想低于别人”。

**可复用启发**

- 对青少年愤怒个案，不要只写 anger management。
- 要评估愤怒背后的私人逻辑：我必须最好、失败等于低人一等、身份处境要求我证明自己。
- Sean 后续把双种族身份与完美主义需要联系起来，说明阿德勒概念化必须纳入社会文化因素。

**Kisha 案例：家庭系统排列访谈与身份再定向**

**问题描述**

- Kisha 18 岁，Dr. Murray 通过家庭系统排列访谈探索她与弟弟 Shane 的关系。
- Shane 是家庭小丑，笑话常以 Kisha 为代价；他运动但不够出色，父亲认为他不努力。
- Kisha 保护 Shane，又怨恨他；她说 Shane 和父亲更能做自己想做的事，而自己甚至不知道“我的方式”是什么。

**生活风格理解**

- Kisha 的自我感模糊且从属。
- 她似乎形成了一个基本错误：为了在家庭中被接受，自己需要让位给更吸引注意的年轻男性。
- 她既想获得父母认可，又因从属位置阻碍了身份发展。
- 家庭动力中，父亲与 Shane 更结盟，母亲通常不主张，除非事情特别重要。

**治疗师如何工作**

- Dr. Murray 一边收集材料，一边温和验证 Kisha 的观点。
- 她不是直接批评 Shane 或家庭，而是帮助 Kisha 看见自己总在谈弟弟，很少谈自己。
- 初始生活风格解释聚焦两个问题：Kisha 是谁？Kisha 想要什么？

**干预方向**

- Acting as if：一周内像“知道自己想要什么”的人那样行动，观察感受、想法和行为。
- 汤里吐痰：当 Kisha 把聚光灯放在 Shane 身上，她就看不见自己的方向。
- 创造新形象：从“Shane 可以做想做的事”转为“Kisha 也可以做自己想做的事”。

**可复用启发**

- 家庭系统排列访谈不是收集家谱，而是看来访者如何通过家庭成员定义自己。
- 阿德勒解释可以温和但很有力：不是“你错了”，而是“这个旧生活风格不再帮助你面对现在的生活任务”。

### 证据、文化、性别与灵性

**证据基础**

- 阿德勒个体、婚姻、家庭、团体治疗的严格研究相对少。
- 少量研究显示阿德勒治疗略优于安慰剂，与精神分析和个人中心疗效相近。
- 阿德勒游戏治疗有小型 RCT 支持，可减少小学生课堂破坏行为。
- Ready to Learn、Student Success Skills 等学校项目有经验支持。
- 社会兴趣与较少精神症状、较好健康和更高幸福感相关。
- 出生序研究复杂，必须区分心理出生序和实际出生序。

**结果测量**

- 可用 BASIS-A、Sulliman Social Interest Scale、Social Interest Index、Social Interest Scale。
- OQ-45 可追踪症状、人际关系和社会角色。
- 自我效能量表也适合评估阿德勒治疗中的“面对任务能力”。

**文化敏感**

- Adler 服务工人阶级，强调社会平等和社会动力，具有强社会正义传统。
- 但出生序、家庭系统排列、生活任务都需要文化适配，不能假设西方核心家庭模式普遍适用。

**性别与性取向**

- Adler 是早期女性主义心理治疗师之一，反对女性低劣论。
- 但其时代对性少数仍有局限，当代阿德勒治疗必须采取肯定性和多元文化取向。

**灵性**

- Adler 将灵性视为生活任务之一。
- 他的灵性观与“爱邻如己”和 compassionate community 接近。
- 宗教/灵性应被看作连接与贡献的资源；若它制造羞耻、排斥或暴力，也需要被临床探索。

## Agent 使用建议

- **使用精神分析/心理动力学时**
  - 优先用于关系重复、防御强、早期经验影响明显、移情强烈、人格模式、亲密关系困难、复杂羞耻或依赖冲突。
  - 个案概念化写法：当前症状 -> 早期关系/内部工作模型 -> 无意识冲突 -> 防御 -> 移情/反移情 -> 重复人际主题 -> 治疗关系中的新经验。
  - 技术顺序不要跳：先联盟和角色诱导，再自由联想和材料收集，再防御分析、移情观察、解释和修通。
  - 案例调用：Katie 用于早期匮乏与内部工作模型；沉默案例用于防御先于冲突解释；机器人案例用于移情/反移情；Andrew 用于依附知情和治疗距离。

- **使用阿德勒治疗时**
  - 优先用于自卑、完美主义、气馁、生活任务回避、家庭角色、归属困难、青少年行为问题、身份模糊。
  - 个案概念化写法：主诉行为的目的 -> 私人逻辑 -> 生活风格 -> 基本错误 -> 家庭系统排列/心理出生序 -> 生活任务 -> 社会兴趣 -> 鼓励与再定向。
  - 技术顺序：平等关系 -> 目标对齐 -> 生活风格评估 -> 家庭系统排列/那个问题/早期回忆 -> 初始解释 -> 再定向行动。
  - 案例调用：Sarah/Keith 用于早期回忆；Sean 用于完美主义和愤怒；Mark 用于汤里吐痰及其失败后的调整；Kisha 用于家庭系统排列和身份再定向。

- **两章共同注意**
  - 两者都重视早期经验，但精神分析偏无意识冲突和关系重复，阿德勒偏目标、归属和生活风格。
  - 深层解释、移情解释、梦解释、汤里吐痰、悖论策略都要求联盟、时机和督导。
  - 不要把精神分析写成“所有问题都是性冲突”，也不要把阿德勒出生序写成固定人格公式。
  - 所有解释都要保留假设性，并纳入文化、性别、性取向、灵性和社会处境。
