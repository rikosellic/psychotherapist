# Counseling and Psychotherapy Theories in Context and Practice 3rd edition_index

来源：`output/markdown/Counseling and Psychotherapy Theories in Context and Practice 3rd edition.md`

原书：John Sommers-Flanagan & Rita Sommers-Flanagan, *Counseling and Psychotherapy Theories in Context and Practice: Skills, Strategies, and Techniques*, Third Edition.

用途：供 agent 查询主要咨询与心理治疗理论的历史脉络、核心概念、个案概念化方式、典型技术、治疗关系、文化/性别/灵性适配和整合治疗思路。内容为中文版提取，保留关键英文术语。

说明：各章节文件均包含“关键治疗技术和干预方法”，技术名称保留英文对照，并将案例/示例直接嵌入对应技术条目下，便于 agent 按理论生成个案概念化或干预方案时直接调用。

## Agent 使用建议

如果任务是“按某个理论生成个案概念化”，优先读取该理论所在文件，并使用每章的“个案概念化模板”。

如果任务是“比较不同理论如何理解同一案例”，读取 index 后选择 2 至 4 个理论文件，按“问题来源、维持机制、治疗目标、关系姿态、核心技术、文化风险”六项对照。

如果任务是“为来访者选择治疗方法”，不要只按症状匹配理论；同时考虑来访者目标、改变阶段、文化身份、治疗关系偏好、风险水平和可获得资源。

如果任务涉及创伤、自杀、严重精神病性症状、虐待、儿童保护或医疗风险，本书内容只能作为理论参考，应优先使用危机评估、伦理法律规范和专业督导。
