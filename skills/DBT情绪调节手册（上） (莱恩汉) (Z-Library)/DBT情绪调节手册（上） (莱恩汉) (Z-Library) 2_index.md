## Agent 使用建议

如果任务是理解 DBT 的理论、适用范围或课程规划，先读 `_01_02`。

如果任务是设计或运行一个技能训练团体，读 `_03_05`，再按目标模块读 `_06_07` 或 `_08_10`。

如果任务是分析问题行为、建立正念基础或做链锁分析，读 `_06_07`。

如果任务是处理人际冲突、情绪调节、危机生存、接纳现实或成瘾冲动，读 `_08_10`。

涉及自杀、自伤、暴力、严重成瘾戒断、精神病性症状、躁狂或医疗危险时，本资料只能作为技能训练参考；应优先使用危机评估、急诊、精神科、专业治疗师或当地危机资源。
