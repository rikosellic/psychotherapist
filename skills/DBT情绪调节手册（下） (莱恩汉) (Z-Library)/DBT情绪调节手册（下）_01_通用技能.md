用途：供 agent 理解 DBT 技能训练的总体框架、目标设定准则、生物社会理论基础，以及行为分析的核心方法（链锁分析与遗漏环节分析）。

## 通用技能模块：介绍与行为分析

### 章节主线

- 本章包含两组内容：介绍性讲义（通用讲义 1-5，用于技能训练项目启动阶段）和行为分析讲义（通用讲义 6-8，用于个体治疗或技能训练中分析问题行为）。
- 生物社会理论（Biosocial Theory）是 DBT 的理论基石，解释为何某些人难以管理情绪和行为——先天情绪易感性与不被认可（Invalidating）环境交互作用的结果。
- 链锁分析（Chain Analysis）和遗漏环节分析（Missing-links Analysis）是 DBT 中最重要的行为评估工具，分别用于分析"做了不该做的事"和"没做该做的事"。

### 技能训练的目标

**通用目标**
- 当困扰于自己的行为、情绪和想法时，学会去改变。

**明确目标：减少的行为**
- 无法正念、空虚、失去联结、妄下结论
- 人际冲突与压力、孤单感
- 失去复原力、难以改变
- 情绪起伏不定、极端情绪、难以调节情绪
- 冲动、不愿意思考、难以接纳、偏执、成瘾

**明确目标：增加的技能**
- 正念技能（Mindfulness Skills）
- 人际效能技能（Interpersonal Effectiveness Skills）
- 情绪调节技能（Emotion Regulation Skills）
- 痛苦忍受技能（Distress Tolerance Skills）

**解决任意问题的四种选项（通用讲义 1a）**
1. **解决问题**：使用人际效能技能 + 问题解决技能
2. **自我安抚**：调节对问题的情绪反应，使用情绪调节技能
3. **接纳问题存在**：全然接纳问题及应对方式，使用痛苦忍受技能与正念技能
4. **停留在痛苦中**：不使用任何技能（让事情更糟）

### 技能训练的准则（通用讲义 3）

- 连续缺席四次视为中止训练和治疗
- 学员相互支持，保密、准时、努力练习技能、认可彼此、避免批评
- 不诱导他人从事问题行为，训练前不得饮酒或吸毒
- 训练之外不得有私密关系（性关系、危险行为、犯罪等）

### 技能训练的假设（通用讲义 4）

DBT 技能训练中所有成员和训练师必须遵守的 7 条假设：
1. **每个人都尽量做到最好**——在任何时候，每个人都竭力做到最好。
2. **每个人都想改进**——每个人都想改善生活，拥有快乐。
3. **每个人都需要做得更好、更努力、更有动力改变**——虽努力做到最好，但不意味着问题能自动解决。
4. **问题不一定是自己造成的，但不管怎样还是要解决**——必须改变自己的行为反应和环境。
5. **新的行为技能必须在相关情境中获取**——在相关情境下练习，不仅在首次学习的环境。
6. **所有行为（行动、想法、情绪）都有其原因**——即使我们不知道是什么原因。
7. **与其评判与指责，不如找出原因**——要改变就必须改变事件链锁。

### 生物社会理论（Biosocial Theory）（通用讲义 5）

**核心论点**：情绪失调源于先天生物因素与社会环境因素的交互作用。

**情绪易感性（Emotional Vulnerability）——先天基因**
- 对情绪刺激更为敏感，能注意到他人没注意到的细微情绪
- 更容易体会到情绪，反应更强烈、持续时间更长
- 情绪表达激烈，"情绪来袭时像一头狮子"

**冲动性（Impulsivity）——先天基因**
- 觉得冲动行为很难抑制
- 经常冲动行事然后陷入困境
- 情绪阻碍目标实现，总是"情绪上头，冲动行事"

**不被认可的环境（Invalidating Environment）**
- 情绪不被理解、不被接纳，被视为怪异、错误或糟糕的
- 情绪反应被忽略，无法得到帮助
- 不认可的人可能不知道如何认可，或害怕认可会助长情绪
- 可能本身承受压力、时间紧迫或资源太少
- 也可能是"玫瑰园里的一朵郁金香"——不适应社会环境

**交互作用**
- 先天基因和社会环境相互影响
- 个人反过来影响其所处的社会环境
- 社会环境中的各因素又共同影响个人

### 行为分析

#### 链锁分析（Chain Analysis）（通用讲义 7、7a）

**用于**：当你做出无效行为时，分析事件链锁及后果。

**8 步骤方法**：

**步骤 1：描述具体的问题行为**
- 要非常具体、详尽，不模棱两可
- 描述强度及其他重要特征
- 如果是"没做某事"，检查是：（a）不知道要做；（b）忘记；（c）拖延；（d）想到但拒绝做；（e）强烈拒绝或有情绪干扰

**步骤 2：描述诱发事件（Prompting Event）**
- 具体是什么事情导致了链锁反应？
- 在问题行为/冲动发生的前一刻发生了什么？
- 为什么在那一天而非前一天发生了问题行为？

**步骤 3：描述情绪易感性（Vulnerability Factors）**
- 身体是否有恙、饮食/睡眠问题、是否受伤
- 是否服用药物或饮酒
- 是否有压力事件（正面或负面）
- 强烈的情绪（悲伤、愤怒、恐惧、孤独）
- 之前的压力行为是否延续影响

**步骤 4：详细描述事件链锁**
- 列出链锁中的所有环节，像写剧本一样具体
- 环节可能包括：肢体语言/行动、身体感觉/感受、认知/信念/想法、相关事件、情绪
- 检查在同样情境下其他人是否有截然不同的想法、感觉或行为

**步骤 5：描述问题行为的后果**
- 他人的当时反应与之后的反应
- 行为对自己和环境的影响

**步骤 6：针对每个环节提出替代的应对行为或技能行为**
- 回到链锁中，在关键环节打圈
- 提出具体的技能替代

**步骤 7：制订预防计划**
- 如何减少情绪易感性
- 如何预防诱发事件再次发生

**步骤 8：修复伤害**
- 审视对他人或自己的既成伤害
- 努力修复（"不要用鲜花来修补打破的窗户"——修复背叛需要长期积累信任）

#### 遗漏环节分析（Missing-links Analysis）（通用讲义 8）

**用于**：当你无法做出有效行为时，找出遗漏环节。

**4 问题框架**：
1. **你知道需要或期望的有效行为是什么吗？**
   - 如果否→解决阻碍认知的问题（不专心、指导语不清晰、负荷过重等）
2. **你愿意采取需要的有效行为吗？**
   - 如果否→解决阻碍意愿的问题（练习全然接纳、做利弊分析、相反行为）
3. **你想过要去实践该行为吗？**
   - 如果否→让想法进入大脑（记在日历上、设闹钟、提前应对）
4. **什么阻碍了你立刻采取行动？**
   - 如果是拖延、延误、没心情、忘记方法→设立奖励、练习相反行为、利弊分析

### 练习单

- **通用练习单 1**（对应讲义 1）：使用技能的利弊分析——比较使用技能与不使用技能的利弊，在智慧心念中做决定。
- **通用练习单 2/2a**（对应讲义 7/7a）：问题行为的链锁分析——8 步骤记录表，含行为链锁环节列表（行动、身体感觉、认知/想法、事件、感受）。
- **通用练习单 3**（对应讲义 8）：遗漏环节分析——4 问题框架记录表。

### Agent 使用建议

- **适用场景**：当 agent 需要理解 DBT 的理论基础（生物社会理论）、设定治疗目标、或分析来访者问题行为时使用。
- **可直接引用**：链锁分析的 8 步骤可作为"行为分析"的标准化框架；技能训练的 7 条假设可用于评估来访者的改变动机；生物社会理论可用于个案概念化中解释情绪失调的成因。
- **需要结合**：`DBT情绪调节手册（下）_02_正念技能.md` 至 `_05_痛苦忍受技能.md` 各模块的具体技能，将分析转化为干预方案。
