# DSM5中文版_index

来源：`skills/DSM5中文版/book/DSM5中文版.md`

## Agent 使用建议

- **按症状匹配诊断**：先读 index 确定症状所属章节，再读取对应文件获取诊断标准、鉴别诊断和标注。
- **比较两个诊断**：读取两个相关文件，按"核心特征、关键鉴别点、共病模式、病程"四项对照。
- **为个案概念化提供 DSM-5 框架**：结合来访者症状表现，查找所有可能相关的诊断文件——DSM-5 允许多重诊断，共病是常见规则。
- **DSM-5 仅为诊断分类工具**：如需干预方案，必须结合其他治疗取向的 skill 文件（CBT、DBT、心理动力学等）。
- **文化背景考量**：评估时必须参考 SecI 文件的文化概念化框架和 SecIII 文件的 CFI 工具，避免将文化差异误解为精神病理。
- **紧急风险**：自杀风险评估（所有心境障碍、BPD、PTSD）、物质戒断的医疗急症、NMS 识别、谵妄——这些需要优先处理。
