# ICD-11精神、行为与神经发育障碍临床描述与诊断指南  B5_00_使用说明与总览

来源：`output/ocr-markdown/ICD-11精神、行为与神经发育障碍临床描述与诊断指南  B5.md`

覆盖：诊断指南使用方式、ICD-11 结构、临床描述与诊断要求、编码和限定词、agent 使用边界。

## 1. 本书性质

本书是 ICD-11 中“精神、行为与神经发育障碍”的临床描述与诊断指南，目的不是提供治疗手册，也不是提供量表分数换算表，而是帮助临床人员在评估资料充分时作出更一致的诊断分类。

对 agent 而言，它最适合作为三类用途：

- **结构化识别**：把用户描述的症状放入较大的诊断类别中，形成“需要进一步评估的假设”。
- **鉴别诊断提示**：提醒哪些症状、病程、物质/躯体因素、发展背景和功能损害需要补充询问。
- **个案概念化支架**：把临床表现同发育、应激、人格、物质使用、躯体疾病、文化处境和风险因素联系起来。

它不适合作为：

- 自动诊断器。
- 单凭文字聊天就给出确定诊断的依据。
- 取代精神科、临床心理、神经心理或医学评估的工具。
- 危机干预或治疗建议的唯一来源。

## 2. ICD-11 诊断条目的基本组成

多数条目包含以下信息：

- **代码与名称**：如 6A20 精神分裂症、6A70 单次发作抑郁障碍。
- **临床描述**：概括该障碍的典型表现、核心心理行为特征、病程和功能影响。
- **诊断要求**：指出达到该诊断通常必须具备的最低特征，例如症状数量、持续时间、功能损害、排除条件。
- **边界与排除**：说明何时应考虑其他障碍、躯体疾病、物质/药物作用、文化认可的行为或正常反应。
- **限定词/说明符**：用于描述病程、严重程度、当前状态、伴随特征或病因。

理解时要区分“临床描述”和“诊断要求”。临床描述帮助识别典型病例，诊断要求帮助判断是否达到诊断阈值。很多真实个案不完全符合典型描述，但仍可能达到诊断要求；反过来，出现某个典型症状也不等于一定满足诊断。

## 3. 诊断使用的基本判断顺序

agent 在阅读个案材料时，可按以下顺序组织信息：

1. **症状群**：用户描述的是情绪、焦虑、强迫、精神病性、分离、进食、物质使用、人格、认知，还是发育问题？
2. **时间线**：起病年龄、起病方式、持续时间、发作性或持续性、是否复发、是否随应激或物质使用变化。
3. **严重程度与功能损害**：学习、工作、人际、家庭、自我照料、法律/安全风险是否受损。
4. **排除因素**：是否由物质、药物、躯体疾病、神经系统疾病、睡眠不足、文化/宗教实践、正常哀伤或发展阶段解释。
5. **共病与主次关系**：多个障碍可共存；不要把所有表现都硬塞进一个诊断。需要判断哪个问题是当前主要临床焦点。
6. **风险评估**：自伤、自杀、他伤、精神病性命令、冲动暴力、严重营养不良、戒断、谵妄、虐待和照护风险需优先处理。
7. **进一步评估**：需要面谈、家属/学校/工作场所信息、医学检查、量表、神经心理测验或精神科评估时，应明确指出。

## 4. 症状不等于障碍

ICD-11 的多数精神障碍要求症状达到明显强度，并造成显著痛苦或功能损害。单个症状本身不能直接推导出诊断。

示例：

- 焦虑可以出现在广泛性焦虑、惊恐障碍、PTSD、强迫障碍、抑郁障碍、物质戒断、甲状腺疾病或现实压力中。
- 幻听可以见于精神分裂症、心境障碍伴精神病性症状、物质诱发性精神病、神经系统疾病、睡眠相关现象，也可能受文化宗教背景影响。
- 注意力差可以来自 ADHD、抑郁、焦虑、睡眠不足、创伤、物质使用、学习障碍或环境压力。
- 进食减少可以是神经性厌食、回避-限制性摄食、抑郁、躯体疾病、药物副作用或经济/文化因素。

因此，在 agent 输出中更合适的语言是：“这些表现可能提示……需要进一步评估……”而不是“你就是……”

## 5. 病程和发育背景很关键

ICD-11 中许多障碍的诊断依赖时间线：

- 神经发育障碍通常在发育期起病，表现为认知、语言、社交、学习、运动或注意控制的发展偏离。
- 精神分裂症等精神病性障碍需要持续的特征性症状和现实检验受损。
- 心境障碍需要区分抑郁、躁狂、轻躁狂、混合状态，以及发作次数和间歇期状态。
- PTSD、复合性 PTSD、延长哀伤障碍和适应障碍必须结合应激事件或丧失事件后的反应模式。
- 神经认知障碍需要关注认知功能相对既往水平的下降，而不是单次表现差。

儿童、青少年、老年人的表现经常不同。儿童的焦虑可表现为哭闹、黏人、躯体不适、拒学；老年人的抑郁可突出躯体症状和认知抱怨；谵妄在老年人和躯体病患者中尤其需要警惕。

## 6. 文化、宗教与语境

诊断不能脱离文化和语境。某些行为、体验或表达方式在特定文化、宗教或社群中可被理解为正常或被认可的实践。只有当体验超出文化可理解范围、造成显著痛苦或功能损害，或伴随现实检验明显受损，才更应考虑精神障碍。

使用时要询问：

- 这种体验在其文化或宗教背景中是否被认可？
- 个体本人和周围重要他人如何理解？
- 是否造成持续痛苦、危险或功能损害？
- 是否伴随其他症状，如思维紊乱、情感高涨/低落、回避、强迫、分离或认知下降？

## 7. 物质、药物和躯体病因

很多精神行为表现可能由物质、药物或躯体疾病引起。ICD-11 专门设置了物质使用所致障碍和继发性精神行为综合征。agent 在处理个案时应常规询问或提示评估：

- 酒精、阿片类、大麻、镇静催眠药、可卡因、兴奋剂、致幻剂、尼古丁、咖啡因等使用情况。
- 处方药和非处方药，如糖皮质激素、某些抗帕金森药、兴奋剂、镇静药、止痛药等。
- 神经系统疾病、内分泌疾病、感染、代谢紊乱、头部外伤、癫痫、睡眠障碍等。
- 急性意识改变、定向障碍和波动性注意障碍，这些应优先考虑谵妄或医学急症。

## 8. 共病与层级

ICD-11 允许多个诊断共存，但共病不能替代临床判断。常见共病线索包括：

- 抑郁与焦虑常共存。
- PTSD 可伴抑郁、物质使用、分离症状和躯体痛苦。
- ADHD 可伴学习障碍、对立违抗、焦虑、抑郁、物质使用风险。
- 人格障碍可与心境、焦虑、物质使用、进食、冲动控制问题共现。
- 神经认知障碍可伴精神病性症状、抑郁、焦虑、行为紊乱。

在输出中应把“主要临床焦点”和“需鉴别/共病评估”分开，避免给用户一串标签。

## 9. 风险优先原则

当材料提示以下情况时，应把安全和转介放在诊断讨论之前：

- 明确自杀计划、近期自伤、无法保证安全。
- 暴力风险、命令性幻听、被害妄想伴防御性攻击风险。
- 严重进食障碍导致低体重、晕厥、心律问题、补偿行为失控。
- 酒精、苯二氮卓类或其他物质戒断风险。
- 意识障碍、定向障碍、急性混乱、发热或躯体病恶化，可能为谵妄或医学急症。
- 儿童、老人、残障者受到虐待或照护缺失。

## 10. Agent 输出模板

在使用 ICD-11 信息时，可采用以下结构：

```text
根据描述，最需要进一步评估的方向是：
1. 可能的 ICD-11 类别：……
2. 支持线索：……
3. 尚缺信息：起病时间、持续时间、功能损害、风险、物质/药物/躯体因素、既往史、家族史……
4. 需要鉴别：……
5. 下一步建议：专业评估、量表/医学检查、危机处理或治疗转介……
```

对用户直接表达时，应避免确定诊断式断言，尤其是涉及精神病性障碍、人格障碍、神经认知障碍、性欲倒错障碍、儿童青少年诊断和法律风险的内容。
