# CBGT回家作业手册_index

来源：`skills/【人手一本，带着来上课】CBGT 回家作业手册/book/【人手一本，带着来上课】CBGT 回家作业手册 .md`

原书：上海市精神卫生中心，*【人手一本，带着来上课】CBGT 回家作业手册*（基于个体化评估的强迫症新一代 CBT 远程治疗研究 · 家庭作业册（团体 CBT 用））。

用途：供 agent 查询 OCD 团体 CBT（即认知行为团体治疗， Cognitive behavioral group therapy，CBGT）治疗中的核心评估工具（SUDs）、症状自我监测方法、暴露与反应预防（ERP）的家庭作业结构、暴露等级清单构建以及逐周 ERP 练习记录单。内容为中文版提取，保留关键英文术语。

说明：本手册是操作性极强的临床工具书，所有内容均围绕 OCD 的 CBT 团体治疗方案展开。各章节文件包含可直接引用的表单字段、操作指令和练习参数，便于 agent 在生成 OCD 个案概念化、制定 ERP 家庭作业或指导治疗师设计暴露练习时直接调用。

## 文件导览

### CBGT回家作业手册_01_评估工具与症状监测.md

覆盖：主观痛苦程度评分量表（SUDs）+ 第 1 周家庭作业（症状自我监测）。

适合查询：
- SUDs 量表 0–100 分级描述与临床应用
- OCD 症状自我监测表单结构与字段
- 强迫思维、情绪/躯体感受（SUDs）、强迫行为的记录方式
- 家属/陪伴者在 CBGT 中的角色与任务
- 症状监测的频率要求与示例

### CBGT回家作业手册_02_暴露计划与ERP练习.md

覆盖：第 2 周家庭作业（暴露项目清单）+ 第 3–11 周家庭作业（ERP 复习与在家自我执行 ERP 练习）。

适合查询：
- 暴露等级清单（exposure hierarchy）的构建方法与示例
- 真实暴露（in-vivo exposure）与想象暴露（imaginal exposure）的区分
- ERP 家庭作业的每日练习时长、频率与环境要求
- 真实暴露记录单与想象暴露记录单的字段结构
- SUDs 在暴露练习中的动态监测（开始/结束/高峰/时间间隔）
- 强迫行为阻止（ritual prevention / response prevention）在记录中的体现

## Agent 使用建议

- **适用场景**：当 agent 需要为 OCD 来访者设计 CBT/ERP 家庭作业、理解 CBGT 团体治疗的逐周作业结构、或提取 OCD 评估与暴露练习的表单模板时使用。
- **可直接引用**：SUDs 量表分级描述、症状监测表单字段、暴露等级清单格式、ERP 练习指令（时长/频率/环境要求）、真实暴露与想象暴露记录单结构。
- **需要结合**：ERP 的理论基础应参考 `Counselling and Psychotherapy Theories in Context and Practice` 中行为治疗与 CBT 章节；OCD 诊断标准应参考 DSM-5 或 ICD-11 相关材料。
- **注意**：本手册来自中国临床研究情境（上海市精神卫生中心），表单中的文化表达（如"思想化的强迫行为"、"家属或陪伴者"）反映了中国临床实践语境，在跨文化应用中需做适配。
