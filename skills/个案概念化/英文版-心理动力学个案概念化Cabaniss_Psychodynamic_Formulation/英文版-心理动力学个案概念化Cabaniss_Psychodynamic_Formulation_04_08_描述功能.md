## Part Two 导言：DESCRIBE 的核心原则

### 章节主线

在解释之前必须先描述。描述的对象包括"问题"（Problem，来访者当前最大的困扰）与"人"（Person，来访者的特征性思维、感受和行为模式）。本书采用**维度化方法**（dimensional approach，而非分类法），使用五个基本功能领域：自体、关系、适应、认知、工作与玩乐。对每个领域，都要识别优势（strengths）与困难（difficulties），并区分意识层面与无意识层面的模式。

### 核心原则

- **问题 vs 人**：同样的问题（如离婚）在不同的人身上反应截然不同，因为其人格模式不同
- **维度化 vs 分类化**：Hippocrates 的体液分类、Freud 的心性发展阶段、DSM 的症状清单都是分类法；维度描述更贴近实际功能
- **优势与困难**：人是复杂的镶嵌体（mosaics），在同一领域内可能既有优势也有困难，不同领域之间也可能强弱不等
- **意识与无意识**：有些人对自己的模式有意识（如"我一直有自尊问题"），有些则通过行为间接表达无意识困难（如通过贬低他人来维护自尊）

### 描述"问题"（The Problem）

回答："为什么这个人现在来见我？"
- 一般是但不仅限于来访者给出的求助原因
- 有时治疗师与来访者对主要问题的看法不一致——仍需承认并处理来访者的担忧

### 描述"人"（The Person）

回答：这个人通常如何思考、感受和行为？
- 通过描述其**特征性模式（characteristic patterns）**来实现
- 五个功能领域是描述这些模式的维度化框架

## 第 4 章：自体（Self）

### 章节主线

自体体验（self-experience）是心理功能的核心。描述自体模式使用两大变量：自我感知（self-perception，包含身份认同和自体幻想）与自尊调节（self-esteem regulation，包含对威胁的脆弱性、内部反应和利用他人进行调节）。

### 变量与描述框架

**自我感知（Self-Perception）**

| 子变量 | 定义 | 健康表现 | 困难表现 |
|--------|------|----------|----------|
| 身份认同（Identity） | 知道自己是谁、喜欢什么/不喜欢什么、才能和限制 | 能做出稳定选择（职业、关系），生活轨迹连贯 | 决策困难、频繁改变方向、缺乏方向感 |
| 自体幻想（Fantasies about the Self） | 关于自我的意识和无意识幻想 | 幻想与现实能力匹配，提供目标和动力 | 不切实际的幻想损害自尊；无意识幻想可通过梦和行为推断 |

**自尊调节（Self-Esteem Regulation）**

| 子变量 | 定义 | 适应良好 | 适应不良 |
|--------|------|----------|----------|
| 对自尊威胁的脆弱性（Vulnerability） | 对外界伤害的敏感程度 | 重大打击（失业、疾病）仍能维持 | 微小批评即全面崩溃（fragile self-esteem） |
| 内部反应（Internal Response） | 遭受威胁时使用的机制 | 幽默、升华、利他 | 夸大（grandiosity）/ 自我贬低 / 自虐（masochism） |
| 利用他人调节自尊（Use of Others） | 是否需要他人持续赞赏来维持自尊 | 能内化赞赏、自我认可 | 不断要求关注和认可、缺乏共情（empathy） |

**关键区分：嫉妒（Envy）vs 羡慕（Jealousy）**
- **Envy**：想摧毁别人拥有的东西（如说别人穿的衣服"俗气"）
- **Jealousy**：想与别人平起平坐（如想买同款围巾）
- Envy 是更具破坏性的模式，与自尊脆弱性高度相关

### 临床评估问题

- **自我感知**："你认为自己对优势和困难的感知准确吗？别人会怎么描述你是一个'知道自己是谁'的人？"
- **自尊脆弱性**："当你处在一群看起来更有钱/更有成就/教育程度更高的人中间时，你感觉如何？" "告诉我一次你没有得到想要的东西的经历，你当时感受如何？"
- **内部反应**："你觉得自己倾向于认为周围人无能吗？别人会说你是一个有竞争性的人吗？"
- **利用他人**："你做得好时会自己知道而不需要别人称赞吗？你能在没有他人意见的情况下做决定吗？"

## 第 5 章：关系（Relationships）

### 章节主线

关系模式是理解一个人功能的核心。描述关系使用五个变量：信任（trust）、自我与他人的感知（sense of self and other）、安全感（security）、亲密（intimacy）、互惠性（mutuality）。关系的变异性和不均一性很重要——同一人可能在不同类型的关系中表现不同。

### 变量与描述框架

| 变量 | 定义 | 健康表现 | 困难表现 |
|------|------|----------|----------|
| 信任（Trust） | 相信他人会关心和照顾自己 | 有一些真正信任的人 | 持续恐惧被攻击、被忽视、孤独；过度信任也有问题 |
| 自我与他的感知（Sense of Self and Other） | 三维看待自己和他人的能力 | 能看到好坏两面、他人有独立心智 | 分裂（splitting，全好全坏）、缺乏心智化 |
| 安全感（Security） | 关系中感到安全，即使有分离和分歧 | 多种持久关系、慢慢建立信任 | 快速进入/退出关系、充满焦虑 |
| 亲密（Intimacy） | 分享私人想法、感受和愿望 | 能在情感和性上适度开放 | 过度分享太快或过度保留从不敞开 |
| 互惠性（Mutuality） | 双向的给予和接受 | 健康地给予和接受 | 付出过多（无边界）或索取过多（缺乏共情） |

**关键概念：心智化（Mentalization）**
- 理解他人有与自己不同的想法和感受的能力
- 是共情的基础
- 缺乏心智化：无法想象他人可能有不同视角（如"她没回电话，一定是因为讨厌我"）

**关键概念：分裂（Splitting）**
- 将人视为全好或全坏——正常于幼儿期，持续至成人期则高度不适应
- 阻止三维感知他人，破坏关系的持续性和深度

### 临床评估问题

- **信任**："这个世界上有你真正信任的人吗？你一般觉得人们会关照你吗？你认为我能帮助你吗？"
- **安全感**："你独处时感觉如何？你倾向于慢慢还是快速建立关系？你经常担心被抛弃吗？"
- **亲密**："你能舒适地告诉朋友你不那么自豪的事情吗？有人你可以倾诉几乎一切吗？"
- **互惠性**："你觉得你的伴侣/朋友给你你需要的吗？他/她得到了他/她需要的吗？"

## 第 6 章：适应（Adapting）

### 章节主线

适应（adapting）指调节内部和外部刺激的能力。每个人有其独特的刺激容忍阈值和应对模式。使用四个变量描述：防御机制（defense mechanisms）、情绪管理（managing emotions）、冲动控制（impulse control）、感觉调节（sensory regulation）。

### 防御机制分类表

防御是无意识地应对压力以维持功能的方式——像平衡感一样自动且持续运作。

| 层级 | 防御机制 | 定义 | 示例 |
|------|----------|------|------|
| **较少适应性** | 分裂（Splitting） | 将好与坏的感受分离 | 将母亲体验为"全好"，将姐姐体验为"全坏" |
| | 投射（Projection） | 将不可接受的品质感知为来自外部 | 担心朋友不喜欢自己，实际是自己不喜欢朋友 |
| | 投射性认同（Projective Identification） | 将感受投射到他人然后互动使之体验该感受 | 不意识到自己对男友的愤怒，然后迟到1小时让他生气 |
| | 病态理想化与贬低 | 对他人赋予过度正面或负面情感 | "你是最好的治疗师，我上一个治疗师是白痴" |
| | 否认（Denial） | 否认不可接受的感受和想法 | 酒精戒断导致癫痫却说"我没有饮酒问题" |
| | 解离（Dissociation） | 将不可接受的想法/感受与现实断开 | 不记得被老板训斥 |
| | 见诸行动（Acting Out） | 通过行动表达不可接受的想法或感受 | 困难治疗后吃了一加仑冰淇淋 |
| | 退行（Regression） | 使用早期发展阶段的应对策略 | 考试周不洗澡不打扫房间 |
| **中等适应性** | 情感隔离（Isolation of Affect） | 忘记感受但保持对想法的意识 | "对丈夫离开我没有感觉" |
| | 理智化（Intellectualization） | 用过度思考替代感受 | 患癌后专注于研究数据而非感受恐惧 |
| | 合理化（Rationalization） | 逻辑化解释不可接受的行为 | "被解雇也没关系，反正我不喜欢那份工作" |
| | 置换（Displacement） | 将感受转向他人或活动 | 对妻子的愤怒转向对儿子发火 |
| | 躯体化（Somatization） | 将不适感体验为身体症状 | 被差评后头痛请病假 |
| | 压抑（Repression） | 将不适想法逐出意识 | 不愉快治疗后忘记下次预约 |
| | 反向形成（Reaction Formation） | 将不可接受的感受转化为其对立面 | 对 toddler 烦躁后变得过度保护 |
| | 抵消（Undoing） | 通过相反行为"修复" | 对朋友不好后买贵重的生日礼物 |
| **最适应** | 幽默（Humor） | 以笑话表达不适的想法或感受 | 被逃婚后说"我们本该选另一个婚庆公司的" |
| | 利他（Altruism） | 将痛苦感受转化为帮助他人 | 父亲患癌去世后创办癌症患者基金会 |
| | 升华（Sublimation） | 将不可接受的冲动转化为有益形式 | 悲伤时创作动人的画作 |
| | 抑制（Suppression） | 有意识地决定不关注困难感受 | 面对工作截止日期，决定稍后再处理家庭冲突 |

**防御的灵活性**：能否根据情境使用不同防御，而非僵化使用同一策略。无灵活性的人常显得控制欲强、难相处或脆弱。

**防御与感受/想法的关联**：
- **强迫型防御风格（Obsessive Style）**：倾向于隔离感受（如理性化）
- **癔症型防御风格（Hysterical Style）**：倾向于隔离想法（如躯体化）

### 其他适应变量

| 变量 | 定义 | 临床评估 |
|------|------|----------|
| 情绪管理（Managing Emotions） | 体验、容忍和表达广泛情感的能力 | "你生气或焦虑时会发生什么？你能'坐住'那种感受吗？" |
| 冲动控制（Impulse Control） | 管理欲望、攻击、赌博等冲动的能力 | "别人会说你是一个冒险者吗？你发现很难克制做不该做的事吗？" |
| 感觉调节（Sensory Regulation） | 对噪音、气味、质地等感觉的耐受能力 | "你对噪音或气味特别敏感吗？" |

## 第 7 章：认知（Cognition）

### 章节主线

思维的方式反映在几乎所有行为中。描述认知功能使用五个变量：一般认知能力、决策与问题解决、自我反思/现实检验、心智化、判断力。认知功能的发展变化可以为发展中出现问题的时间提供线索。

### 变量与描述框架

| 变量 | 定义 | 临床观察方法 |
|------|------|-------------|
| 一般认知能力（General Cognitive Abilities） | 智力、记忆、注意、言语/语言 | 叙述是否连贯？是否准时？是否能记住预约？必要时使用 MMSE 或神经心理学测试 |
| 决策与问题解决（Decision Making and Problem Solving） | 如何做出选择和解决问题 | 细节导向 vs 印象主义；研究导向 vs 直觉；计划者 vs 随机应变；创造性 |
| 自我反思（Self-Reflection）与心理-mindedness | 审视自己的思想和行为的能力；思考无意识动机的能力 | 通过试探性解释（trial interpretation）评估："你觉得可能你不想去才睡过头吗？" |
| 现实检验（Reality Testing） | 区分现实与幻想的能力 | 是否有人际偏执或离奇信念？是否在压力下短暂丧失现实检验？ |
| 心智化（Mentalization） | 理解他人有不同想法和感受的能力 | "你认为对方可能有不同的看法吗？" "你觉得我可能有什么感受？" |
| 判断力（Judgment） | 考虑行为后果并根据认知行动的能力 | 是否使用保护措施？是否法律问题？关键区分：不知对错（判断差）vs 知道但控制不住（冲动差） |

**认知功能的临床意义**：
- 认知困难可影响自尊（"我太笨了"）
- 认知困难可影响人际（被同伴嘲笑、被老师批评）
- 认知问题出现的时间可为发展干扰提供线索（如三年级后成绩突然下降）

## 第 8 章：工作与玩乐（Work and Play）

### 章节主线

Freud 说人需要能"爱与工作"（love and work），后人加上"玩乐"（play）。描述工作和玩乐模式不是评判"应该做什么"，而是评估这些活动是否适合其个人生活和社会环境。

### 评估变量

| 变量 | 评估要点 | 提问示例 |
|------|----------|----------|
| 与发展阶段/才能/限制匹配 | 是否与训练水平一致？（博士做入门级=不匹配） | "你的工作需要训练吗？你的学习/工作与你的兴趣匹配吗？" |
| 舒适与满意度 | 是否享受？是满足还是必须？是否变成强迫性？ | "你享受你的工作/玩乐吗？有让你期待去做的事吗？" |
| 对自我和依赖者的照顾 | 经济是否维持基本生活？是否有定期锻炼？ | "你能维持生计吗？你对他人的经济资助感觉如何？" |
| 文化适应性（Culturally Sanctioned） | 是否合法？在文化背景下是否合理？需考虑跨文化差异 | "你曾有过法律问题吗？你的休闲活动涉及非法物质吗？" |

### 玩乐的关键维度

- **时间**：占用多少时间
- **社交性**：独自还是与他人
- **深度/广度**：专注一项 vs 多项浅尝
- **性作为玩乐**：是否定期、是否满足、是否在关爱关系中
- **玩乐缺失**：完全无休闲活动是重要临床信号

## Putting it Together：完整 DESCRIBE 示例（Mr A）

**个案简介**：64岁同性恋男性，母亲4个月前去世后感到"漫无目的"和孤独。与伴侣B同居15年，关系稳定但各自生活较独立。即将面临退休。

**DESCRIBE 摘要**：

| 领域 | 描述 |
|------|------|
| **自体** | 身份认同未完全建立——能识别自己为好的照顾者，但面对退休不知所措；对自尊威胁较脆弱（莫名嫉妒伴侣）；内部反应偏向自我贬低（"他为什么想要和我在一起？"） |
| **关系** | 15年关系是优势；但近期出现信任困难（担忧被抛弃）；理想化伴侣影响三维感知；亲密能力中等；互惠性偏单向（付出多于接受） |
| **适应** | 一般适应良好；使用合理化和理想化防御；在压力下防御灵活性下降；无冲动控制问题 |
| **认知** | 主要为优势领域——一般认知功能良好，能经营自己的企业；自我反思能力一般（能理解与丧母的关联，但无法反思嫉妒感受）；心智化可能有些困难 |
| **工作/玩乐** | 工作匹配才能、有满足感、能自给自足；玩乐能力严重受限——没有工作外的兴趣和爱好 |

- **注意**：描述时务必同时记录优势和困难；区分意识与无意识模式；对每个变量的描述需有具体行为证据支撑（如不能说"防御灵活性差"而不举出实例）。
