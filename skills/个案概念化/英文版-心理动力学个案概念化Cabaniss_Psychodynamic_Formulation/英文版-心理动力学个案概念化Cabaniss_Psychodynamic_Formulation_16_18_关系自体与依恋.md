## 第 16 章：与他人的关系（Relationships with Others / 客体关系理论）

### 基本理念

**客体关系理论（Object Relations Theory）**由 Fairbairn, Winnicott, Balint, Bowlby, Guntrip 等人发展，基于 Klein 的早期概念。核心观点：早期与重要照料者的互动经验被**内化（internalization）**为无意识**关系模板（templates）**，影响成年后所有的关系。我们的心灵不仅仅是冲突的战场（如自我心理学所强调），更是由关系经验塑造的。

### 关系模板的形成

```
需求满足 → 正面模板：
  被爱和被照顾的儿童  ←→  充分提供爱和照顾的照料者

需求受挫 → 负面模板：
  需求未满足的、贫乏的儿童  ←→  不足够的照料者
```

- 儿童可以同时对同一照料者形成正面和负面模板
- 若正面经验 > 负面经验 → 基本"足够好"的整合，能三维看待他人
- 若负面经验占主导 → 使用**分裂（splitting）**来保护理想化照料者的形象

### 模板如何运作

- 成人关系激活**与当前情境相似**的早期模板——即使当前现实并不贴合
- 模板是**多维的**：人可以体验模板中任一方（儿童/照料者）的角色（通过认同，identification）
- **代际传递**：当成人面对照料自己孩子的情境时 → 可能认同自己的父母 → 对孩子反应如自己曾被反应的那样

### 适用场景

| 问题广度 | 表现 | 提示的早期问题 |
|----------|------|---------------|
| **弥散性**关系问题 | 不信任所有人、无法建立任何亲密关系 | 早期关系模板的全面负面影响（最早期二人关系） |
| **局限性**关系问题 | 特定情境下不现实地期待他人（如总认为权威人物会抛弃自己） | 已达到客体恒常性，但特定早期模板在相似情境中被激活 |

### 治疗方向

- 帮助来访者**认识到**有问题的模板并发展**新的、更健康的**模板
- **移情（Transference）**：来访者将在治疗关系中重新激活早期关系模板 → 可以被解释和理解
- 随着对负面模板的意识增强 → 对矛盾情感的容忍能力提高 → 分裂需求降低 → **客体恒常性（object constancy）**改善
- 治疗关系本身就是新的关系模板来源（如被专注的治疗师体验 vs 被分心的父母体验）

## 第 17 章：自体的发展（The Development of the Self / 自体心理学）

### 基本理念

**自体心理学（Self Psychology）**由 Heinz Kohut 创立，聚焦于自体（self）的发展。核心观点：早期照料者执行对儿童自体发展至关重要的功能，称为**自体客体功能（selfobject functions）**。

### 核心概念

**自体客体功能（Selfobject Functions）**的两个核心类型：

| 功能 | 定义 | 照料者的行为 | 失败后果 |
|------|------|--------------|----------|
| **镜映（Mirroring）** | 共情地、准确地反映孩子的能力和内在状态 | 对孩子"看我！"的回应：兴奋地肯定而非夸大或忽视 | 低自尊、羞耻倾向、对批评极度敏感 |
| **理想化（Idealization）** | 允许孩子将自己理想化，从而感到强大、安全和良好 | 照料者允许被仰望，然后孩子逐渐意识到照料者是独立的、有缺陷但基本良好的人 | 无法获得力量感、持续需要寻找完美他人 |

**关键术语**：
- **自体客体（Selfobject）**：该词不加连字符——反映 Kohut 认为幼儿将其照料者体验为不是完全独立于自己的
- 健康的自体发展需要：共情的镜映 + 被允许的夸大（appropriate grandiosity）+ 能理想化照料者

### 镜映失败的三种类型

| 类型 | 临床表现 |
|------|----------|
| 1. **不足的镜映** | 低自尊、羞耻倾向、对批评极度敏感——"我做什么都不够好" |
| 2. **过度/不准确的镜映** | 脆弱自尊、夸大但不敢面对现实限制——父母"你是最特别的"但现实并非如此 |
| 3. **理想化的失败** | 照料者不可被仰望（如父亲说"别像我一样浪费生命"）→ 孩子无法通过理想化获得力量感 |

### 区分自体心理学与客体关系理论

| 维度 | 客体关系理论 | 自体心理学 |
|------|-------------|------------|
| 核心关注 | 内化的关系模板如何影响关系 | 自体的发展和自尊调节 |
| 主要问题 | 关系模式反复 | 自尊脆弱、空洞感、共情困难 |
| 治疗焦点 | 识别和修正有问题的关系模板 | 通过治疗关系的共情调谐修复自体 |
| 关键机制 | 解释移情中的模板激活 | 转变内化（transmuting internalization） |

### 适用场景

- **自尊调节问题**（过低或过于脆弱）——"我永远不够好"
- **共情困难**：未被共情对待的人 → 难以共情对待他人（包括自己的孩子）
- **嫉妒（Envy）问题**：无法维持自尊的人 → 被他人拥有的东西所威胁 → 攻击性嫉妒

### 治疗方向

- **通过治疗关系修复**：治疗师的共情调谐提供"矫正性情感体验"
- **共情失败与修复**：治疗师不可避免地会有共情失败 → 适时讨论和修复 → 来访者将治疗师视为独立的、有缺陷但基本良好的他人 → **转变内化（transmuting internalization）**
- 最终目标：来访者能够为自己提供曾经需要治疗师提供的功能——自我确认、自我安抚、自我验证

## 第 18 章：依恋（Attachment）

### 基本理念

**依恋理论（Attachment Theory）**始于 John Bowlby 的基础工作。Mary Ainsworth 的"陌生情境"（Strange Situation）实验将依恋模式分类为安全型和不安全型。成人依恋类型可被分类为安全型或三种不安全型——这些类型与儿童期的依恋类型有对应关系，并被认为由儿童气质、父母依恋模式、亲子互动和环境共同塑造。

### 成人依恋类型

| 成人类型 | 特征 | 童年对应 |
|----------|------|----------|
| **安全型（Secure）** | 容易回忆关系经历、能整合痛苦记忆、三维地看待他人、能理解他人情感视角；对亲近和依赖都舒适 | 安全依恋 |
| **回避/忽视型（Dismissive/Avoidant）** | 情感过度调节、僵化；忽视依恋价值、记住很少的童年关系；可能理想化他人；表面强大独立，实则无法面对早期失望 | 不安全-回避型 |
| **焦虑/专注型（Preoccupied/Anxious）** | 为关系问题责备自己、理想化早期照料者；焦虑、担忧关系和他人如何看待自己；难以有组织地讲述过去；在成人关系中追求高度亲密、非常依赖 | 不安全-矛盾型 |
| **未解决/混乱型（Unresolved/Disorganized）** | 讨论丧失/创伤时出现思维紊乱；可能发呆、进入恍惚状态；在关系中行为不稳定、令人困惑 | 不安全-紊乱型 |

### 依恋的代际传递（Intergenerational Transmission）

- 每种成人依恋类型倾向于有相关类型的子女
- **关键机制**：照料者的**心智化（mentalization）**能力——理解和处理孩子情感体验的能力
  - 能心智化的照料者 → 孩子发展出**情感调节（affect regulation）**能力和安全依恋
  - 不能心智化的照料者 → 孩子无法发展情感调节 → 不安全依恋

### 依恋理念 vs 客体关系理念

两者都关注早期关系的影响，但侧重点不同：
- **客体关系理念**：关注"关系模板"的内容——你期望关系中是什么样的
- **依恋理念**：关注"关系中的安全感"——你在关系中的安全/焦虑程度
- 实践中两者常互补使用

### 适用场景

- **自我调节和情感管理问题**：在丧失、分离和人生转折时凸显
- **共情和心智化困难**：无法考虑或回应他人的情感需求（尤其明显的在育儿情境中）

### 治疗方向

- **觉察依恋模式**：来访者与治疗师一起观察和识别依恋类型 → 创建新的自我叙事
- **发展新依恋模式**：治疗关系本身提供矫正性依恋体验
  - 特别是对紊乱型依恋：治疗师帮助来访者在冲突时刻**心智化**——"你能想象其他方式来看待刚才发生的事情吗？"
  - 反复体验 → 来访者逐渐内化 → 最终改变依恋类型
- 治疗关系提供的安全感 → 来访者感到足够安全去考虑管理情绪的新方式

## Putting it Together：完整 LINK 示例（Mr C）

**个案简介**：28岁平面设计师，因与交往6个月的女友争吵后抑郁就诊。害怕被抛弃、过度联系女友，被威胁报警后才离开。

**聚焦 DESCRIBE**：最大困难在**关系领域**——不安全的关系、缺乏信任、快速但肤浅的亲密、缺乏共情、害怕被抛弃

**聚焦 REVIEW**：
- **遗传/产前**：母亲有未治疗的焦虑（可能是气质性焦虑）
- **婴幼儿期**：家庭暴力、母亲焦虑且分心（无法形成安全依恋）
- **童年中期**：父亲离开后与母亲"过于亲近"、被带入母亲床铺、后来被母亲的新男友突然取代
- **童年晚期**：继续不安全—焦虑的母子关系

**LINK（使用依恋 + 早期认知/情绪困难理念）**：

> Mr C 在关系领域有显著困难，表现为不安全的关系模式、缺乏信任、快速但肤浅的亲密以及对被抛弃的极度恐惧。他可能生来就有气质性焦虑（焦虑的家族史），并且早期生活在暴力环境中。母亲焦虑且分心，因此无法为他提供情感调谐——这导致他无法发展出安全依恋，也无法发展对自己的情感状态或他人情感状态的感知能力。这可以解释他在成人关系中的共情和心智化困难。后来，父亲离开后他与母亲过于亲近，之后又突然被母亲的新男友取代——这加剧了他的不安全感和对被抛弃的恐惧。在成人关系中，他表现出过度亲近需求和极度害怕被抛弃的模式。

**治疗方向**：帮助 Mr C 在治疗关系中发展更安全的依恋模式，通过治疗师的一致性、可靠性和心智化能力，为他提供矫正性的依恋体验。

## Agent 使用建议

- **适用场景**：当 agent 需要将 DESCRIBE 和 REVIEW 的材料转化为因果假设，且来访者的问题与关系模式、自尊发展或依恋安全相关时使用本文件。
- **可直接引用**：客体关系模板形成机制、自体心理学镜映失败分类、成人依恋类型对照表、六种理念速查表。
- **选择理念的策略**：
  - 关系模式反复 → 客体关系理念（关系模板）
  - 自尊/共情/嫉妒问题 → 自体心理学理念（镜映/理想化）
  - 情感调节困难 + 丧失/分离反应 → 依恋理念
  - **可组合使用**（如依恋 + 客体关系在关系问题中常互补）
- **需要结合**：
  - 使用前先读取 `_01_03_总论与三步法.md` 理解 LINK 的方法论
  - 与 `_13_15_创伤困难与冲突防御.md` 中的前三种理念组合使用
  - 完整的 formulation 写完后参考 `_19_22_临床应用与实践.md` 了解如何在不同场景中应用
- **注意**：客体关系理念和依恋理念都关注早期关系但侧重点不同——前者关注"模板内容"，后者关注"安全感"。书写 formulation 时务必使用假设性措辞。
