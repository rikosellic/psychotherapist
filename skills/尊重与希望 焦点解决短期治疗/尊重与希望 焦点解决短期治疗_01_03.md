## 第 1 章 范式转移的代表：焦点解决短期治疗的兴起与咨询哲学

### 章节核心

本章说明 SFBT 的兴起、哲学背景和与传统问题焦点咨询的差异。SFBT 是短期治疗、后现代取向、社会建构论取向的代表之一，源自家族治疗传统，由 Steve de Shazer、Insoo Kim Berg 等人在观察有效治疗互动的过程中发展出来。

原书强调：SFBT 晤谈以现在与未来为导向，主要任务是探讨当事人所欲的未来，以及为了达成想要的不同需要做什么，而不是以过去的问题或现在的冲突为主要核心。SFBT 是一种“希望与尊重的实用主义”。

### 关键观点

**社会建构论影响。** SFBT 不把知识、问题和解决之道视为唯一客观事实，而强调意义是在社会互动和语言中共同建构的。当事人的主观现实、语言、生活脉络和价值观具有核心地位。

**当事人是自身生命的专家。** 咨询师不应高估自己的专家角色，而要把当事人放在优势位置。解决之道不是咨询师传授给当事人的标准答案，而是在合作对话中由当事人根据自己的生活经验建构出来。

**问题语言不同于解决语言。** 原书区分问题焦点和解决焦点：问题焦点重抱怨、过去、缺点、僵局、复杂化、咨询师知道最多；解决焦点重愿景与目标、未来、优势、资源、第一小步、进展、双方合作。

**谈论解决之道会扩大正向知觉。** 当事人刚开始常只看到问题。SFBT 通过目标、资源、优势、力量、进展的探讨，让当事人看到更多“白”的部分。问题本身可能暂时没有变化，但当事人对问题的知觉比例和定义会改变。

### 方法提取

第 1 章适合 agent 用来建立 SFBT 的基本判断：

- 不急于追问“为什么会这样”，而问“你希望有什么不同”。
- 不急于分析病理和缺陷，而问“什么时候情况没有那么糟”。
- 不以咨询师语言替换当事人语言，而使用当事人自己的目标词汇。
- 不把目标设为“问题消失”，而设为“生活中会多出什么可观察的不同”。

可用问句：

- “如果这次谈话对你有帮助，你希望带走什么？”
- “你希望未来有什么不同？”
- “当情况稍微好一点时，你会最先注意到什么？”
- “过去有没有哪怕一点点接近你想要的样子？”
- “哪些资源、支持或做法已经在帮你撑住？”

### 案例线索

原书以夫妻争吵为例说明 SFBT 的焦点：不是只研究争吵如何形成，而是询问没有吵架时如何沟通、争吵较轻微时有什么不同、争吵中双方共同在乎什么、争吵如何可能对婚姻有正面贡献、过去如何处理好争吵结尾。这样做会让当事人看见沟通中的可用经验和正向循环。

### Agent 使用提示

当用户要求“分析问题原因”时，SFBT 取向可以先承认原因可能重要，但应把回答导向：

```text
问题如何形成可以帮助理解处境；但如果按 SFBT 工作，下一步更关键的是：
1. 你希望生活具体多出什么不同？
2. 什么时候这个问题没有完全控制你？
3. 你已经做了什么让情况没有更糟？
4. 下一小步怎样能让你靠近想要的方向？
```

## 第 2 章 尊重的赋能：焦点解决短期治疗的人性观与专业价值

### 章节核心

本章说明 SFBT 的人性观和专业价值。原书指出，SFBT 相信“复原”开始于当事人愿意来谈、愿意改变的那一刻。人虽然会受过去经验影响，但不会只被过去或心理疾病局限；无论何时，都可以创造自己想要的未来。

SFBT 的尊重不是礼貌，而是治疗性立场：相信当事人有资源、有能力、有权决定主题和目标；咨询师通过尊重、一般化、同步、监测改变和可能性思维，促进当事人自我赋能。

### 关键观点

**复原从愿意来谈开始。** 当事人愿意出现、愿意讲一点、愿意表达不满，都是改变的起点。即使是被转介或非自愿个案，也可寻找其最低限度的在乎和愿意。

**当事人拥有解决问题与发现解决之道的能力。** 咨询师应聚焦当事人的力量、应对技巧、自我决定，而非病理和缺陷。

**尊重当事人吐露的程度和速度。** SFBT 不要求当事人深入揭露所有痛苦历史；当事人可以决定为了建构解决之道，需要谈哪些内容。

**一般化与暂时性语言。** 咨询师会说明在此处境中出现类似情绪和反应是可以理解的，并使用“目前”“这一阶段”“暂时”等语言，提醒经验会变化。

**改变一直在发生。** SFBT 认为问题、目标和成功的定义都可能随晤谈变化。小改变很重要，有效就多做一点，无效就做点不同。

### 方法提取

第 2 章可形成 SFBT 的回应姿态：

- 用尊重而非诊断压倒当事人。
- 用一般化降低羞耻。
- 用应对问句离开受害者位置。
- 用暂时性语言降低问题固定化。
- 用改变监测帮助当事人看到进展。

可用表达：

- “在这样的处境中，你有这种反应是可以理解的。”
- “目前你是这样感受，并不代表它会永远如此。”
- “你愿意来谈这件事，本身已经说明有一部分你希望事情不同。”
- “你怎么让情况没有变得更糟？”
- “如果这件事曾经有一点点不同，那时发生了什么？”

### 案例线索

拒学青少年案例中，咨询师没有把“不上学”直接解释成叛逆或问题，而是以尊重、好奇、不批判的姿态问：“你一定有一个重要的理由，才会选择不去学校？”当青少年说父母只关心功课，咨询师回应其在乎：“你很希望爸妈能有不同的看待你、对待你的态度。”这把愤怒转向目标和关系期待。

案例还展示了一般化、应对和评量：咨询师承认许多同学也会生气，询问他生气时如何让自己好过一点，随后使用奇迹问句和评量问句探索父母理解他的程度。

### Agent 使用提示

处理非自愿、抗拒、愤怒个案时，优先找“行为背后的在乎”：

```text
你这么生气，似乎说明这件事对你很重要。
如果不是只看这个行为本身，而是看它想表达什么，它可能在表达什么需要？
如果别人真的懂了你的需要，他们会做些什么让你知道？
```

## 第 3 章 合作理解的基础：焦点解决短期治疗的咨询关系

### 章节核心

本章说明 SFBT 咨询关系的基础是合作理解。咨询师对当事人怀有真诚和好奇，接收当事人的各种知觉，并展现愿意倾听的姿态。SFBT 咨询师的好奇心以当事人的知觉为焦点，并朝建构解决之道的方向前进。

### 关键观点

**咨询关系是共同建构。** 治疗师不是上位专家，当事人也不是被修理的对象。双方通过语言、问句和回应共同创造对问题和解决之道的新理解。

**未知态度是关系基础。** 咨询师不预设已经知道当事人的意义，而是持续确认：“我这样理解接近吗？”“对你来说这代表什么？”

**好奇心有方向。** SFBT 不是无限探索问题细节，而是好奇当事人在乎什么、想要什么、何处有例外、如何应对、下一步可能是什么。

**合作不是咨询师替当事人做决定。** 咨询师要向当事人“学习、求教”如何协助他，帮助当事人参与决定主题和目标。

### 方法提取

建立合作关系时可用：

- “今天你觉得谈什么会最有帮助？”
- “别人可能希望你改变一些事，那你自己希望今天不白来，最好发生什么？”
- “我想确认我有没有理解对。”
- “你愿意让我知道哪些背景，就足够我们开始往前走？”
- “这件事对你最重要的部分是什么？”

### 案例线索

拒学青少年案例展示了合作理解的建立：咨询师先谈其音乐爱好，赞美其品味和音乐素养；随后理解其拒学行为背后的理由，而不是马上说服他返校。咨询师不断跟随其语言，如“看见我的价值”“真正关心我这个人”，并用这些词继续追问愿景和分数。

### Agent 使用提示

SFBT 风格输出应避免：

- “你真正的问题是……”
- “你需要认识到……”
- “你应该……”

更合适：

```text
从你的描述看，你似乎很在乎……
如果用你的语言来说，你希望变成……
我不确定这样理解是否贴近你：……
如果贴近，我们可以再看下一小步是什么。
```
