## 第 4 章 合作对话的重要元素：建构解决之道的晤谈阶段

### 章节核心

本章说明 SFBT 晤谈如何通过合作对话建构解决之道。原书指出，SFBT 的代表性问句促发当事人通过个人知觉中的过去例外经验、现在目标及未来可能性，逐步形成属于自己的解决之道。

SFBT 晤谈不是漫谈，也不是机械技术堆叠，而是围绕“当事人所欲不同”组织对话。咨询师在每一阶段都要维持尊重、合作、未来导向和资源导向。

### 关键阶段

**开场与合作。** 理解当事人为什么来，确认他希望本次谈话有什么帮助。被转介个案要区分别人的期待和当事人自己的期待。

**目标形塑。** 把“不要什么”转化为“想要什么不同”。目标越具体、越行为化，后续越容易推进。

**例外与资源探索。** 找出问题较少、目标已经部分发生、当事人曾经成功应对的时刻。

**代表性问句。** 使用奇迹问句、评量问句、关系问句、应对问句等，扩大当事人的知觉。

**暂停与反馈。** 咨询师整理当事人语言中的资源、目标和可能行动，以赞美、桥梁、建议形式反馈。

**后续推进。** 追踪进展，深究改变如何发生，扩大和维持小改变。

### 方法提取

初次晤谈可按此流程：

```text
1. 今天谈什么对你有帮助？
2. 如果谈话有用，你离开时会带走什么？
3. 当情况变好，你会最先注意到什么？
4. 过去有没有一点点接近这个状态？
5. 你是怎么做到的？
6. 从 1 到 10 分，现在在哪里？
7. 下一小步是什么？
```

### 案例线索

原书多处案例都体现“问题式谈话 -> 未来式谈话 -> 策略式谈话”。咨询师不是让当事人一直讲问题，而是用问句让其描述想要的生活、已经出现的例外、可继续尝试的行动。

### Agent 使用提示

生成 SFBT 晤谈时，不要只列技能名。要按阶段推进，并让每个问题服务下一步：

- 先建立目标。
- 再找例外。
- 再问分数。
- 最后设计小步。

## 第 5 章 从梦想到现实：焦点解决短期治疗的目标架构

### 章节核心

本章聚焦“目标架构”。SFBT 坚持由当事人的立场、以当事人的语言，确认偏好未来与所欲目标。这也符合短期治疗“知道何时是终点”的理念：一旦知道想去哪里，才能倒回来引导晤谈方向。

### 目标的性质

SFBT 目标不是咨询师认为应该改变什么，而是当事人认为有意义、想要靠近、能观察和行动的不同。

不够 SFBT 的目标：

- “我不要焦虑。”
- “我希望不要吵架。”
- “我想孩子正常。”
- “我想变好。”

更符合 SFBT 的目标：

- “焦虑来时，我能先呼吸三次再决定是否退出。”
- “下次和伴侣谈钱时，我能用平稳声音说完第一句话。”
- “孩子这周愿意有一天走到校门口。”
- “我早上能按时起床并完成一件小事。”

### 目标形塑原则

**正向表述。** 说明要出现什么，而不是只说不要什么。

**具体可观察。** 能被自己或别人看见、听见、记录。

**小步可行。** 目标要在当事人当前能力和意愿范围内。

**当事人语言。** 使用当事人自己的词汇，例如“被看见”“有信任”“不再一开口就吵”。

**关系脉络。** 追问重要他人会注意到什么不同。

### 目标问句

- “你希望通过这次谈话，生活出现哪一点不同？”
- “如果问题没有那么控制你，你会多做什么？”
- “谁会最先发现你不一样？他会看到什么？”
- “你说想被理解，当被理解发生时，对方会做什么？”
- “如果只是好 10%，那会是什么样？”

### 案例线索

拒学青少年案例中，当事人最初表达的是愤怒：“爸妈只关心功课。”咨询师把它转成目标：“你希望他们如何看待你、对待你？”当事人说出“看到我这个人的价值”“真正关心我这个人，而不是只重视面子”。咨询师继续追问行为表现：他们会问今天过得好不好、听他说心情、考试不好时鼓励、考好时肯定努力。

这就是从抱怨到目标，从抽象愿望到可观察行为。

### Agent 使用提示

当用户给出负向目标时，按以下方式转换：

```text
你说“不想再……”，如果把它换成“想要多一点……”，会是什么？
如果这件事真的多了一点，别人会看到你做什么不同？
如果只从现在往前走半步，这半步是什么？
```

## 第 6 章 生命的亮光：焦点解决短期治疗的例外架构

### 章节核心

本章聚焦“例外架构”。原书指出，例外即优势、资源、力量、成功的统称。例外思维是找寻优势、方法与资源的放大镜；例外内容是形塑自我赋能与解决之道的重要素材。

SFBT 认为，问题不会在所有时间、所有地点、所有关系中以同样强度发生。只要存在一点不同，就可以探索其条件、行为、意义和复制可能。

### 例外的类型

- 问题没有发生的时刻。
- 问题较轻的时刻。
- 当事人更能应对的时刻。
- 目标已经部分发生的时刻。
- 别人注意到当事人不同的时刻。
- 情况没有继续恶化的时刻。

### 例外问句

- “什么时候问题没有那么严重？”
- “有没有一点点比较接近你想要的样子？”
- “上一次分数高一点是什么时候？”
- “那时你做了什么？”
- “谁帮了你？”
- “这说明你有什么能力？”
- “如果要让它多发生一点，你可以做什么？”

### 放大例外

找到例外后，不要停在“有过一次”。要继续问：

- 发生在什么时候？
- 发生前你做了什么？
- 当时你怎么想？
- 身体和情绪有什么不同？
- 别人做了什么？
- 你如何回应？
- 为什么那样有用？
- 下次能复制哪一小部分？

### 案例线索

拒学青少年案例中，当事人觉得父母不理解。评量问句显示父母目前在 3 分。咨询师追问“怎么能有 3 分？”当事人想到：生病时父母会关心、会找医生；自己不高兴时父母会停止不说。这些就是关系中的例外。它们让当事人从“完全不被理解”转向“仍有一点关心存在”，也为 3 分到 4 分提供可能。

### Agent 使用提示

例外架构特别适合处理“从来没有”“完全不可能”“一直都这样”的叙事：

```text
如果真的一直都是 0 分，你可能不会撑到现在。
有没有哪怕 1% 的时候，情况没有那么糟？
如果找不到变好的时刻，那是什么让它没有更糟？
```
