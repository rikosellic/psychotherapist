## 第 7 章 身后一步的引导：未知态度问句的使用

### 章节核心

本章强调咨询师处在“身后一步引导”的位置：一方面尝试理解当事人，并确认理解是否正确；另一方面思考如何让当事人在回答问句时引发自我赞美和自我赋能。

### 关键观点

未知态度不是咨询师没有知识，而是不以自己的理论和判断抢先定义当事人。咨询师通过好奇、确认和追问，让当事人解释自己的意义。

“身后一步”包含两个动作：

- 跟随当事人的语言、知觉和在乎。
- 用问句引导当事人朝目标、资源、例外和可能性移动。

### 方法提取

未知态度问句：

- “我这样理解对吗？”
- “对你来说，这句话最重要的意思是什么？”
- “你说的‘被尊重’，具体会是什么样？”
- “如果用你的话来说，你希望发生什么？”
- “你是怎么想到这个答案的？”

引发自我赞美的问句：

- “你怎么能做到这一点？”
- “这说明你有什么能力？”
- “在那么难的情况下，你如何让自己撑住？”

### Agent 使用提示

避免直接解释当事人。先确认，再推进：

```text
我不确定是否理解准确：你真正难受的不是事件本身，而是里面有一种没被看见的感觉。这个理解贴近吗？
如果贴近，你希望别人怎样做，才会让你知道自己被看见？
```

## 第 8 章 好的开始是成功的一半：正向开场的诚挚邀请

### 章节核心

本章说明 SFBT 的开场。开场阶段需要理解、共情、支持当事人的主观知觉与反应，同时大量使用目标形塑的未知态度问句，协助当事人开始知觉生命中的可能性，提高自信，并愿意采取行动改善现状。

### 开场任务

- 建立安全和合作。
- 理解来谈脉络。
- 区分转介者期待和当事人期待。
- 接纳当事人当前情绪。
- 询问本次谈话想要的帮助。
- 开始形塑目标。

### 正向开场问句

- “今天你愿意先从哪里开始？”
- “如果这次谈话有一点帮助，你希望是什么？”
- “你来这里之前，谁最希望你来？那你自己希望今天不要浪费的是什么？”
- “在这么不容易的情况下你仍然来了，这说明什么？”
- “我们今天谈完，你希望自己有什么不同？”

### 关键边界

正向开场不是要求当事人积极，也不是忽略痛苦。正确做法是先承认痛苦，再邀请方向：

“听起来这段时间确实很难。为了让今天对你有帮助，我们可以一边理解这个困难，一边看看你最希望先有哪一点不同。”

### Agent 使用提示

初始访谈不要一开始就问大量病史。可以按：

```text
来谈期待 -> 当事人在乎 -> 所欲不同 -> 具体表现 -> 例外线索
```

## 第 9 章 创造改变的动能：赞美的充分运用

### 章节核心

本章说明赞美在 SFBT 中的治疗作用。原书指出，欣赏与赞美是 SFBT 晤谈的基调。当事人能看到自己的优点、资源、成功时，自尊感会提高，面对问题的困难感和改变恐惧会降低，对问题产生新的正向观点，也更愿意处理困境。

### 赞美的功能

- 增加当事人自我效能感。
- 强化例外和有效行为。
- 让当事人看见自己被忽略的资源。
- 促进合作关系。
- 引发下一步行动动机。

### 赞美类型

**直接赞美。**

- “你能在那么生气时说出真正希望被理解，这很不容易。”
- “你愿意来谈，说明你仍然希望事情不同。”

**间接赞美。**

- “你是怎么做到没有让情况更糟的？”
- “你怎么能想到这个办法？”
- “这说明你具备什么能力？”

间接赞美常更符合 SFBT，因为它让当事人自己说出能力。

### 赞美注意事项

赞美必须具体、真实、贴近当事人经验。空泛赞美会显得敷衍。

不要说：

- “你很棒。”
- “你要加油。”

更好：

- “你已经很累了，却仍然每天把孩子送去学校，这说明你在低能量状态下仍守住了重要责任。”

### Agent 使用提示

生成 SFBT 回应时，每个赞美最好指向一个具体行为：

```text
我注意到你……
这说明你……
你是怎么做到的？
```

## 第 10 章 拾起散落的光亮：一般化与重新建构的精髓掌握

### 章节核心

本章说明一般化与重新建构。原书指出，它们是 SFBT 晤谈的重要形塑技巧，体现常态化、正常化、去病理化的接纳与正向框架，也考验咨询师对不同生命阶段当事人的理解。

### 一般化

一般化是把当事人的情绪和反应放回生活脉络中，让其知道自己的反应可以理解。

示例：

- “在长期被误解的处境中，很多人都会愤怒。”
- “经历这么大的失落后，悲伤反复出现是自然的。”
- “当孩子觉得只有成绩被看见时，用不上学表达抗议，也许是他能想到的方式。”

一般化降低羞耻，让当事人愿意继续探索。

### 重新建构

重新建构不是强行正向，而是在问题行为或负向情绪中看见在乎、价值、保护功能和可能资源。

示例：

- 愤怒可重新建构为“你很在乎这段关系中的尊重”。
- 拒学可重新建构为“你在用自己的方式表达希望父母看见你这个人”。
- 担心可重新建构为“你对这件事很重视，希望它不要变坏”。

### 使用步骤

1. 先接住当事人的痛苦。
2. 找出行为或情绪背后的在乎。
3. 用试探性语言表达。
4. 等当事人确认或修正。
5. 连接目标和下一步。

试探性表达：

- “我不知道这样理解是否贴近你……”
- “听起来，这份愤怒背后也有一个很希望被理解的部分。”
- “也许你不是不在乎，而是太在乎了，所以很难受。”

### 案例线索

拒学案例中，咨询师将“我用不上学抗议爸妈”重新建构为“你特别在乎父母的态度”，并进一步探索父母如何看见他的价值。这让当事人从愤怒进入愿景，从抗议进入可讨论的关系目标。

### Agent 使用提示

一般化 + 重新建构组合：

```text
在这样的处境下，你会有这种反应是可以理解的。
同时，这个反应里似乎也有一个很重要的在乎：……
如果这个在乎被更好地看见，你希望对方会做什么不同？
```
