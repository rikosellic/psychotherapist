## Agent 使用建议

如果任务是理解 DBT 的理论、适用范围或课程规划，先读 `_01_DBT技能训练原理`。

如果任务是设计或运行一个技能训练团体，读 `_02_05_技能训练规划与课程设计`，再按目标模块读 `_06_通用技能行为分析` 或 `_07_10_正念人际效能情绪调节与痛苦忍受技能`。

如果任务是分析问题行为或做链锁分析、遗漏环节分析，读 `_06_通用技能行为分析`。如需完整案例参考，该文件包含"对伴侣大吼"和"开会迟到"两个完整分析范例。

如果任务是正念练习、处理人际冲突、情绪调节、危机生存、接纳现实或成瘾冲动，读 `_07_10_正念人际效能情绪调节与痛苦忍受技能`。该文件包含核对事实、相反行为、全然接纳等关键技能的多维度案例。

涉及自杀、自伤、暴力、严重成瘾戒断、精神病性症状、躁狂或医疗危险时，本资料只能作为技能训练参考；应优先使用危机评估、急诊、精神科、专业治疗师或当地危机资源。
