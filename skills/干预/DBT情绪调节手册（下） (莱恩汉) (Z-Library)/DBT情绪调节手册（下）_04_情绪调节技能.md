## 情绪调节技能模块

### 章节主线

- 情绪调节的目标是**减少情绪痛苦，而非摆脱情绪**——情绪有重要功能，完全摆脱既不现实也不可取。
- 四大板块：了解并命名情绪（识别与理解）、改变情绪反应（干预技术）、减少情绪心念易感性（预防策略）、管理极端情绪（危机策略）。
- 改变情绪的三种核心方法：核对事实（Check the Facts）、相反行为（Opposite Action）、问题解决（Problem Solving）——由"情绪是否符合事实"和"按情绪行动是否有效"两个维度决定使用哪种。

### 情绪调节的目标（讲义 1）

- 了解并命名情绪：辨别、了解情绪功能
- 减少不想要的情绪的频率：在第一时间阻断或改变
- 减少情绪心念的易感性：提升情绪复原力和正面情绪
- 减少情绪痛苦：管理极端情绪，不让事情更糟

### 了解并命名情绪

#### 情绪的功能（讲义 3）

1. **激发（并组织）行动**：情绪为行为做准备，节省反应时间，帮助克服障碍
2. **与他人沟通并影响他人**：面部表情、肢体语言、语调传达信息，情绪影响他人
3. **帮助与自己沟通**：情绪反应提供重要信息——直觉（gut feeling）是对情境中重要信息的反应；但注意情绪愈强烈，我们愈相信情绪是事实（如"如果独处让人感到寂寞，那我不该独处""如果我害怕，那一定存在危险"）

#### 造成情绪调节困难的因素（讲义 4）

- 生理因素
- 缺乏技能（不知道如何调节）
- 强化后果（环境让情绪更激烈）
- 情绪化（情绪而非智慧心念控制行为）
- 情绪负担过重（过于激动无法使用技能）
- 情绪误区（错误信念影响调节能力）

#### 关于情绪的误区（讲义 4a）

21 条常见误区，例如：
- "每个情境中总有一个感受是对的"
- "让人知道我现在很难过，那意味着我很软弱"
- "负面情绪是不好的，会摧毁一个人"
- "情绪化意味着我失控了"
- "感受到的才是事实"

#### 描述情绪的模式图（讲义 5）

情绪由复杂反应组成，改变其中任何元素会改变整体：
- 先存脆弱因素 → 诱发事件 → 注意力/知觉 → 对事件的解释（想法/信念） → 神经系统的变化（生理变化/躯体感觉） → 行为冲动 → 表达（面部和肢体语言/言语/行动） → 后续影响（继发情绪等）

#### 描述情绪的方法——10 种特定情绪（讲义 6）

讲义 6 详细列出了 10 种情绪各自的：诱发事件、对事件的解释、生理反应与感受、表达和行为、后续影响。

**愤怒（Anger）**：重要目标遇障碍/被攻击/失去权力→觉得不公、指责→肌肉紧绷、脸发热→攻击、批评、摔门

**厌恶（Disgust）**：看见排泄物/背叛/残酷/被迫接触不想接触的东西→觉得被污染、道德优越→想吐、避开、清洗

**妒羡（Envy）**：别人拥有自己没有的东西→觉得不公、自己不如人→肌肉紧绷、想对方倒霉→努力获取或攻击批评

**恐惧（Fear）**：生命/健康/幸福被威胁→认为会遭伤害或失败→心跳加速、窒息→逃跑、僵住、回避

**快乐（Joy）**：美好惊喜/被爱/达成目标→实事求是→兴奋、有活力→浅笑、拥抱、分享好心情

**嫉妒（Jealousy）**：重要关系受威胁/所爱之人被关注→认为伴侣不忠/自己要失去→呼吸被抑制、心跳加快→施暴、窥探、紧抓不放

**爱（Love）**：想要被满足/共度美好时光→相信对方爱自己→兴奋、温暖、安全→说"我爱你"、触碰、性行为

**悲伤（Sadness）**：无法挽回的人或事/被拒绝→觉得会永远失去或毫无希望→疲惫、空虚、想哭→退缩、回避、哭泣

**羞愧（Shame）**：被心仪之人拒绝/做错事被发现→觉得自己有缺陷/不够好→胃痛、想躲起来→躲避、低头、反复道歉

**内疚（Guilt）**：做/想自己觉得不对的事→觉得自己该被谴责→发热、紧张→希望补偿、道歉、忏悔

### 改变情绪反应

#### 三种改变情绪的技能概览（讲义 7）

- **核对事实（Check the Facts）**：检查情绪反应是否符合事实，改变信念和假设
- **相反行为（Opposite Action）**：情绪不符合事实或行动无效时，采取与情绪冲动相反的行为
- **问题解决（Problem Solving）**：事实本身就存在问题时，解决问题减少负面情绪

#### 核对事实（讲义 8、8a）

**核心原理**：很多时候是对事件的解释（而非事件本身）引发情绪和行为。事件→看法→情绪，但情绪也会影响看法。

**6 步骤**：
1. 我想改变哪种情绪？
2. 具体哪件事激发了情绪反应？——描述事实，不加评判
3. 我对这件事的解释、看法和假设是什么？——想想其他可能的解释
4. 我的假设可能暗含威胁吗？——评估威胁事件发生的概率
5. 最糟糕的结果是什么？——想象真实场景和应对方式
6. 我的情绪/反应强度是否符合事实？

**符合事实的情绪举例**：
- **恐惧**：生命/健康/幸福受到威胁
- **愤怒**：重要目标被迫中断、被攻击侮辱
- **悲伤**：永远失去了某人某物
- **羞愧**：被在乎的人或团体拒绝且被公之于众
- **内疚**：行为与价值观或道德规范相背离

#### 相反行为 vs. 问题解决——决策流程（讲义 9）

```
情绪是否符合事实？
├── 是 → 按情绪冲动行动有效吗？
│   ├── 是 → 按情绪冲动行动，坦然接纳后果
│   └── 否 → 相反行为
└── 否 → 按情绪冲动行动有效吗？
    ├── 是 → 改变想法以适应现实 / 问题解决
    └── 否 → 相反行为
```

#### 相反行为（讲义 10、11）

**7 步骤**：
1. 辨识并给想改变的情绪命名
2. 核对事实——情绪是否符合事实
3. 辨别与描述行为冲动
4. 询问智慧心念：按此冲动行动有效吗？
5. 如果不符合事实或行动无效——弄清相反行为
6. 采取相反行为
7. 持续直到情绪改变

**针对 9 种情绪的相反行为要点**：

| 情绪 | 行为冲动 | 相反行为要点 |
|:---|:---|:---|
| 恐惧 | 逃跑/回避 | 靠近、接近恐惧的事物、自信姿势、调节呼吸 |
| 愤怒 | 攻击 | 温和回避、暂时离开、共情对方、浅笑、掌心向上 |
| 厌恶 | 推开/避开 | 试着体验、理解对方、接触厌恶感 |
| 妒羡 | 毁坏/攻击 | 远离毁坏、列出拥有的、核对事实 |
| 嫉妒 | 控制/窥探 | 放手、分享、不再刺探、关注感受 |
| 爱（无效时） | 靠近/依恋 | 远离、转移注意力、提醒为何不合理 |
| 悲伤 | 退缩/自闭 | 主动靠近、做胜任的事、愉快活动、正视当下 |
| 羞愧（无效时） | 躲藏/回避 | 公开行为（在安全的人面前）、不再道歉 |
| 羞愧（有内疚时） | — | 坦诚、道歉、修复、原谅自己、放下 |
| 内疚（无效时） | 弥补/道歉 | 重复行为（在安全的人面前）、绝不道歉 |
| 内疚（有羞愧时） | — | 隐藏行为、人际效能、加入新团体 |

#### 问题解决（讲义 12）

**7 步骤**：
1. 观察与描述问题情境
2. 核对事实——确保正确理解问题
3. 辨识解决问题的目标
4. 头脑风暴大量解决方案（不批评）
5. 选择符合目标又可行的方案
6. 将方案付诸行动
7. 评估成果——无效则回到步骤 5

### 减少情绪心念的易感性：ABC PLEASE（讲义 14-20）

**A——积累正面情绪（Accumulate Positive Emotions）**

**短期（讲义 15）**：
- 增加产生积极情绪的愉快事件
- 对正面体验保持觉察力
- 不再关注担忧

**愉快事件清单（讲义 16）**：共 225 项活动，包括听音乐、看电影、散步、与朋友促膝长谈、享受美食、做瑜伽、跳舞、与宠物玩等。

**长期——建立值得过的人生（讲义 17-18）**：
- 7 步骤：避免回避→确认重要价值观→确认当前可执行的价值观→确认相关目标→选择当前可执行目标→确认行动步骤→立即行动
- **价值观分类**（讲义 18）：14 类 58 项，包括处理人际关系、成为集体的一分子、有影响力、实现生命目标、生活快乐满足、尊重自己、以自我为导向、有灵性、有安全感、致力于自我发展等

**B——建立自我掌控（Build Mastery）（讲义 19）**
- 每天做一件有意义的事累积成就感
- 以成功为宗旨计划，做有难度但可行的事
- 随时间的推移逐步增加难度
- 主动面对挑战

**C——提前应对（Cope Ahead）（讲义 19）**
1. 详细描述可能有问题的情境
2. 找出适应方法和应对技能
3. 在心中详细描述情境
4. 心中预演一遍
5. 演练后练习放松

**PLEASE——照顾身体（讲义 20）**
- **P**（PhysicaL illness）：治疗身体疾病
- **E**（balancing Eating）：均衡饮食
- **A**（avoiding mood-Altering substances）：远离改变情绪的物质
- **S**（balancing Sleep）：均衡睡眠（7-9 小时）
- **E**（getting Exercise）：适当运动（每天 20 分钟）

**噩梦应对（讲义 20a）**：放松准备→找出反复噩梦→写出噩梦→改写噩梦结局→睡前演练

**睡眠卫生指南（讲义 20b）**：规律作息、避免咖啡因/尼古丁/酒精、保持黑暗安静凉爽环境、如睡不着可起身读书、用 9-0 冥想练习等

### 管理极端情绪（讲义 21-24）

**对当下的情绪保持正念（讲义 22）**
- 退一步关注情绪，感受情绪如波浪来去
- 不压制、不夸大、不推开情绪
- 运用身体感觉练习正念
- 记住：你≠你的情绪
- 全然接纳情绪

**技能崩溃点（Skills Breakdown Point）（讲义 23）**
- 感觉非常痛苦、受不了、要失控
- 大脑乱哄哄，无法使用复杂技能
- 此时应：使用危机生存技能（TIP/转移注意力/自我安抚/改善当下）→对情绪保持正念→再尝试其他情绪调节技能

**情绪调节技能疑难解答（讲义 24）**
依次检查：生理易感性→技能是否正确→强化物（情绪是否在传递信息）→心情（是否有时间精力）→情绪超负荷→情绪误区

### 情绪调节技能复习图（讲义 25）

```
命名情绪 → 描述情绪 → 改变情绪反应（相反行为/核对事实/问题解决）
                            ↓
                   对当下情绪保持正念
                            ↓
                 危机生存策略（降低情绪反应和强度）
                            ↓
           减少情绪易感性（ABC PLEASE：积累/建立/提前应对/PLEASE）
```

### Agent 使用建议

- **适用场景**：当 agent 需要为来访者设计情绪调节方案、分析特定情绪的功能与触发因素、选择具体干预策略时使用。
- **可直接引用**：核对事实的 6 步骤作为"认知重新评估"的标准化流程；相反行为的决策流程图作为"是否改变情绪"的操作指南；ABC PLEASE 作为"提升情绪复原力"的综合方案。
- **需要结合**：`02_正念技能.md` 中的观察/描述技能用于情绪识别；`05_痛苦忍受技能.md` 中的危机生存技能用于技能崩溃点时的应急干预；`01_通用技能.md` 中的生物社会理论用于理解情绪失调的成因。
- **注意**：情绪调节的第一步是"了解并命名情绪"——如果来访者无法识别自身情绪，任何高级干预技术都难以生效。agent 应先确保情绪识别基础，再进入改变策略。
