# 团体心理治疗：理论与实践(note)_02_治疗师任务与此时此地

来源：`book/团体心理治疗：理论与实践(note).md`
抽取范围：治疗师的基本任务、团体文化建立、发展自我监督团体、促进自我暴露、成员互助、此时此地的两层面模型、激活技术、进程评论与反馈、责任意志与改变、对过去的使用、团体整体进程评论。
用途：供 agent 获取团体带领者的核心技术——如何建立团体文化、激活此时此地的互动、做出有效的进程评论和反馈，以及促进成员的改变。

## 章节主线

- 治疗师在团体治疗中承担三重基本任务：创立与维系团体、建立团体文化、激活"此时此地"并予以阐释。三者缺一不可。
- 此时此地的运作需要两个层面配合：体验层面（成员真实地对彼此产生感受）和进程阐明层面（团体退一步理解互动的意义）。只有体验没有反思，影响稍纵即逝；只有反思没有体验，团体变得枯燥。
- 治疗师的干预应在"太少"和"太多"之间保持平衡——带领者活跃度与疗效呈倒 U 型曲线。

## 治疗师的基本任务

### 一、团体的创立与维系

- 治疗师是团体的守门人，需要防止成员脱落。如有脱落，需补充新成员直至人数达到理想状态。
- 好的团体通常能对相同主题进行连续讨论，而不是每次都被第一个讲话的人带走。
- **团体主题优先级**：新成员的融入和团体的维系必须优先于其他任务。

### 二、团体文化的建立

**团体规范（norms）**

- 团体文化不是自然形成的，而是治疗师通过说明、示范和强化塑造的。
- 重要规范包括：积极参与会谈、不带评判地接纳他人、尊重和保护团体工作、大量自我暴露、渴求自我了解、描述当下体验、诚实和自发地表达。

**治疗师的双重角色**

- **技术专家（technical expert）**：在早期会谈中明确强调团体规范，抓住机会突出和阐明规范。
  - 言语与非言语方式：点头、微笑、身体前倾、发出"嗯"的声音→强化有益行为；不做评论、无动于衷、转移注意力、抬起眉毛→减少无益行为。
- **示范性参与者（model participant）**：当成员看到治疗师能自由参与互动而未造成负面结果时，他们会有勇气改变自己的行为。
  - 治疗师应示范人际间真诚、自发的沟通，同时谨记成员当下的需求。
  - Eg: ① 承认失误："我一直在回想上次会谈，我希望当时我能理解这一点，这样我会用不同的方式来处理。"
  - Eg: ② 肯定成员的成长："过去你既想分享又害怕这么做，一定很不容易。"
  - Eg: ③ 面对蔑视团体的成员："你知道我喜欢你什么地方吗？是你的恐惧和缺少信心。"

**治疗师需要避免的两种极端姿态**

- 躲在完美的、隐藏自我的专业角色后——这会传递"你做任何事都无法触碰或伤害我"的信息。
- 放弃带领者职责，过度成为团体成员的一部分——回避带领者的固有焦虑和责任。

**促进自我暴露**

- 强调来访者必须自我暴露：假如要从治疗中获益，早晚必须与其他成员分享内心最隐秘的部分。
- 同时让成员知道：每个人的分享可以按照自己的步调，等对团体足够信任时再分享。
- 成员分享秘密后，治疗师的处理重点：
  - 探索**过程**而非只处理内容："你每周前来却对秘密保持沉默，有什么感觉？"
  - 探索**之前的恐惧**："你之前担心团体会如何反应？今天是什么让你决定说出来？"
  - 如果团体过度催促，保护成员："团体似乎很急切，但他还没有足够的安全感去表达。"
  - 如果暴露引发冲突，制止破坏性同时指出："团体中发生了非常重要的事情。"——自我暴露是安全的。
- 自我暴露后治疗师要警惕：是否因自己的反移情与其他成员合谋，共同攻击分享秘密的成员。

**成员成为彼此的帮助者**

- 功能良好的团体不是治疗师一人帮助所有人，而是成员也成为彼此的帮助者。
- 当成员直接问治疗师对自己的看法时，可回应："这个房间里有很多人都非常了解你，为什么不问问他们？"
- 也可问："回忆过去 45 分钟，谁的评论对你最有帮助？谁的最没有帮助？"
- 这样能激活利他主义、现实反馈和团体凝聚力，减少成员对治疗师的过度依赖。

**发展自我监督团体**

- 早期就需要让成员为自身的运作承担责任。
- 可在会谈中途或结束前打断，引导成员反思过程："今天的会谈进行得怎么样？你们满意吗？哪些部分吸引你？哪些最不吸引你？"
- 如果成员反馈大部分时间在闲聊，追问："那么，你为什么让它继续呢？怎样才能让这种情况停止？"

### 三、主要任务与次级满足

- **主要任务（primary task）**：解除痛苦，与他人建立更好的关系，生活得更充实完整。
- **次级满足（secondary gain）**：获得特殊地位、塑造最有魅力/最智慧/最优越的形象、与某个成员形成特殊联盟等。次级满足的吸引力有时会超过完成主要任务。
- 治疗师可问自己：来访者是否正在为其主要任务工作？
- 最有效的做法是提醒成员他们的主要任务是什么。

## 此时此地（Here-and-Now）与进程干预

### 此时此地的两个层面

**第一层——体验性部分**

- 团体成员生活在此时此地，对其他成员、治疗师和团体产生强烈的感受。
- 如果只有这一层，成员会深度投入、情绪表达激烈，在会谈结束后赞叹"这真是强大的体验"——但这是稍纵即逝的。

**第二层——进程阐明（process illumination）**

- 团体退一步，理解刚刚发生的互动的意义。
- 如果缺少这一层，此时此地的做法不会产生持久效用。
- 如果只有进程审视而没有体验，团体将变成内容枯燥的理性训练。

**内容（content）与进程（process）的区别**

- 内容：口头述说的字句、实质话题和提出的论点。
- 进程：互动的个体（包括成员和治疗师）之间的关系性质——**为什么**来访者会**在某个时间**，**以某种方式**对**某个人**说这样的话。
- 理解进程的简单方式：问"到目前为止，你们每个人在这次会谈中的体验如何？"
- 进程评论在日常生活常被视为粗鲁或有侵犯性；但在团体中，这是促进改变的核心工具。

### 激活此时此地的技术

**治疗师的任务**

- 把团体焦点从外部转向团体内部，由抽象转向具体，由普遍转向个人，由个人转向人际互动。

**第一阶段——激活期**

- 首次会谈示例：成员已分享了许多关于自己和痛苦的事情，但还有更重要的事情正在发生——你们在评估彼此，形成对彼此的印象。现在可以花一些时间讨论每个人目前的想法。

**贯穿团体带领过程的技术**

- **开始时聚焦于正向互动**：
  - "在团体中，你感觉谁最温暖？"
  - "谁最与你相似？"
  - "你最钦佩和羡慕他哪一点？哪一点让你最难以接受？"

- **询问具体化问题**：
  - "我做了什么让你对我置之不理？"
  - "我身上的哪方面让你喜欢或不喜欢？"
  - "当你诚实地面对自己的感受时，我感觉与你更亲近；而当你毫无感情地分析别人时，我感觉离你很远。"

- **面对团体沉默**：
  - "这段静默中有许多宝贵信息等待发掘。每个人能否分享一下，自己想说但没有说的是什么？"

- **邀请反馈会谈中有未言明的东西**：
  - "现在是 6 点，还剩半小时。想象会谈已经结束，你们正在回家路上，今天的会谈有什么让你们感觉失望的地方？"

- **同时表达正负面**：
  - "我很高兴你能自在地参与，但你所说的话太抽象、离你太遥远，使别人很难回应。"

- **避免"背后议论"缺席成员**：
  - 可以讨论缺席成员，但要在下一次会谈中与该成员分享讨论内容——这不会让缺席者成为替罪羊。

**持续引导进入此时此地**

- 治疗师需要持续不断地打断团体对"彼时彼地"的讨论，引导进入此时此地的互动。
- 能顶着团体压力知难而进，既深化可能完成的工作，又展示带领者承担风险和克服焦虑的能力。

**带领者的活跃度**

- 呈倒 U 型曲线：做得太少，团体成员不知所措；做得太多，团体缺乏独立性，成员过分依赖带领者。

### 进程评论与反馈

**进程评论的推论层级**（从低风险到高风险）

1. **单一行为的观察**："你在打断我。""你说话声音紧绷，一直在攥紧拳头。"
2. **行为引发的感受**："当你那样做时，我感觉受到了威胁，有时甚至害怕。"
3. **推测行为的动机**（需要信任关系）："我想知道你是否觉得自己缺乏竞争力，因此试图贬低我。"
4. **一段时间内的行为模式**："我注意到你对团体中所有男性都做了相同的事。"
5. **指出不恰当重复的模式**："我认为你太在乎自己对女性的吸引力，以至于将男性都视为竞争对手。"
6. **包含推论性的内容**："你描述了一个我们俩打斗、然后拥抱着跌倒在地的梦——我认为你其实很想接近我。"
7. **将行为与深层情感连接**："你在这里很孤独，感觉被忽略和缺乏关爱，这勾起了无价值感。"
8. **指出此时此地与团体外情境的相似之处**："你当初的目标之一是了解自己为何没有同性好友，但现在你疏远了这里所有男性，你对此有何感想？"

如果只使用高推论性内容（6-8），来访者会处于无所适从的位置，削弱团体治疗的合作本质。

**反馈的基本原则**

- **焦点是行为，不是人格特征**——任何时候都不能贬低或批评来访者的整体人格。
- **反馈的四步组织法**：
  1. 你的行为在他人眼中的样子——成员学会从别人视角观察自己。
  2. 你的行为带给他人的感受。
  3. 你的行为影响他人对你的看法（重视、讨厌、不悦、尊重、回避等）。
  4. 基于上述信息，成员形成自我评价（自我价值、被人喜爱的程度）。
- 随后询问改变动机："你对自己所做的事情和你创造的环境满意吗？"

**支持性的、不带责难的方式**

- 使用"我"语句而非"你"指责。
- 避免标签化用语（依赖、自恋、剥削、自大）——成员有充分理由拒绝这些标签。
- 在提出高风险推论之前，先建立相互信任的支持关系。

### 沉默、抽象和冲突的处理

**沉默**

- 沉默常包含大量未表达内容。治疗师应主动打开："这段静默中有很多宝贵信息，每个人能否分享一下自己想说但没有说的话？"

**抽象与理智化**

- 当成员过度抽象或理智化，要具体化：谁、何时、做了什么、你有什么感觉、对方有什么反应。
- 可同时表达正负两面，例如肯定成员能参与，同时指出说法太抽象使别人难以回应。

**冲突的化解**

- 被攻击者通常无法承认事实——因为承认事实会被视为背叛了自己，也帮了攻击者的忙。
- 处理方式：用更高的目标吸引他们，或限定指责范围增加接受度。
  - Eg: "我看得出你把自己封闭了起来，否认他所说的一切。但你已敏锐地指出了他言论中的弱点，这样下去你什么都没有得到。"
- 袒露真心的话可为干预发挥作用：
  - Eg: 用社交魅力的成员说："当我这样微笑时，我心中其实很痛苦，别再让我继续这样。"
  - Eg: 用泪水控制团体的成员说："我哭泣时其实很愤怒，我不会崩溃，不要再安慰我。"

**建立契约**

- 当来访者在会谈中特别积极时，可问："你今天工作得很努力，也很愿意接纳反馈。你感觉如何？我们是不是把你逼太紧了？"

### 责任、意志与改变

**意志（will）**

- 意志是发起行为、将意图和决心转化为行动的心灵动力。
- 治疗师无法替成员创造意志，只能协助成员解除限制意志的束缚，提升改变的动力。

**成员需要接受的四个前提**

1. **只有我能够改变我自己创造的世界**——来访者对创造的环境负有责任，也对自身人际环境的转变负有责任。
2. **改变没有危险性**——鼓励来访者迈出一小步，尝试害怕的行为（如对被打断表达生气、对习惯性迟到成员表达指责），消除破坏性猜想。
3. **为了获得真正想要的东西，我必须改变**——帮助来访者面对相互矛盾的欲望，选择满足哪一部分，舍弃无法满足的欲望。
4. **我可以改变，我有力量改变**——"为什么你的行为与自己的最佳利益背道而驰？"对于这一问题的主要治疗手段是诠释。

**诠释的作用**

- 诠释可以使生活中的事件变得有条理、可预期——行为的内在体验不再是未成形的、失控的，而是获得掌控感和自我效能感。
- 诠释的效力与它给来访者提供多少力量感成正比。
- 有经验的治疗师知道，探索和理解的过程可以改变过去的记忆。

**来访者需要对自己的行为诚实**

- 仅有善意的努力是不够的。治疗师要帮助来访者直面其行为违背本身利益的困境。
- 指出阻挠来访者改变的推论性障碍："根据你的假设，好像改变会有重大危险降临在你头上。"

### 对过去的使用

- 影响人类行为的强大的无意识因素绝非仅限于过去。
- **团体治疗中，过去是手段，不是目的**——解释过去是为了理解来访者当前与他人关系中的模式。
- 聚焦于当下：对爱的渴望、笃信唯有凭借特定方式才会获得爱、牺牲自主性、伴随羞耻感——我们只能改变当下和未来。
- 团体不关注过去本身，而关注过去给个体带来了什么：利用过去来更好地理解（和改变）来访者现在与他人互动的方式。

### 团体整体的进程评论

- 使用"团体""我们""我们所有人"作为主语。
- **团体整体的力量是凝聚力**——促进团结和共鸣。
- **团体整体的力量对治疗形成阻碍时**需要干预：移除阻碍整个团体进步的障碍，如充满焦虑的议题和对治疗起反作用的规范。
- **团体遁逃（group flight）**：正常团体功能的退化，例如讨论空泛的社交话题、大部分时间花在无关主题上。带领者需要指出这一现象。
- **判断原则**：非常年轻的团体表现出上述行为不一定是阻抗——可能尚在摸索和了解团体规范。议题过于痛苦时，成员会联合起来讨论"安全"话题以自保。
- 对团体来说，被绕开的议题十分重要，不容回避——需要让成员深入探索阻抗的源头，直面焦虑。
  - Eg: 成员戴着口罩表达无关的互相攻击，治疗师停下来进行进程检视："你们非常珍惜这个团体，因为被剥夺而愤怒；但你们的愤怒又削弱了你们珍视的团体氛围。"
- **团体整体进程的议题，总是优先于范围更窄的人际议题。**

### 治疗师对成员缺席的处理

- 可以讨论缺席成员的感受——这不会让缺席者成为替罪羊，只要在下次会谈中与缺席成员分享当时讨论的内容。

### 治疗师对自身感受的使用

- 单纯地表达感受就足以协助来访者做进一步探索——但感受需要尽可能可靠与准确。
- Eg: "当你说话时，我常常感觉你是一名成熟干练的女性，但有时又觉得你像一个年幼的孩子，在努力取悦每一个人。我不确定这是否准确，想知道这对你是否有意义。"
- **区分客观反移情（objective countertransference）和主观反移情（subjective countertransference）**：
  - 客观反移情：来访者的行为对治疗师和他人产生的人际影响——可作为理解来访者模式的材料。
  - 主观反移情：治疗师独有的反应，体现着治疗师自身经验在关系中的投射。

- **可直接引用**：此时此地的两层面模型、进程评论的推论层级、反馈的四步组织法、责任与改变的四个前提、团体整体进程评论的原则。
