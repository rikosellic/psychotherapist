## 第一章：成为抑郁症的 IPT 治疗师

### IPT 的基本定位

人际心理治疗（Interpersonal Psychotherapy, IPT）是一种以依恋为基础（attachment-based）、聚焦当下人际关系的短程心理治疗。核心假设：当情感依恋受到威胁、破坏或中断时，抑郁症状容易发生或加重。

- **双重焦点**：①将抑郁症状与当前人际事件、人际压力和关系改变连接；②帮助来访者改善沟通、修复关系、发展或利用社会支持以减轻抑郁。
- **不把人格重塑**作为急性期首要目标，**不深入探索移情关系**，聚焦"此时此地"（here and now）的人际问题。

### 三大理论基础

| 理论 | 核心观点 | IPT中的应用 |
|------|----------|-------------|
| **依恋理论**（Attachment Theory） | 痛苦源于未满足的依恋需求；不安全依恋风格导致难以有效表达需求或寻求支持 | 不以改变依恋风格为目标，**以沟通方式和社会支持作为改变靶点** |
| **人际理论**（Interpersonal Theory） | 微观层面的沟通模式影响对方回应；僵化、负面的沟通引发拒绝和疏离 | 帮助来访者识别并调整无效沟通模式 |
| **社会理论**（Social Theory） | 当前社会关系和社会支持本身影响心理痛苦，无需诉诸无意识过程 | 强调开发和利用社会支持网络是IPT的关键目标 |

### 生物-心理-社会/文化模型（Biopsychosocial/Cultural Model）

IPT 将抑郁症放在此模型中理解：遗传易感性、早年经历、依恋类型、当前人际关系、社会支持、文化因素共同影响抑郁风险。临床中**不把IPT等同于"只谈关系"**，而是承认抑郁症的医学疾病属性，同时将人际压力和社会支持作为可操作的治疗入口。

### IPT的四大核心特征

IPT 的核心特征包括：

1. **以人际交往为导向**：不同于CBT聚焦内部认知、精神分析聚焦早期经历，IPT首先关注来访者当前的人际沟通和社会支持。
2. **基于生物-心理-社会/文化模式**：来访者的功能被概念化为气质、个性和依恋风格的产物，叠加生物、文化因素。
3. **具有时限性**：急性期通常8-20次访谈；治疗后应提供维持治疗（Maintenance Treatment）以降低复发风险，而非"终止"（termination）。
4. **治疗师主动非中立**：治疗师是来访者的盟友和倡导者，采取支持性立场；移情信息用于理解来访者治疗外的人际模式，**但不在治疗中直接讨论**。

### 病人角色（Sick Role）与教练类比

- **病人角色**：帮助来访者理解抑郁是一种可治疗的医学疾病（非人格失败），降低自责；同时强调来访者需积极参与治疗。
- **教练类比**：治疗师像游泳教练一样，与来访者一起"下水"，观察问题所在，制订训练方案，激励和陪伴，但**游泳的人必须是来访者自己**。

### IPT 的适应证与边界

IPT 适用于轻中度抑郁，尤其是抑郁症状与人际事件、关系冲突、角色变化、丧失、孤独或社会支持不足明显相关时。孕期抑郁、产后抑郁、老年抑郁、青少年抑郁等也有应用价值。

对重度抑郁、强烈自杀风险、精神病性症状、严重认知损害、严重物质依赖等情况，单一 IPT 往往不足，需要精神科评估、药物、危机干预或联合治疗。

### 抑郁症评估要点

IPT 治疗师需要具备基本抑郁症评估能力。评估内容包括：

- 当前核心症状：情绪低落、兴趣减退、快感缺乏。
- 心理症状群：思维迟缓、消极态度、自我负面评价、自责自罪、自杀意念和行为、焦虑、精神病性症状、疾病自知力。
- 躯体症状群：睡眠障碍、食欲和体重变化、性欲下降、疲乏、疼痛或不适。
- 功能损害：照料自我和他人的能力、人际关系、社会支持、工作学习、风险行为。
- 过去症状和治疗史。
- 创伤经历、躯体疾病、精神障碍家族史。
- 依赖/滥用问题。

心理访谈原则包括：稳步建立关系，多倾听共情；从日常生活进入，降低防御；接纳不同观点；避免诱导暗示；时长和环境适当；灵活循环提问；必要时争取亲友和医生支持；危机即时干预，重症尽快转介。

### IPT治疗师的胜任特征

| IPT特点 | 所需胜任特征 |
|---------|-------------|
| 治疗师的盟友关系及友好态度 | 建立关系的基本态度 |
| 作为来访者拥护者，灌注希望 | 利他性 |
| 结合医学模式 | 专业知识与技能 |
| 明确的时效性与结构性 | 自我控制力 |
| 早期个案概念化 | 人际理解和洞察 |
| 保持干预简练，避免深入移情 | 自我觉察、自我控制力 |
| 保持治疗行为和反射式倾听的平衡 | 弹性 |

## 第二章：抑郁症的IPT操作方案

### 标准IPT三阶段结构

| 阶段 | 次数 | 核心任务 |
|------|------|---------|
| **初始阶段**（Initial Phase） | 第1-3次 | 建立联盟，评估症状与风险，建立人际关系清单，确定问题领域，人际问题概念化，心理教育，统一治疗目标 |
| **中期阶段**（Middle Phase） | 第4-10次 | 聚焦问题领域，沟通分析、角色扮演、决策分析，构建社会支持，减轻症状，改善人际关系 |
| **终止阶段**（Termination Phase） | 第11-12次 | 讨论终止，回顾收获，巩固技能，复发预防，评估是否需要维持治疗或换方案 |

必要时增加**维持治疗阶段**（Maintenance Treatment），降低复发风险。

### 12次简版流程（参考）

- **第1次**：建立联盟，采集病史，评估情绪和安全性（PHQ-9/HAMD），介绍IPT、保密和知情同意，初步梳理人际关系和压力源
- **第2-3次**：建立人际关系清单，确定问题领域，将抑郁与重要人际事件连接，心理教育，讨论治疗目标
- **第4-10次**：每周评估症状和人际事件，聚焦问题领域开展沟通分析、角色扮演、决策分析，探索社会支持选项
- **第11-12次**：回顾治疗收获，讨论结束情绪，巩固技巧，制订复发预防计划

### 人际关系清单

人际关系清单是初始阶段核心工具。它帮助治疗师和来访者识别重要关系、关系质量、支持资源和压力来源。

可询问：

- 谁是你生命中重要的人？
- 谁和你生活在一起？
- 谁与你联系最多？
- 你比较愿意向谁求助？
- 和谁在一起最舒服？和谁最紧张？
- 最近哪些关系发生了变化、冲突或中断？
- 哪些关系最影响你的情绪？

人际关系清单不是社交名单，而是带有情感能量的关系地图。治疗师要关注关系中的支持、冲突、期待、失望、依赖、距离和沟通方式。

### 四大问题领域（Problem Areas）

| 问题领域 | 相关生活事件 | 治疗目标 |
|---------|-------------|---------|
| **悲伤反应/复杂哀痛**（Grief/Complicated Bereavement） | 配偶/亲人/挚友/宠物去世 | 促进哀悼（Facilitate Mourning），重建兴趣和关系 |
| **角色冲突**（Interpersonal Disputes） | 婚姻/亲子/同事冲突 | 识别冲突阶段，调整期待或沟通方式，选择行动计划 |
| **角色转换**（Role Transitions） | 工作变动/孩子出生/退休/离婚/搬家 | 哀悼旧角色，接纳新角色，发展新技能和支持 |
| **人际缺陷**（Interpersonal Deficits） | 社会隔离，缺乏或无法维持亲密关系 | 减少社交孤立，建立新的人际关系 |

**选择原则**：优先选择①最近发生、最痛苦的人际事件；②与抑郁发作时间线最吻合；③来访者最有动机处理；④短期最可操作的焦点。**通常只聚焦一个问题领域**。

## Agent 使用建议

- **适用场景**：当 agent 需要为抑郁症来访者建立IPT概念化框架、选择问题领域、理解IPT治疗结构时使用。
- **可直接引用**：三大理论基础的定义、四大核心特征、12次简版流程表、四大问题领域的鉴别要点。
- **需要结合**：本系列文件中各问题领域的案例（03_04、05_06），用于将理论框架与具体干预技术对接。
- **注意**：IPT的适应证覆盖广泛，但重度抑郁、自杀风险高、精神病性症状等情况需联合药物治疗或转介精神科，不应单独使用IPT。
