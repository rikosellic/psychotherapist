抽取范围：第3-4章，**悲伤反应或复杂的哀痛的IPT治疗**（IPT for Grief or Complicated Bereavement），**角色冲突的IPT治疗**（IPT for Interpersonal Disputes）。

> **章节主线**：第三章和第四章分别聚焦于IPT四大问题领域中的前两个——悲伤反应（Grief）和角色冲突（Interpersonal Disputes）。两章均以案例贯穿治疗三阶段（初始、中期、终止），展示如何将抑郁症状与具体人际问题领域关联，并使用沟通分析、角色扮演、决策分析等技术推动改变。第三章强调促进哀悼与增强社会支持两大核心任务；第四章强调识别冲突阶段、调整期待和改善沟通模式。

## 第三章：悲伤反应或复杂哀痛的 IPT 治疗

### 问题领域定位

悲伤反应或复杂哀痛适用于抑郁发作与重要人物死亡密切相关的个案。正常哀悼会随时间波动并逐渐整合到生活中；复杂哀痛则表现为长期、强烈、阻滞的悲伤，使来访者难以恢复功能和重新投入现有关系。IPT 的核心不是让来访者"忘记逝者"，而是帮助其完成哀悼过程，同时重新连接现实生活中的支持和角色。

复杂哀痛可表现为：对逝者强烈思念、难以接受死亡事实、回避相关话题、觉得生命失去意义、与现有关系疏远、内疚自责或愤怒，常伴抑郁、睡眠问题或躯体不适。

> 以下以 **C 女士案例** 贯穿三阶段展示具体操作。C 女士（70岁）相伴 50 年的丈夫因肺癌去世一年后出现中度抑郁（PHQ-9=15，HAMD-17=18），虽在服用舍曲林但效果不理想，与女儿关系也日趋紧张。依恋评估为**恐惧型**（高依赖、高回避）。

### 初始阶段（第1-3次）

#### 一般任务

- 评估抑郁严重度、自杀风险和是否需要药物或其他治疗。
- 理解丧失事件和症状发生时间线。
- 建立人际关系清单，识别逝者和现存支持系统。
- 赋予病人角色，降低自责。
- 将抑郁症状与丧失、孤独、关系疏离连接起来。
- 与来访者共同确定悲伤反应/复杂哀痛为问题领域。
- 设定治疗目标。

#### C 女士案例进展

- **第1次**：面容憔悴，PHQ-9=15（中度抑郁），HAMD-17=18。讲述丈夫患癌及离世经过——她同意抽取胸腔积液后丈夫隔天离世，**一年来坚信这是医疗失误**，反复自责。向女儿倾诉反被劝"不要多想"，关系转差。治疗师讲解治疗模式（10-12次，每周45分钟）。
- **第2次**：建立人际关系清单——丈夫是核心支柱，女儿喜（关系紧张）、深圳好友年华（唯一心灵支柱）、歌舞团队长康、堂妹春等构成有限支持圈。依恋评估为恐惧型，需给予更多鼓励不可操之过急。
- **第3次**：使用个案概念化图表梳理生理/心理/社会/文化因素，C 女士选择**悲伤反应**为聚焦点。共同制定目标：①帮女儿家庭做力所能及的事；②在香港和内地两边走动、联系内地朋友。

### 中期阶段（第4-10次）

#### 一般任务

两大核心任务：**促进哀悼过程** + **增加社会支持**。

促进哀悼包括：

- 鼓励来访者详细讲述逝者、生前关系、死亡过程和丧失后的变化。
- 帮助来访者表达悲伤、思念、愤怒、内疚、遗憾、恐惧等复杂情绪。
- 识别回避哀悼的方式，如封闭、过度忙碌、拒谈逝者、完全依赖他人。
- 帮助来访者从“被困在死亡事件”逐渐转向“带着记忆继续生活”。

增加社会支持包括：

- 识别可以联系的人。
- 帮助来访者表达需要。
- 鼓励与亲友恢复沟通。
- 处理因悲伤而疏远的关系。
- 规划新的生活活动和支持结构。

常用技术：沟通分析、角色扮演、澄清技术、心理教育、肯定鼓励、解决问题技术。

#### C 女士案例进展

- **第4次（探索关系）**：回顾与丈夫从歌舞团相识到定居香港的历程。运用**澄清技术**——她开始觉得"丈夫对自己的健康也有责任"，自责开始松动。
- **第5次（面对死亡）**：详细回顾十年抗癌到临终全过程。C 女士反复问"是否是医疗失误"，治疗师以倾听和共情回应。她首次感到被理解，咨访关系稳固建立。
- **第6次（处理归咎）**：情绪反复，带来所有医疗报告。治疗师利用医学背景**逐项解读**——癌细胞已广泛扩散至脑部和肝脏。她**恍然大悟**：死亡是自然发展，非医疗失误。自责显著减轻。
- **第7次（释怀→修复关系）**：心情好转，首次能入睡。焦点转向**修复母女关系**。通过**沟通分析**发现：她说"担心患癌"，真实需求是"担心年老无人照应"但未直接表达。用**角色扮演**训练直接表达感受。
- **第8次（关系突破）**：主动向女儿道歉，直接说出担心年老无人照顾。**女儿回应**"就算妈妈走不动了也会照顾你"。关系显著改善，抑郁随之好转。用**头脑风暴**探讨帮女儿家庭做事（目标①）。
- **第9次（扩展支持）**：主动提出照顾孙儿，女儿很高兴；已开始联系内地朋友（目标②进展）。
- **第10次（准备终止）**：倒数第二次。治疗师强调改善靠她自己的努力。C 女士提出多回内地旅游，**女儿表示支持**。

### 终止阶段（第11次）

#### 一般任务

- 回顾抑郁症状改善和哀悼过程进展。
- 总结来访者如何重新连接现实关系。
- 讨论未来可能触发悲伤的情境，制定复发预防计划。
- 处理治疗结束带来的分离感。
- 必要时安排维持治疗。

#### C 女士案例结局

- **第11次**：回顾治疗过程与学到的技巧，讨论复发早期症状和求助途径。安排1个月和2个月后各20分钟维持面谈。
- **治疗成果**：PHQ-9 从 **15分降至3分**，HAMD-17 从 **18分降至1分**。
- **一年后随访**：心情完全康复，已停用抗抑郁药；与女儿关系比以前更好；经常往返香港和内地与好友旅游；对丈夫的思念转为甜蜜回忆。

### 临床要点总结

- 逝者关系越核心，丧失后的角色和支持空缺越大。
- 自责和归咎是阻滞哀悼的常见障碍——心理教育（如解读医疗报告）可有效化解。
- 悲伤常伴随对现有亲人的误解和疏离，**沟通分析+角色扮演**可直接修复现实支持。
- 恐惧型（高依赖+高回避）依恋者需更多鼓励，不可操之过急；终止阶段提前预告以减轻分离焦虑。

## 第四章：角色冲突的 IPT 治疗

### 问题领域定位

角色冲突适用于抑郁症状与重要关系中的期待不一致、沟通失败、争执或关系僵局有关的个案。常见关系包括伴侣、父母、子女、手足、同事、上级、朋友等。

IPT 不把角色冲突简单理解为“谁对谁错”，而是分析双方各自的期待、沟通模式、情绪反应和可改变行动。

### 角色冲突阶段

角色冲突可分为：

- 协商阶段：双方仍能讨论分歧，有修复空间。
- 僵局阶段：双方反复争执或回避，沟通无效。
- 关系破裂阶段：一方或双方想结束关系，冲突已严重破坏关系。

不同阶段决定治疗目标。协商阶段可直接改善沟通；僵局阶段需要识别循环和调整期待；关系破裂阶段可能要帮助来访者决定是否修复、保持距离或结束关系。

### 初始阶段任务

初始阶段包括：

- 评估抑郁症状、风险和治疗需求。
- 建立联盟和病人角色。
- 建立人际关系清单，找出与抑郁最相关的冲突关系。
- 梳理冲突时间线和抑郁时间线。
- 明确冲突双方的角色期待。
- 共同确定角色冲突为治疗焦点。
- 设定可量化目标，如减少争吵、增加有效沟通、改善家庭支持。

### 中期阶段目标

中期治疗目标：

- 识别冲突和冲突阶段。
- 了解双方期待和情绪。
- 选择行动计划。
- 调整不现实期待。
- 训练更有效沟通。
- 增加社会支持。
- 减轻抑郁症状。

### 沟通分析

沟通分析是角色冲突的核心技术。治疗师帮助来访者回顾一次具体互动：

- 当时发生了什么？
- 对方说了什么、做了什么？
- 来访者说了什么、做了什么？
- 来访者当时感受是什么？
- 想表达的真正需求是什么？
- 实际表达出来了吗？
- 对方如何反应？
- 这个反应是否符合来访者目标？
- 下次可以如何表达？

重点是把模糊抱怨转化为具体沟通行为。

### 期待调整

角色冲突常来自双方期待不一致。治疗师帮助来访者区分：

- 合理期待。
- 不现实期待。
- 未表达的期待。
- 文化和家庭角色带来的期待。
- 对方可能有但未说出的期待。

目标不是让来访者放弃需求，而是把需求表达得更清楚、更可被回应。

### 角色扮演

角色扮演用于练习新沟通方式。治疗师可扮演对方，让来访者练习：

- 直接表达感受。
- 提出具体请求。
- 认可对方感受。
- 避免指责和泛化。
- 处理对方拒绝或防御。

角色扮演后，应回顾来访者体验，并布置现实生活中的小行动。

### 终止阶段

终止阶段需要：

- 回顾冲突变化和症状改善。
- 总结有效沟通策略。
- 识别仍未解决的冲突。
- 讨论未来冲突复发时的应对。
- 处理治疗结束。

## Agent 使用建议

- **适用场景**：当 agent 处理的抑郁症来访者的问题与丧亲（悲伤反应）或人际关系冲突（角色冲突）明显相关时使用。
- **可直接引用**：悲伤反应的初始阶段评估要点、促进哀悼的具体策略、角色冲突的三阶段分类、沟通分析的步骤化提问、角色扮演的脚本拟定方法。
- **需要结合**：`01_02_基础理论与操作方案.md` 中的IPT整体结构和问题领域选择原则，以及 `07_治疗常见问题处理.md` 中的沟通分析和角色扮演详细技术。
- **注意**：两个问题领域可能同时存在（如丧亲后与家人发生角色冲突），治疗师应选择一个主要焦点。案例均进行了文化调适（如中国文化中哀悼的表达方式、家庭角色期待），需考虑个体差异。

## 综合分析

第3章（悲伤反应）和第4章（角色冲突）虽然分属不同问题领域，但在临床实践中常相互关联。例如，一位丧偶的来访者（悲伤反应）可能与子女在遗产处理或居住安排上发生冲突（角色冲突），或与女婿/儿媳在新家庭角色上产生矛盾。治疗师应通过人际关系清单和时间线梳理，判断哪个问题是抑郁发作的最直接诱因，并优先处理。悲伤反应的治疗更侧重于情感宣泄与哀悼过程，而角色冲突更侧重于沟通分析和期待调整——两者在技术上有交叉（均使用沟通分析和角色扮演），但在治疗焦点上有所不同。

如果个案是角色冲突：

- 识别冲突对象、阶段、期待差异和沟通循环。
- 使用沟通分析、期待调整、角色扮演和行动计划。
