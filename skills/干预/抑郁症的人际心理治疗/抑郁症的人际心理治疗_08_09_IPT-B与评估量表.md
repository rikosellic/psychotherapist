## 第八章：简短人际心理治疗 IPT-B

> **章节主线**：IPT-B（Brief Interpersonal Psychotherapy）是标准IPT的8次访谈简短版本，由Klerman和Weissman等设计，最初针对抑郁症母亲群体。其核心策略是保留IPT基本结构，但通过压缩初始阶段、更早确定焦点、使用人际家庭作业和行为激活来加速进程。本章分为概述（起源、循证、与IPT异同、治疗结构）和案例解析（孕期角色冲突的C女士）两部分。

### IPT-B的起源与定位

- IPT-B起源于临床现实：许多来访者（学生、孕产妇、上班族）无法完成12-16次标准疗程。研究发现"完成"标准IPT的女性平均只参加了8次访谈（Swartz et al., 2004）。
- 中国需求突出：孕28周抑郁检出率25.4%，产后抑郁发生率约14.7%；学生群体受限于学业时间。
- 实证支持：半数抑郁症来访者在8次治疗内实现症状缓解（Howard et al., 1986）；英国RCT发现8次CBT和动力性人际关系疗法的疗效不逊于16次（Shapiro et al., 1994）。
- IPT-B设定8次访谈（可加1次摄入性访谈），一周1次，14周内完成。从治疗开始就对疗程设限，促使来访者和治疗师更有效率地工作。

### IPT-B的循证依据

- 与舍曲林比较：IPT-B组比药物组改善更快（Swartz et al., 2004）。
- 针对有精神障碍孩子的抑郁症母亲：HAMD评分降低11.3分（Swartz et al., 2006）；后续RCT随访9个月确认IPT-B组显著优于常规治疗组（Swartz et al., 2008a）。
- 基层卫生保健抑郁症来访者：IPT-B组抑郁症状减轻程度高于等待治疗组（Graham, 2006）。
- 孕期抑郁症（妊娠12-30周）：IPT-B组社会支持满意度显著提高（Lenze & Potts, 2017）。

### IPT-B与标准IPT的异同

**共性**：理论基础均为依恋理论和人际理论；将抑郁症状与人际事件连接；使用病人角色和心理教育；建立人际关系清单；四大问题领域中IPT-B省略了人际缺陷；使用沟通分析、角色扮演等技术；强调治疗结束和复发预防。

| 领域 | IPT | IPT-B |
|------|-----|-------|
| 目标 | 解决抑郁发作 | 解决抑郁发作 |
| 治疗时长 | 12～16周 | 8周 |
| 初始/中期/终止 | 3次 / 7～11次 / 2～3次 | 2次 / 5次 / 1次 |
| 频率与时长 | 每周45分钟 | 每周45分钟 |
| 问题领域 | 四大领域全含 | 省略人际缺陷 |
| 人际家庭作业 | 非正式 | 正式要求 |
| 行为激活 | 未特别说明 | 鼓励参加以前喜欢的活动 |

**关键差异**：

- **时限性**：每次访谈开始时提醒剩余次数，强化"时间杠杆"效应，更紧密聚焦一个核心问题领域。
- **治疗师角色**：更加主动——更快完成人际问题概念化，更果断地在来访者偏离目标时重新定向，谨慎权衡只选一个聚焦点。
- **聚焦此时此地**：使用缩减版人际关系清单（"你现在与谁同住？""最近向谁倾诉？""这周常与谁联络？"），仅在与当前抑郁相关时才讨论过去关系。
- **医学模式**：与IPT一致，赋予病人角色，与药物治疗完全兼容。
- **情绪与事件的关联**：保留核心观念，鼓励来访者探索症状与人际事件之间的联系。

### IPT-B治疗结构

IPT-B保留标准IPT的三阶段结构，但采用一系列策略压缩时间、加速进程。

#### 初始阶段（第1-2次）

- 目标与标准IPT相同，但压缩为2次访谈。
- **第1次访谈末尾**即给出初步个案概念化（"最佳猜想"），第2次修正确认。
- 使用缩减版人际关系清单，聚焦当前最重要、可触及的关系。原生家庭信息仅在与当前抑郁相关时才讨论。
- 只选**一个**问题领域，且须考量在8次内的可行性——基于来访者现有能力和资源、能在短期内做出努力的问题，而非纠正长期个性或人际模式。**避免选择人际缺陷**。
- 强调病人角色，鼓励来访者承担更多责任尝试人际冒险，同时维持推动改变与充分共情的平衡。

#### 中期阶段（第3-7次）

- 每次访谈从回顾上周开始，将人际事件与情绪联系或反之。
- 使用与标准IPT相同的技术（沟通分析、角色扮演、激发情感、澄清等），但频率更高。
- **人际家庭作业**是IPT-B加速进程的核心技术：每次访谈末尾与来访者共同商定本周的人际实操练习。
  - 示例：与丈夫沟通感受并要求更多陪伴；和女儿散步/谈话；邀请孩子同学家长一起出游；分配家务给子女；每天半小时做只为自己做的事；给母亲扫墓并请朋友陪同；寻找感兴趣的自我提升课程。
  - 完成则积极反馈并询问对情绪的影响；未完成则归咎于抑郁症而非来访者，调整难度。
- **行为激活**：借鉴CBT，帮助来访者探索阻碍参与活动的原因，逐级渐进重新激活有益的社交行为。
- 持续支持自我效能感：鼓励循序渐进完成任务以获得成功经验；对成就和能力表达赞赏。

#### 终止阶段（第7-8次）

- 比标准IPT来得更早，但治疗结束从始至终未远离来访者和治疗师的意识。
- 任务：引起和处理对治疗结束的情绪反应；回顾治疗效果和获得的技能；识别尚未解决的问题；讨论如何应对将来可能的抑郁复发。
- **预期改善时间点**：第四次访谈后开始观察到改善迹象。
- **若无效**：强调是治疗方法而非来访者的问题——"IPT-B不适用于所有人，还有其他心理治疗、药物治疗、物理治疗可选"。若治疗期间出现明显恶化（如自杀意念），应立即考虑替代疗法。
- **结束不等于终止**：可继续药物治疗；可转入维持性治疗（尤其有复发可能或慢性抑郁病史者）；60%完成IPT-B的来访者报告接受了后续心理治疗（Swartz et al., 2008a）。IPT-B也可作为筛选需要更密集心理服务者的"筛子"。

### 案例：C女士（孕期角色冲突/IPT-B）

> **案例概要**：新冠疫情期间，C女士怀二胎6个月，有一个7岁女儿。丈夫作为公务人员经常不在家，叫来婆婆帮忙照顾。但婆媳因教育理念、家务方式和重男轻女态度产生矛盾，C女士受困于"孝顺"文化无法表达不满。丈夫一味要求忍让，C女士情绪逐渐恶化，莫名哭泣。问题领域确定为**角色冲突**（与婆婆的隐匿冲突+与丈夫的沟通失败）。

**关键治疗节点**：

- **心理健康教育与病人角色**：让C女士理解她目前是抑郁障碍病人，需要照料自己的情绪而非一味压抑。
- **人际关系清单**：丈夫（关系不错但近期冲突）、女儿（紧密但辅导作业有压力）、婆婆（无表面冲突但相处不愉快）、母亲（支持但在外省因疫情无法前来）。
- **沟通分析**：发现C女士与丈夫沟通时很少直接表达感受和需求，基本停留在现实问题描述上。
- **人际家庭作业**：①与丈夫沟通自己的状态和感受——丈夫惊讶，但开始理解；②与丈夫讨论让孩子上晚托班、婆婆回家的构想及可行性。
- **角色扮演**：治疗师扮演丈夫，练习如何开口让婆婆回家。换位思考后找到合适理由（以公公需要为由请婆婆回去），避免"赶人"的不孝困境。
- **结果**：婆婆回家后情绪缓解；夫妻交流方式改进——从小心翼翼到直接表达获得支持和理解；C女士主动问候婆婆、送母亲节礼物，关系改善。抑郁分数显著下降。

## 第九章：治疗中常见评估量表

### 总述：量表在IPT中的角色

量表在 IPT 中不是形式化检查，而是用于：

- 评估抑郁严重度。
- 追踪治疗进展。
- 识别风险。
- 辅助确定是否需要药物或转介。
- 判断治疗是否有效。
- 帮助来访者看到变化。

量表不能替代临床访谈和风险评估。评分结果应结合来访者叙述、功能状态和治疗师观察解释。


### 第一节：他评量表

#### 汉密尔顿抑郁量表 24 项版（HAMD-24）

性质：他评量表。

适用范围：适用于有抑郁症状的成人，可用于抑郁症、双相情感障碍、神经症等多种疾病中的抑郁症状评定，尤其适用于抑郁症。

施测方式：通常采用交谈和观察，由经过训练的评定员完成。一次评定约 15-20 分钟，严重阻滞或合作困难时耗时更长。用于 IPT 前后比较时，可在入组时评定前一周情况，再在干预 2-8 周后再次评定。

评分方式：

- **计分**：部分项目0-2分三级评分，部分项目0-4分五级评分。总分为24项累加。

**24项按症状域归类**：

| 症状域 | 包含项目 |
|--------|---------|
| **核心情绪** | 抑郁情绪(1)、有罪感(2)、自杀(3) |
| **睡眠障碍** | 入睡困难(4)、睡眠不深(5)、早醒(6) |
| **活动与动力** | 工作和兴趣(7)、阻滞(8)、激越(9) |
| **焦虑** | 精神性焦虑(10)、躯体性焦虑(11) |
| **躯体症状** | 胃肠道症状(12)、全身症状(13)、性症状(14)、体重减轻(16) |
| **认知与自我评价** | 疑病(15)、自知力(17)、能力减退感(22)、绝望感(23)、自卑感(24) |
| **其他** | 日夜变化(18)、人格解体/现实解体(19)、偏执症状(20)、强迫症状(21) |

**结果判定**：<8分正常；8-20分可能有抑郁症状；21-35分可能轻或中度抑郁；>35分可能严重抑郁。

> **Agent提示**：HAMD-24适合专业评定和治疗前后疗效比较。若自杀(3)、绝望(23)、精神病性症状项目突出，应优先做风险和精神科评估。

### IPT 效果问卷，治疗师评定版 / IPT 问题领域评估卷

性质：治疗师评定工具。

用途：帮助治疗师评估 IPT 治疗相关进程和实际治疗效果。治疗师可根据访谈情况评定，也可观看治疗录像进行评定。

书中呈现的核心内容是“IPT 问题领域评估卷”，包括两部分：A. 人际问题领域；B. 治疗任务的构建。

#### A. 人际问题领域

1. 悲伤：

- 标记有或无。
- 若有，区分不复杂的悲伤或复杂悲伤。
- 需要识别逝者、与来访者的关系、去世日期、抑郁发作和去世的时间间隔。

2. 角色冲突：

- 标记有或无。
- 识别重要人物或伴侣。
- 判断是否存在僵局。
- 识别冲突主要议题：权威/主导、依赖、性问题、抚育孩子。
- 判断上述议题中哪一个是主要议题。
- 记录角色冲突大概持续了几个月。

3. 角色转换：

- 标记有或无。
- 识别地理位置搬迁、结婚/同居、分居/离婚、毕业新工作、失业/退休、健康问题、其他变化。
- 若有多个角色转换，判断哪一个更占主导。
- 记录抑郁发作和事件发生间隔有几个月。

4. 人际缺陷：

- 标记有或无。
- 识别回避、依赖、受虐待、边缘性、分裂型、缺乏社交技能、其他。
- 若有多个问题，判断哪一个更占主导。

#### B. 治疗任务的构建

书中要求对每个圈定为“有”的问题领域继续评估：

- 评估其对情绪的影响程度：1 = 非常重要，2 = 次重要，3 = 不那么重要。
- 为悲伤、角色冲突、角色转换、人际缺陷分别打分。
- 选择用于和来访者确定治疗合同的问题领域，最多选 2 个，1 = 最重要。
- 写明选择该问题领域的理由。
- 判断视频或访谈中的面询者是否以其对问题领域的意见影响评估者判断。
- 判断视频或访谈是否提供足够信息来选择和明确问题领域。
- 记录其他评论。

Agent 使用提示：这个工具可直接用于 IPT 个案概念化，尤其适合在初始阶段从多个问题领域中选择主焦点。

### 病人健康问卷抑郁自评量表，PHQ-9

性质：自评量表。

用途：简便、有效地评估抑郁症状严重程度，可辅助抑郁障碍诊断，也可在治疗过程中评估疗效。

指导语要点：根据过去两周情况，回答是否存在下列状况及频率。

- **计分**：每项0-3分（0=没有，1=有几天，2=一半以上时间，3=几乎天天），总分9项累加。

**9项归类**：

| 维度 | 项目 |
|------|------|
| **核心抑郁** | 兴趣减退(1)、情绪低落(2) |
| **躯体/精力** | 睡眠障碍(3)、疲倦(4)、食欲变化(5) |
| **认知/自我** | 自我否定(6)、专注困难(7) |
| **行为/风险** | 精神运动变化(8)、自伤自杀念头(9) |

**结果判定**：1-4分轻微；5-9分轻度；10-14分中度；15-19分重度；20-27分极重度。

> **Agent提示**：PHQ-9适合每周或阶段性追踪。第9项涉及自伤/自杀念头，**任何非0答案**都应进一步评估自杀风险，不能只按总分处理。

### IPT 治疗师自我检查列表

- **性质与用途**：治疗师每次访谈后自评，用于督导反馈和会谈质量检查，不评估来访者症状。
- **所有阶段**：是否将情感/症状与人际经验联系？是否使用共情性开放式提问？是否讨论社会支持？是否跟踪症状和自杀意念？
- **开始阶段**：是否讨论了同期社会心理压力？是否提供抑郁症/病人角色/IPT心理教育？是否建立人际关系清单？是否讨论治疗目标？
- **中间阶段**：以"自从上次见面以来您怎么样？"开场→聚焦选定领域。悲伤→探索死亡细节与应对；角色转换→哀悼旧角色、探索新角色挑战与机遇；角色冲突→探索冲突关系与分歧，使用沟通分析；人际缺陷→鼓励社交互动、讨论困难模式。
- **终止阶段**：是否肯定努力与进步？是否回顾治疗经历？是否准备维持疗效？是否处理结束担忧？是否制定复发应急计划？

### 量表使用原则与要点

- **指导语**：他评量表由治疗师念指导语后通过结构性访谈完成；自评量表由来访者自行阅读完成，若受文化程度限制可由他人帮忙解释。
- **时限**：无明确规定，但要求阅读流畅的情况下尽可能一次性完成。
- **隐私保护**：评估过程及结果属个人隐私，应选择无干扰环境完成，结果由治疗师保管。
- **结果解读**：可让来访者自行对照计分，治疗师也可解读。注意不要因测验结果给来访者轻易贴标签，结果应结合人际事件和临床访谈综合判断。


### 量表在IPT三阶段的整合使用

| 阶段 | 推荐量表 | 重点 |
|------|---------|------|
| **初始** | HAMD-24（基线）+ PHQ-9 + IPT效果问卷 | 评估严重度和风险、确定问题领域、建立基线 |
| **中期** | PHQ-9（每周/每两周） | 追踪症状变化、检查人际行动是否带来情绪改善 |
| **终止** | HAMD-24（对比）+ PHQ-9 | 用数据让来访者看到变化、讨论残留症状和复发风险 |

## Agent 使用建议

- **适用场景**：当 agent 需要在有限次数内完成IPT治疗（IPT-B），或在治疗过程中需要使用标准化评估工具追踪疗效、评估风险时使用。
- **可直接引用**：IPT-B与标准IPT的异同对照表（含阶段次数对比）、IPT-B三阶段治疗结构（初始2次/中期5次/终止1次）、缩减版人际关系清单的提问方式、人际家庭作业示例列表、C女士孕期角色冲突案例的完整治疗节点；HAMD-24的24项症状域归类表与结果判定、IPT效果问卷的四领域评估框架、PHQ-9的9项维度归类表与结果判定、IPT治疗师自我检查列表的压缩版要点（按阶段）、量表使用原则与要点（指导语/时限/隐私保护/结果解读）、量表在IPT三阶段的整合使用表。
- **需要结合**：`01_02_基础理论与操作方案.md` 中的标准IPT结构，以及各问题领域案例文件中的治疗具体操作。IPT-B的异同对比表可直接用于判断来访者是否适合IPT-B；C女士案例可与标准IPT中的角色冲突案例对比，理解简短版与标准版在节奏和焦点上的差异。HAMD-24和PHQ-9的计分标准应与各章节案例中的量表使用实例结合参考。
- **注意**：IPT-B不适用于所有来访者（人际缺陷领域被省略、严重自杀风险/精神病性症状/重度抑郁需优先考虑联合治疗）。量表结果应结合临床访谈综合判断，不能单独作为诊断或疗效依据。IPT-B结束不等于治疗终止，60%完成者会继续接受后续心理治疗。

## 综合分析

第8章（IPT-B）和第9章（评估量表）在IPT实践中紧密关联：IPT-B的短期结构对症状追踪和疗效评估的需求更高，PHQ-9等自评量表在IPT-B中的使用比标准IPT中更为频繁。治疗师应在每次IPT-B访谈开始时使用PHQ-9评估症状变化，将评分趋势作为判断治疗是否有效的客观依据。治疗师自我检查列表可帮助IPT-B治疗师在更紧凑的节奏中保持对结构的遵守。第九章新增的量表使用要点（指导语/时限/隐私保护/结果解读）为量表的合规使用提供了操作规范。

## Agent 快速调用

如果用户问 IPT-B：

- 回答为“更短、更聚焦、更主动”的 IPT 变体。
- 说明适用轻中度、资源有限或孕产期等场景。
- 强调 8 次左右、快速选择问题领域、每次访谈最大化。

如果用户问量表：

- PHQ-9 适合自评和每周追踪，第9项涉及自杀必须即时处理。
- HAMD-24 适合专业他评和治疗前后比较。
- IPT效果问卷用于初始阶段的个案概念化，帮助确定问题领域。
- IPT治疗师自我检查列表用于督导和会谈质量检查。
- 量表使用须遵循指导语/时限/隐私保护/结果解读四项要点，结果必须结合临床访谈。
