# 抑郁症的人际心理治疗_index

来源：`./skills/抑郁症的人际心理治疗/book/抑郁症的人际心理治疗.md`
原书：黄满丽（主编），*抑郁症的人际心理治疗：理论与案例实践*。浙江科学技术出版社，2021。
用途：供 agent 使用中文快速查询"抑郁症的人际心理治疗（IPT）"框架。本书系统介绍了人际心理治疗（Interpersonal Psychotherapy, IPT）的理论基础、操作方案、四大问题领域的案例实践、常见问题处理及简短人际心理治疗（IPT-B）。提取时保留英文专业术语，按主题关联进行文件拆分。

## 文件导览

### 抑郁症的人际心理治疗_01_02_基础理论与操作方案.md

覆盖第1-2章：**成为抑郁症的IPT治疗师** + **抑郁症的IPT操作方案**。

适合查询

- IPT的理论基础（依恋理论Attachment Theory、人际理论Interpersonal Theory、社会理论Social Theory）
- IPT的四大特征、循证证据、适应证（成人/青少年/老年/围产期抑郁等）
- 抑郁症的症状表现、成因、评估与综合治疗
- IPT治疗师的胜任特征
- IPT三阶段治疗结构（初始、中期、终止）及12次简版流程
- 人际关系清单（Interpersonal Inventory）的建立方法与结构
- 四大问题领域分类：悲伤反应/复杂哀痛（Grief/Complicated Bereavement）、角色冲突（Interpersonal Disputes）、角色转换（Role Transitions）、人际缺陷（Interpersonal Deficits）

### 抑郁症的人际心理治疗_03_04_悲伤反应与角色冲突.md

覆盖第3-4章：**悲伤反应或复杂的哀痛的IPT治疗** + **角色冲突的IPT治疗**。

适合查询：

- 复杂哀痛（Complicated Grief）的症状、风险因素及IPT治疗证据
- 悲伤反应IPT的初始/中期/终止阶段案例（C女士，丧偶）
- 哀悼过程（Facilitate Mourning）与增强社会支持的具体技术
- 中国文化中哀悼的特殊议题及IPT调适
- 角色冲突（Interpersonal Disputes）的定义与三阶段：再协商（Negotiation）、僵持（Impasse）、关系破裂（Dissolution）
- 角色冲突IPT的初始/中期/终止阶段案例（M女士，夫妻冲突）
- 沟通分析（Communication Analysis）、决策分析（Decision Analysis）、角色扮演（Role Play）等技术应用

### 抑郁症的人际心理治疗_05_06_角色转换与人际缺陷.md

覆盖第5-6章：**角色转换的IPT治疗** + **人际缺陷的IPT治疗**。

适合查询：

- 角色转换（Role Transitions）的定义与常见情境（生育、升职、搬家、退休等）
- 角色转换IPT的初始/中期/终止阶段案例（L女士，二胎母亲）
- 哀悼旧角色、发展新技能、建立新支持的五大策略
- 人际缺陷（Interpersonal Deficits）的识别标准与治疗难点
- 人际缺陷IPT的初始/中期/终止阶段案例（L女士，社交隔离）
- 沟通分析、情感宣泄（Expression of Affect）、决策分析技术在人际缺陷中的应用
- 社会资源活动清单的制定

### 抑郁症的人际心理治疗_07_治疗常见问题处理.md

覆盖第7章：**治疗中常见问题的处理**。

适合查询：

- 多个人问题领域同时出现时的处理策略与焦点选择依据
- 向成人、儿童/青少年解释抑郁症症状的心理教育方法
- "此时此地"（Here and Now）焦点下如何处理过往经历
- 沟通分析的五种无效沟通模式与四种有效沟通技巧
- 角色扮演的具体操作步骤、脚本拟定、应对抗拒的方法
- 换位思考（Perspective Taking）的启发性提问技术
- 人际关系圈（Interpersonal Circle）的绘制与探索要点
- 终止治疗与维持治疗的策略
- 治疗不顺利的情况处理（被动、离题、沉默、迟到、提前结束等）
- 自杀风险评估与干预（三级预防）
- 礼物馈赠、家长询问未成年人情况等伦理边界问题

## Agent 使用建议

- **适用场景**：当 agent 需要对抑郁症来访者进行人际心理治疗的个案概念化、选择问题领域、设计分阶段干预方案时使用。
- **可直接引用**：各问题领域的定义、治疗三阶段的核心任务表、案例中的具体对话示范和技术运用步骤。
- **需要结合**：`./skills/模板/` 中的概念化框架，用于将IPT理论与具体来访者情况整合分析。
- **注意**：IPT是一种结构化短程治疗（通常8-20次），案例均基于中国人群背景进行了文化调适，使用时需考虑个体差异和诊断完整性。

### 文件选择指引

如果用户问"IPT 如何治疗抑郁症"，优先读取 `_01_02_基础理论与操作方案`。

如果用户给出个案并要求确定 IPT 问题领域，读取 `_01_02_基础理论与操作方案` 的问题领域分类，再根据具体领域读取 `_03_04_悲伤反应与角色冲突` 或 `_05_06_角色转换与人际缺陷`。

如果个案核心是丧失、死亡、复杂哀痛，读取 `_03_04_悲伤反应与角色冲突`。

如果个案核心是婚姻、亲子、同事或家庭冲突，读取 `_03_04_悲伤反应与角色冲突` 的角色冲突部分。

如果个案核心是产后、退休、毕业、迁居、患病、离婚等身份变化，读取 `_05_06_角色转换与人际缺陷` 的角色转换部分。

如果个案核心是孤独、关系少、社交退缩、缺乏支持，读取 `_05_06_角色转换与人际缺陷` 的人际缺陷部分。

如果用户问治疗过程卡住、终止、多个焦点、沉默、礼物、家庭参与等，读取 `_07_治疗常见问题处理`。

如果用户问短程版本或量表，读取 `_08_09_IPT-B与评估量表`。
