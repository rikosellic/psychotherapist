## 第 16 章：Learning to Listen

### 章节主线

- 倾听有三种模式：环境式、过滤式、聚焦式。治疗师在会谈中需要在这三种模式之间灵活切换。
- 不仅听声音，也听沉默。不仅听内容，也听节奏、音量、音调。
- 倾听目标是识别：情感、自由联想/阻抗、移情、防御、无意识幻想/冲突、梦。

### 三种倾听模式

| 模式 | 特点 | 适用场景 |
|------|------|---------|
| **环境式倾听（Ambient listening）** | 不特意寻找什么，让声音流过 | 会谈开始时的最佳模式 |
| **过滤式倾听（Filtered listening）** | 在保持开放的同时寻找模式 | 会谈中段 |
| **聚焦式倾听（Focused listening）** | 集中于一个主题/情感，排除其他 | 特定时刻需要深入时 |

### 听什么

- **沉默**——可以是平静的、隐秘的、紧张的
- **语言之外**——节奏、音量、音调、音色
- **断裂和不一致**——话题转换、口误（parapraxes）
- **否定/双重否定**——"我不是不去法学院"
- **被动语态**——与能动性保持距离
- **节点（Nodal points）**——重复出现的词/符号/清晰点，通向无意识枢纽
- **重要内容**：情感、自由联想/阻抗、移情、防御、无意识幻想/冲突、梦

## 第 17 章：Learning to Reflect

### 章节主线

- 反思 = 处理收集到的数据，理解含义，决定是否干预以及如何干预。
- 两组原则指导反思过程：**选择原则**（在哪里干预）和**准备原则**（来访者是否能使用听到的内容）。
- 使用三个信息源：来访者的历史信息、自己的临床经验、理论和技术的知识。

### 三个选择原则（Choosing Principles）

| 原则 | 内容 |
|------|------|
| **从表面到深度（Surface to Depth）** | 评论最接近意识层面的材料；不要直接挖掘深层材料 |
| **跟随情感（Follow the Affect）** | 情感是"最好的向导，告诉我们什么对来访者最重要" |
| **注意反移情（Attend to the Countertransference）** | 自己的感受是宝贵的数据 |

### 三个准备原则（Readiness Principles）

| 原则 | 评估问题 |
|------|---------|
| **评估治疗联盟的状态** | 联盟足够强可以承受这个干预吗？ |
| **评估治疗阶段** | 早期还是中期？适合深度解释吗？ |
| **评估当前自我功能** | 来访者今天能接受这个吗？ |

### 临床示例

- **Mr A**母亲走失了——表层是对姐姐的愤怒，深层是对母亲的愤怒。治疗师从表层开始。
- **Ms B**担心朋友的人工流产——表面是对朋友的关心，深层是自己对没有孩子的失望。治疗师跟随情感。

> **从表面到深度示例（Mr A）**：55岁女性与年迈母亲同住。会谈材料："妈妈昨天又走丢了。找了好几个小时——她趁我睡觉时离开的。我穿着睡衣在小区里找。可怜的东西。我姐姐因为我打电话给她气坏了——但我需要帮助。我不理解——我在这——没有一点自己的生活——24/7照护妈妈——我打给姐姐求助，她却只关心能不能赶上孩子的足球比赛。有趣的是，她小时候是受宠的那个。妈妈现在几乎不认识我了——但我必须这么做。我怎么能不做呢？我吃得太多了——我得想办法停下来。"→ **表面**：对姐姐的愤怒、对母亲的共情、对身体的蔑视、认命 → **深层**：对母亲的愤怒（曾是偏爱的那个）。

> **跟随情感示例（Ms B）**：39岁单身女性律师。"我很担心我的朋友Jane——她昨天刚做了人工流产。她很痛苦。我去了她家，带了汤。她的男朋友就去工作了——不过我可以在那里陪她。他是个混蛋——男人都这样。她担心她永远不会有孩子了。你认为会吗？做人工流产会让你更难怀孩子吗？我整晚在网上研究这个——没看到类似的说法。我希望这不会让她抑郁。我25岁做人工流产时，根本没想过这些。我很高兴我没有执着于孩子——如果我有了孩子，我绝不可能升合伙人。也许没有孩子就是我该走的路。"→ **表面**：对朋友的关心、职业抱负 → **深层**：对自己从未有孩子的失望、对自己有孩子的渴望。

## 第 18 章：Learning to Intervene

### 章节主线

- 干预分为三大类：基础干预（所有治疗都使用）、支持性干预（供给型 + 协助型）、揭蔽性干预（面质 → 澄清 → 解释）。
- 干预也可以是非语言的（沉默、姿态、语调）。
- 成功指标：支持性干预成功 → 焦虑降低、功能改善；揭蔽性干预成功 → 进一步自由联想、情感深化。

### 基础干预（Basic Interventions）

用于所有心理动力学治疗，无论取向：
- 指示和心理教育（Directions and psychoeducation）
- 提问（Questions）——封闭式和开放式
- 提供信息（Information）
- 共情性评论（Empathic remarks）
- 邀请自由联想（Calls for associations）
- 沉默（Silence）

### 支持性干预（Supporting Interventions）

#### 供给型（Supplying）——提供来访者自己无法提供的东西

| 簇 | 具体技术 |
|----|---------|
| **鼓励簇（Encouraging）** | 鼓励、启发、给予乐观/希望、赞赏、提醒能力 |
| **命名簇（Naming）** | 命名情绪、将体验转化为语言 |
| **重定向簇（Redirecting）** | 向上解释、重定向、支持性绕过 |
| **强化与劝阻（Reinforcing/Discouraging）** | 强化适应性行为、劝阻适应不良行为 |
| **安抚簇（Soothing）** | 安抚、养育、保证、减轻内疚、保持平静、共情、展示兴趣、明确站在一起 |
| **保护簇（Protecting）** | 保护、设定界限 |
| **建议簇（Advising）** | 建议、提议、指导、提供意见 |
| **结构化簇（Structuring）** | 放慢、结构化、组织、分解任务 |
| **提供视角（Supplying perspective）** | 纠正错误感知、重新框架、普遍化、确认 |
| **治疗外的实际支持** | 协助联系资源等 |

#### 协助型（Assisting）——帮助来访者使用自己弱化的自我功能

| 技术 | 内容 |
|------|------|
| **示范（Modeling）** | 治疗师示范适应性行为 |
| **指导（Instructing）** | 教学式指导 |
| **协作（Collaborating）** | 共同目标设定、共同探究、共同探索替代方案、共同现实检验、共同思考后果、共同问题解决、共同组织/结构化、共同完成项目 |

### 揭蔽性干预（Uncovering Interventions）

| 技术 | 定义 | 示例 |
|------|------|------|
| **面质（Confrontation）** | 唤起对某一现象的注意 | "我注意到你刚才停止了说话。" |
| **澄清（Clarification）** | 连接相似现象 | "你总是在周一停止说话。" |
| **解释（Interpretation）** | 使用"因为"图式解释无意识含义 | "你难以在周一说话，是因为你在保护自己免受再次敞开的痛苦。" |
| **发生学解释（Genetic interpretation）** | 将当前模式连接到早期经历 | "也许这和你小时候周一回到学校时感到的孤独有关。" |
| **构建个人叙事（Constructing a personal narrative）** | 帮助来访者建立连贯的自我故事 | |

### 干预成功指标

- **支持性干预成功**：焦虑降低、功能改善、来访者平静下来
- **揭蔽性干预成功**：进一步自由联想、情感深化、来访者提供更多关联材料

### Agent 使用建议

- **适用场景**：这是心理动力学治疗的核心技术框架文件。当 agent 需要选择具体干预技术（从基础到支持到揭蔽）、或在听/反思后决定说什么/是否说时使用。
- **可直接引用**：三种倾听模式及听什么的检查清单、三个选择原则 + 三个准备原则、干预技术树（基础/支持/揭蔽三级分类及每项技术的具体定义）。
- **需要结合**：`_19_24.md`中的情感、阻抗、移情等技术章节——这些章节展示了如何在具体治疗情境中应用听/反思/干预框架；`_03_06.md`中的功能领域评估——决定使用支持还是揭蔽干预。
- **版本注意**：本章内容在第一版和第二版间基本一致。源引用已更新为第二版。
- **注意**：面质、澄清、解释是一个连续加深的过程。不要跳过面质和澄清直接进入解释。支持性和揭蔽性干预不是互斥的——许多会谈同时使用两者。

> **综合案例：支持性干预实践（24岁大四学生，惊恐发作）**：来访者几周前因"突如其来"的惊恐发作就诊。父母在他3岁时离婚，由母亲抚养至10岁后被送入寄养，原因"她再也受不了我了。我乱发脾气。而且她需要去巡回演出她的歌唱事业"。他说幸运的是寄养父母很有爱心、认识到他的智力并帮他拿到大学奖学金。他的主要困扰是与交往一年的女友之间的紧张升级。此前有一系列动荡关系。目前清醒，但当这些关系不可避免地结束时，他会陷入狂饮和可卡因滥用。本次会谈中，他因不知给女友买什么生日礼物而非常焦虑——"我就是不知道该给她买什么！她说随便。但随便到底意味着什么？每次我问她，她都只说'我也不知道'。"
>
> 治疗师：（笑）那确实会让事情更简单。（更认真地）我猜不确定什么会让她开心是令人沮丧的，尤其是当她把答案告诉你似乎很容易的时候。来访者：没错！
> 治疗师：如果她讨厌这个礼物，最坏会发生什么？来访者：有时我觉得我必须每件事都做对，否则她就会立刻走人。
> 治疗师：你觉得这真的可能吗？来访者：嗯，不，不太可能。但她会一副生闷气的样子。
> 治疗师：那确实会令人失望，但也不是世界末日。如果发生这种情况，也许你可以提出带她去购物，让她挑一个更喜欢的。来访者：（怀疑地）我不知道。也许吧。
> 治疗师：（默默地注意到来访者似乎也觉得自己没从女友那里得到他需要的）不如你多告诉我一些关于她的事，也许我们可以一起想些点子。
>
> **分析**：治疗师使用了信息提供、共情、保证、共同问题解决（协助型）、支持性绕过（她注意到他与女友的模式可能与他童年被母亲抛弃的经验相呼应，但判断为时过早、联盟尚脆弱、来访者自我功能薄弱）。
