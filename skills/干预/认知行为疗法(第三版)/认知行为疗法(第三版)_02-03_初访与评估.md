# 认知行为疗法(第三版)_02-03_初访与评估

来源：`skills/干预/认知行为疗法(第三版)/book/认知行为疗法(第三版).md`
抽取范围：第2-3章，**Initial Interactions with Clients** 和 **The Assessment Process**。
用途：指导 agent 理解 CBT 治疗师如何与来访者进行初次接触和系统评估，包括实际互动技巧和评估工具的使用。

## 第2章：与来访者的初次互动

### 章节主线

- 初次联系的场景因工作环境而异，但核心目标是一致的：建立融洽关系、收集基本信息和开始个案概念化。
- 从初次电话到等候室见面，再到治疗室的"处理业务"（知情同意、保密），每一步都是建立治疗关系的关键时刻。
- 即使只进行有限的互动，也应开始收集对个案概念化的有用信息。

### 初次联系

**基本原则**
- 来访者迈出求助的第一步可能令人望而生畏，治疗师应保持敏感。
- 初次电话不是进行彻底评估的时机，而是收集基本信息以确定下一步计划。

**提出建议**
- 邀请来访者进行更深入的评估。
- 将来访者转介给其他专业人员（需小心避免来访者感到被拒绝）。
- 提供信息而非治疗（如自助手册、支持小组）。

**安排第一次来访**
- 告知会面的目的和内容。
- 确保来访者清楚时间、持续时长和到达方式。
- 获得发送提醒函的许可（注意保密，使用无标记信封）。

**议定费用**
- APA 伦理准则要求 "在可行的情况下尽可能早" 讨论费用。
- 介绍未付费和未赴约的政策，用通俗语言清晰解释。

### 会见来访者之前

- **做好准备**：备好表格（知情同意书、HIPAA 表格）、评估量表、笔、手表/时钟。提前到达，整理办公室，清除其他来访者的信息。
- **手机静音**：治疗师和来访者都应将手机静音。来访者对手机要求的反应可以为个案概念化提供信息。
- **穿戴专业**：
  - 女性：避免低胸、短裙、紧身衣物、突出妆容、叮当作响的首饰。
  - 男性：有领衬衫、齐整裤子（非牛仔裤）、正式鞋。
- **注意焦点**：注意力集中在来访者身上，而非自己的表现。自我关注会错过来访者正在说的内容。

### 在等候室中

- 只称呼名而不说出姓，以保护隐私。
- 介绍自己，欢迎来访者，陪伴其到办公室。

### 与来访者在治疗室中

**社交礼仪与自我介绍**
- 开场使用社交礼仪，建立和谐关系。
- 询问来访者希望如何被称呼。
- 介绍自己的全名和称呼方式。
- 告知作为实习治疗师的身份和督导师信息（APA 伦理准则 10.01c 要求）。

**征询录音/录像许可**
- 解释记录会谈的原因、谁可以看到记录、保密措施。
- 多数来访者不会拒绝。

**简单介绍会谈框架**
- 说明会谈将先"处理业务"（获得许可、讨论保密），接着进行评估。

**处理业务**
1. **获得知情同意（Informed Consent）**：
   - 来访者必须理解获得许可的目的和意义。
   - 告知治疗的潜在获益和风险（不能保证每位来访者都能变好）。
   - 告知其他选择（其他疗法、药物治疗、不作为）及不作为的可能结果。
   - 说明来访者可随时退出，不会受歧视。
   - 包括保密及保密限制的讨论。
   - 建议使用书面同意。

2. **讨论保密（Confidentiality）**：
   - 向来访者保证保密对治疗师和诊所人员的重要性（伦理和法律要求）。
   - 说明在治疗室外相遇时的应对原则——治疗师不应主动接触来访者。
   - 保密例外（必须告知来访者）：(1) 来访者威胁伤害自己；(2) 来访者威胁伤害他人；(3) 怀疑虐待儿童或老人；(4) 法庭传唤记录。
   - 为第三方工作（雇主要求评估、法庭评估）时，保密规则不同。

### 个案概念化（迄今为止）

- 即使只通过有限的互动（来访者对费用的反应、对保密的关注、初次的举止），也能收集到对个案概念化有用的信息。
- 异常强烈的反应可能提示来访者对特定话题敏感，应在评估中深入探究。

### 案例：珍妮的初次电话

- 珍妮·S. 被初级保健医生转介来治疗抑郁，但她不确定是否需要治疗。
- 治疗师以合作性经验主义的方式回应，不强求，提供多种选择。
- 珍妮预约了 2 小时的评估会谈。

## 第3章：评估过程

### 章节主线

- 评估过程对应于旅行比喻中"启程阶段"，任务是熟悉地形——了解来访者生命中所发生的事情。
- 评估不仅是做出诊断，更是用 CBT 术语对症状给出初步解释（个案概念化），用于制订治疗方案。
- 本章涵盖评估目标、评估工具选择和评估过程中的常见挑战。

### 留意你的反应

- 治疗师对来访者信息的反应极大地影响治疗关系。
- **不当反应**：说来访者的经历是怪异的、罕见的或令人不悦；消极的面部表情；间隔太久的反应。
- **恰当反应**：点头、说"我明白""我理解"，把来访者的困难正常化（"我在这儿见过许多有类似困惑的来访者"）。
- 如果来访者感觉被评价（如珍妮的例子），应直接澄清，并示范 CBT 中的"读心术"概念。
- 治疗师表露适度情绪（如眼眶含泪）是传递共情，是没问题的。
- **案例——囤积癖来访者**：囤积物从腐烂食物到身体排泄物，无奇不有。即使资深治疗师也可能感到震惊。当来访者询问是否见过类似案例时，即使没见过也可以说："我没有遇到过一模一样的情况，但你和其他有囤积习惯的来访者有许多相似之处——囤积可能是因为考虑以后会用到，或担心丢掉有意义的东西会导致失去自己的一部分。不论囤积的是什么，治疗方式是相同的。"这样既承认了独特性，又通过与其他案例的关联让来访者感到被理解和有希望。

### 评估过程的目标

1. **做出诊断**：确定保险责任范围、与其他专业人士交流、选择合适的治疗方案。
2. **用 CBT 术语做出初步个案概念化**：理解来访者，而不仅仅是命名症状。

### 根据评估目标选择工具

#### 半结构化临床会谈（Semistructured Clinical Interviews）
- 常用工具：SCID-5-CV（DSM-5 结构化临床会谈）、ADIS-5（DSM-5 焦虑及相关障碍会谈）。
- **新手常见问题**：担心会谈显得生硬、没有人情味。
- **解决**：随着熟练度增加，会越来越关注来访者讲话的内容而非按部就班地问问题。关键在于提出后续问题（follow-up questions）。
- 保持温暖和共情的态度能将刻板的会谈转化为积极体验。

#### 非结构化临床会谈（Unstructured Clinical Interviews）
- 不使用固定的诊断问题，但治疗师使用标准主题大纲。
- **主要主题**：
  - 人口统计学信息 → 目前的问题 → 其他问题 → 家庭背景 → 个人生活史。
- **人口统计学信息**：良好的起点，问题不具有威胁性，帮助建立融洽关系。包括性取向、宗教习俗（不应回避这些问题）。
- **目前的问题**：关注起始、经过阶段、频率、情境、自动思维、情绪/行为/生理反应、逃离和回避行为、过去的治疗经历。
- **其他问题**：筛查情绪、焦虑、进食、物质使用、睡眠、躯体担忧、性/性别认同等。
- **家庭背景**：社会经济地位、父母职业、家族精神病史、家庭关系动力——提供核心信念的线索。

#### 精神状况检查（Mental Status Examination, MSE）
- 对来访者在外表、言语、行动和思维方面的描述。
- 常用工具：MMSE（简易精神状态检查表）、MoCA（蒙特利尔认知评估量表）。

#### 自陈问卷（Self-Report Questionnaires）
- 不能作为唯一工具，但可辅助评估。
- 可用于在评估前让来访者填写，帮助引出不愿口头分享的信息。
- 治疗师应创建"备忘单"记录计分规则和分值意义。
- 注意会谈和问卷之间的差异——需要跟进来访者澄清。

#### 观察来访者在评估过程中的行为
- 来访者是否轻声、矜持、勉强？是否会生气？是否有"禁区"话题？
- 行为线索（如拒绝握手、散发酒气、穿着暴露）都是饶有深意的信息。

#### 自我监控（Self-Monitoring）
- 记录靶行为的发生日期、时间、情境、想法和情绪反应。
- 例：抓挠皮肤行为的来访者通过监控发现持续时间与母亲是否在家、情绪状态有关。
  - **案例——抓挠皮肤行为的自我监控**：来访者双腿遍布条状斑痕，知道触发因素是"进入卫生间看到双腿"，但不清楚为何有时很快结束有时持续数小时。治疗师布置一周的自我监控表格。结果发现：(1) 妈妈在家尤其是急着用卫生间时，行为很快结束；(2) 夜晚妈妈睡着后可独占卫生间，行为持续数小时；(3) 疲乏或伤心时行为频率增加；(4) 需要体会到"完整感"——在一个部位不停挠直到觉得"光滑无物"，不同部位达到"完整感"的时间不同。此外还意外发现她也在拔腿部汗毛——一个她从未意识到的行为，对治疗计划很重要。

#### 向其他专业人士请教
- 必须先获得来访者的允许。
- 新手治疗师常对此感到焦虑，但不去请教才被认为是缺乏能力。
- 可以转介的专家：医生、营养师、职业咨询师等。
- **案例——咨询前治疗师（OCD 拒绝暴露）**：来访者在以擅长 OCD 的暴露与反应阻止治疗（EX/RP）而闻名的诊所治疗过，但未成功。来访者说"从来没做过暴露"。联系前治疗师后发现：来访者完全拒绝暴露疗法，尽管已清晰解释其重要性。新治疗师坦诚与来访者探讨："是否愿意考虑做一些暴露？"并提议逐步进行、与先前成功来访者聊聊。来访者回应绝不会考虑暴露。最终转介给精神科医生考虑药物，并告诉她改变主意后可随时回来。通过与先前治疗师的沟通，避免了重复负面体验，同时为将来敞开了大门。

#### 与来访者生活中的其他人交流
- 必须先征得来访者的明确同意，不能"调查来访者"。
- 使用目标清晰的问题，选择能帮助个案概念化的信息。

### 以初步的问题清单做结束

- 珀森斯（Persons, 1989, 2008）建议创建问题清单作为个案概念化的出发点。
- 问题清单是"包含来访者所有困扰的清单"，不仅包括诊断问题，也包括失业、夫妻冲突、医疗状况等。

### 评估过程中常见的挑战

1. **"如果我暂停整理思路，来访者会认为我不知道该做什么"**
   - 来访者大多认为停顿是治疗师在思考。短暂停顿没问题；长停顿可以告诉来访者需要稍微休息一下。

2. **"如果我遗漏了重要信息，评估可能会出现错误"**
   - 遗漏信息后可以打电话给来访者澄清，来访者通常将此解读为关心。
   - 但过于频繁地"回溯"需要跟督导师讨论。
   - 记住：个案概念化是一个持续进行的过程，不是一次完成的。

3. **"我会把评估搞砸，整个个案将成为灾难"**
   - 所有治疗师都会犯错。督导师就是来纠错的。
   - 错误提供重要的学习机会——下次遇到相似的临床表现时就有经验了。

### 珍妮的评估会谈

- **人口统计学信息**：45岁已婚女性，两个女儿，全职妈妈，双亲酗酒且离异。
- **当前问题**：抑郁症状、社交焦虑、酒精使用、身份认同问题（"我还剩什么用？"）。
- **ADIS-5 评估结果**：符合重性抑郁障碍、社交焦虑障碍和中度酒精使用障碍的诊断标准。
- **行为观察**：否认倾向（需跟进询问才会承认）、防御性（对酒精问题）。
- **问题清单**：抑郁、社交焦虑、酒精使用、身份认同问题、家庭关系问题。
- 珍妮最初不认为喝酒是一个问题，治疗师温和地坚持将其放在清单上。

- **适用场景**：当 agent 需要指导如何与来访者进行初次联系、获取知情同意、进行系统性评估时使用。
- **可直接引用**：初次电话沟通要点、知情同意的关键要素、保密原则及例外、半结构化与非结构化会谈的框架、问题清单创建方法。
- **需要结合**：`认知行为疗法(第三版)_00-01_总论与CBT基础.md` 中的 CBT 认知模型和总论，以及 `认知行为疗法(第三版)_04-05_个案概念化与治疗计划.md` 中的个案概念化方法。
