# 认知行为疗法(第三版)_index

来源：`skills/干预/认知行为疗法(第三版)/book/认知行为疗法(第三版).md`

原书：Deborah Roth Ledley, Brian P. Marx, Richard G. Heimberg 著，王建平 等译，*认知行为疗法：新手治疗师实践指南*（原著第三版），中国轻工业出版社，2022。

用途：供 agent 查询 CBT 临床实践全流程——从治疗准备、初访评估、个案概念化、治疗计划、治疗实施到结束治疗，以及特殊人群工作和督导过程。

说明：本书以"旅行"比喻贯穿全程，以珍妮的完整案例串起 CBT 工作全过程。各章节文件按原书顺序组织，提炼核心概念、操作步骤、临床技巧及案例。

## 文件导览

### 认知行为疗法(第三版)_00_01_总论与CBT基础.md

覆盖导言 + 第 1 章。

**适合查询**：新手治疗师 11 个共同挑战；CBT 认知模型（核心信念→中间信念→自动思维→反应）；行为方法（经典条件作用、操作性条件作用）与认知方法；旅行比喻；治疗准备策略（管理治疗师自动思维 + 采取有益行动）。

## Agent 使用建议

- **帮助新手治疗师准备 CBT**：优先读 `00_01` 和 `02_03`。
- **做个案概念化**：优先读 `04_05` 和 `06_07`。
- **处理治疗困难**：优先读 `06_07` 和 `08_09`。
- **理解督导**：优先读 `10_11`。
- **快速获取全书核心理念**：从 `10_11` 第十一章入手。
- **查找推荐读物、手册和专业资源**：读 `附录_资源与推荐`。
- **注意**：珍妮案例贯穿全书多个文件，跨文件追踪可理解 CBT 完整工作流程。
