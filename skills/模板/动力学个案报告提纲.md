# 动力学个案报告提纲

## 1. 初始访谈

- 转诊背景：咨询的先决条件
- 患者的痛苦与主诉（以患者原话记录）及寻求分析师的原因（显性症状）
- 观察首次接触时患者的情景和非语言行为（如肢体语言、情绪表达）
- 治疗师的自发印象（反移情初步体验）
- 患者情绪状态、精神状态及描述痛苦的方式（如情感基调、认知模式）
- 初步诊断与心理动力学假设（移情/反移情分析）
- 分析师接受患者进行精神分析的原因

## 2. 个人成长史

- 患者发展史与家庭背景
- 职业发展（教育、工作现状）、父母职业/婚姻状况/年龄、兄弟姐妹信息
- 童年/青少年期创伤（疾病、事故、分离、照料者变动等）
- 婚育状况与当前社会心理环境
- 患者如何评价自身的成长发展阶段（如对关键事件的认知与情感）
- 既往治疗史（时间、方式、时长、药物使用）

## 3. 患者-分析师关系

- 治疗初期移情/反移情表现及其演变
- 治疗过程中的关键转折点（如阻抗突破、症状缓解）
- 具体对话记录与分析
- 患者话题焦点、治疗师的理解与解释
- 分析师干预方式（如解释技术）及患者反应（需在会谈后及时记录）

## 4. 诊断与心理动力学分析

- 核心无意识冲突（如依恋模式、内在客体关系）
- 创伤或创伤原因（需区分单次事件与长期累积）
- 症状形成的动力学解释（如压抑、投射等防御机制的作用）
- 无意识妥协机制（症状作为冲突的"折中表达"）
- 有哪些防御机制（如否认、合理化、分裂）
- 治疗中动力学假设是否有变化（如对病因的重新建构）

## 5. 治疗设置

- 时间安排（每周会谈频率、预计疗程）
- 费用与缺席处理（如请假、迟到政策）
- 假期协调（提前告知与替代方案）
