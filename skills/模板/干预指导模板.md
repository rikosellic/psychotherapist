# 干预指导模板

基于 `skills/干预/` 中 12 本干预手册的共性结构提炼。

---

## 1. 问题锚定

- 本干预针对个案概念化中的哪个/哪些维持机制
- 问题行为或模式的操作性定义（目标行为是什么、在什么情境下发生、频率/强度/持续时间）
- 来访者当前对此问题的认识程度与改变动机

## 2. 目标设定

- 所欲未来（来访者希望替代问题的具体画面）
- 具体可衡量目标
- 分阶段目标：短期（1–4 次）/ 中期（5–16 次）/ 长期（16 次以上）

## 3. 资源与优势激活

- 例外资源与应对能力（已存在但未被注意的成功经验）
- 保护因素与社会支持
- 来访者优势与治疗可利用的资产

## 4. 阶段化干预路线

### 短期工作重点（1–4 次）

安全 → 联盟 → 评估 → 稳定化 → 症状缓解 → 治疗框架建立

- 预计次数：
- 阶段目标：
- 完成标志：

### 中期工作重点（5–16 次）

核心维持机制工作 → 技能训练或经验加工 → 关系模式修通

- 预计次数：
- 阶段目标：
- 完成标志：

### 长期工作重点（16 次以上）

人格或关系模式改变 → 复发预防 → 意义整合 → 结案过渡

- 预计次数：
- 阶段目标：
- 完成标志：

## 5. 技术选择与理由

每项推荐技术说明：

- 技术名称与来源（书籍 + 章节）
- 针对哪个维持机制或阶段目标
- 适用性理由（为何适合此来访者当前表现）
- 使用注意事项与禁忌

## 6. 关系与系统维度

- 治疗关系注意事项（联盟风险、移情/反移情预警、阻抗形态）
- 系统视角（来访者所处环境中可动员的支持资源、关系网络）
- 与其他治疗提供者的协调要点

## 7. 监测与调适

- 进展指标（可观察的行为变化、主观评分、量表）
- 阶段性复查节点
- 无进展或恶化的应对预案
- 结案标准与维持计划
