## 第 1 章：什么是心理动力学个案概念化（What is a Psychodynamic Formulation?）

### 章节主线

个案概念化（formulation）不等于报告（reporting）——它是对来访者为何如此思考、感受和行为的**假设**。心理动力学个案概念化的独特之处在于它关注**无意识（unconscious）想法和感受的影响及其发展过程**。它不是一个静态的诊断标签，而是随着治疗深入不断修正的"工作性假设"（working psychodynamic formulation）。一个好的 formulation 同时考虑先天（遗传、气质）和后天（环境、关系）。

### 核心概念

**个案概念化（Case Formulation）的定义**
- Formulation 是**假设（hypothesis）**，而非事实陈述——回答"为什么"来访者会有某种思考、感受和行为模式
- 不同病因指向不同治疗——因此形成假设对于推荐和实施治疗至关重要
- 与"报告"（reporting）的关键区别：报告说"发生了什么"，概念化说"为什么发生"

**心理动力学参考框架（Psychodynamic Frame of Reference）**
- 区别于 CBT formulation、精神药理学 formulation、家庭系统 formulation
- 核心假设：问题常由**意识之外（即无意识，unconscious）**的想法和感受引起
- 帮助来访者觉察无意识想法和感受是心理动力学技术的核心

**发展的视角（A Developmental Process）**
- 心理动力学取向关注童年期——因为需要理解无意识想法和感受**如何发展、为何发展**
- 三种利用发展理解的治疗方式：

| 方式 | 核心操作 | 案例 |
|------|----------|------|
| 揭示模式 | 帮助来访者看到"早期条件仍在影响当下" | 童年被母亲遗忘接送 → 成年后害怕被朋友抛弃 |
| 发展缺失能力 | 帮助来访者发展早期未充分形成的能力 | 从未被表扬的来访者 → 学习自尊调节 |
| 支持受损功能 | 支持因急性/慢性问题受损的功能 | 离婚父母 → 帮助意识到恐惧对孩子的影响 |

- 构建 formulation 是**回溯性工作（retrospective）**——先看当前问题，再回溯发展过程

**先天与后天（Nature and Nurture）**
- 不是二者择一，而是二者皆有
- Freud 称先天为"constitutional factors"，后天为"accidental factors"
- 基因与环境相互作用的复杂性：基因塑造经验，反之亦然
- formulation 必须同时考虑遗传、气质和环境因素

**报告 vs 概念化对比示例**

| 报告 | 概念化 |
|------|--------|
| Mr D 早产、母亲青少年产后抑郁、有严重分离焦虑、成人后无法离开妻子过夜 | Mr D 早产、母亲抑郁 → 可能影响安全依恋发展 → 难以将自己视为独立的个体 → 成人后无法与妻子分离 |
| 报告叙述"发生了什么" | 概念化提出"为什么发生"的因果假设，使用"可能"、"或许"等假设性语言 |

**不同类型的心理动力学个案概念化**
- 可基于少量信息（急诊室单次接触）或大量信息（多年精神分析）
- 可解释单一或多个方面的功能
- 可用于任何治疗设置、短期或长期治疗
- 只要考虑无意识想法和感受的影响与发展，就是心理动力学概念化

**工作性概念化（Working Psychodynamic Formulation）**
- 不是一次性完成的固定解释——假设需要不断检验和修正
- DESCRIBE → REVIEW → LINK 的过程在治疗中反复进行
- 随着新信息的出现（新历史、移情反应、梦境），formulation 持续演变
- 最终是一种**持续的临床思维方式**，而非一份书面文件

## 第 2 章：如何使用心理动力学个案概念化（How do We Use Psychodynamic Formulations?）

### 章节主线

Formulation 是治疗的"地图"（map）。它指导每一个治疗环节——从推荐治疗、设定目标、形成策略，到在治疗过程中倾听、构建干预和帮助来访者创建连贯的生命叙事。有时与来访者分享，有时私下使用。

### Formulation 在治疗中的五项功能

| 功能 | 说明 | 示例 |
|------|------|------|
| 1. 制定治疗推荐和设定目标 | 向来访者解释为何推荐某种治疗 | "你对自己的严厉态度可能源于与母亲的关系，探索这些感受可能对你有帮助" |
| 2. 理解发展性需求 | 识别来访者发展的停顿或缺失 | 来访者未发展出足够的自尊调节能力 → 治疗需要聚焦于此 |
| 3. 形成治疗策略并预测移情 | 预测来访者在治疗中会如何反应 | 预测来访者会将治疗师体验为如早期照料者一般批评或忽视 |
| 4. 构建有意义的干预 | 帮助来访者意识到无意识模式 | "你认为我，像你的母亲一样，会对你失望" |
| 5. 创建生命叙事 | 帮助来访者整合为连贯的个人故事 | "我从未意识到母亲对我的忽视如何影响了我对所有人的看法" |

### 是否与来访者分享 Formulation？

| 情境 | 做法 | 示例 |
|------|------|------|
| 来访者有自我反思能力 | 可以分享 formulation 促进洞察 | 与来访者一起构建"我如何成为现在的我"的叙事 |
| 来访者脆弱/缺乏反思能力 | 私下使用 formulation 指导策略 | 帮助来访者发展实际应对策略而不解读无意识动机 |

**关键原则**：分享 formulation 的目的是促进治疗，而非展示临床医生的智慧。详见第 22 章。

## 第 3 章：如何构建心理动力学个案概念化（How do We Construct a Psychodynamic Formulation?）

### 章节主线

构建 formulation 的三个基本步骤——DESCRIBE（描述）、REVIEW（回顾）、LINK（链接）——是所有心理动力学个案概念化的核心方法。每个步骤都是必不可少的，在 Part Two 至 Part Four 中分别详细展开。

### DESCRIBE / REVIEW / LINK 三步法

**第一步：DESCRIBE（描述主要问题与模式）**
- 不只是主诉（chief complaint），而是来访者思考、感受和行为的**主要模式**
- 分为五个基本功能领域：

| 功能领域 | 核心问题 |
|----------|----------|
| 自体（Self） | 如何感知自己？如何调节自尊？ |
| 关系（Relationships） | 如何与他人建立和维持关系？ |
| 适应（Adapting） | 如何应对内部和外部压力？ |
| 认知（Cognition） | 如何思考、决策、反思和判断？ |
| 工作与玩乐（Work and Play） | 工作和休闲模式是否适应当前生活？ |

- 信息来源：来访者**说什么** + 来访者**做什么**（如口头说关系好但评估中不断与治疗师争执）
- 必须同时记录**优势**和**困难**；区分**意识**与**无意识**模式

**第二步：REVIEW（回顾发展史）**
- 不同于"现病史"（history of present illness）或"过去精神病史"（past psychiatric history）
- 发展史关注**塑造来访者**的因素——即其主导功能模式的形成
- 五大原则：
  1. **包含先天与后天**——遗传 + 环境
  2. **关系是关键**——尤其早期关系
  3. **创伤至关重要**——必须询问并理解其意义
  4. **时序相关**——越早的问题越弥散（global），越晚的问题越局限（circumscribed）
  5. **发展是终身的**——不仅是早期里程碑
- 涵盖从出生前（遗传、产前）到当下的所有发展阶段

**第三步：LINK（使用组织性发展理念链接问题与历史）**
- 将问题/模式与发展史链接，形成纵向叙事
- 六种**组织性发展理念（organizing ideas about development）**：

| 理念 | 核心关注 | 章节 |
|------|----------|------|
| 创伤（Trauma） | 创伤如何影响功能 | Ch13 |
| 早期认知和情绪困难（Early Cognitive and Emotional Difficulties） | 先天/早期困难如何干扰发展 | Ch14 |
| 冲突与防御（Conflict and Defense） | 无意识冲突如何形成妥协 | Ch15 |
| 与他人的关系（Relationships with Others） | 内化的关系模板 | Ch16 |
| 自体的发展（The Development of the Self） | 自体和自尊的发展 | Ch17 |
| 依恋（Attachment） | 关系中的安全/焦虑模式 | Ch18 |

- 不同理念适用于理解不同的问题和模式
- 可以组合多种理念
- 链接方式直接指导治疗策略
