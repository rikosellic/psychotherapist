## Part Two 导言：DESCRIBE 的核心原则

### 章节主线

在解释之前必须先描述。描述的对象包括"问题"（Problem，来访者当前最大的困扰）与"人"（Person，来访者的特征性思维、感受和行为模式）。本书采用**维度化方法**（而非分类法），使用五个基本功能领域：自体、关系、适应、认知、工作与玩乐。对每个领域，都要识别优势（strengths）与困难（difficulties），并区分意识层面与无意识层面的模式。

### 核心概念

- **问题 vs 人**：同样的问题（如离婚）在不同的人身上反应截然不同，因为其人格模式不同
- **维度化 vs 分类化**：Hippocrates 的体液分类、Freud 的心性发展阶段固着、DSM 的症状清单都是分类法；本书使用维度描述法更贴近实际功能
- **优势与困难**：人是复杂的镶嵌体（mosaics），在同一领域内可能既有优势也有困难，不同领域之间也可能强弱不等
- **意识与无意识**：有些人对自己的模式有意识（如"我一直有自尊问题"），有些则通过行为间接表达无意识困难（如通过贬低他人来维护自尊）

## 第 4 章：自体（Self）

### 章节主线

自体体验（self-experience）是心理功能的核心。描述自体模式使用两大变量：自我感知（self-perception，包含身份认同和自体幻想）与自尊调节（self-esteem regulation，包含对威胁的脆弱性、内部反应和利用他人进行调节）。

### 变量与描述框架

**自我感知（Self-Perception）**

| 子变量 | 定义 | 临床指示 |
|--------|------|----------|
| 身份认同（Identity） | 知道自己是谁、喜欢什么/不喜欢什么、才能和限制 | 安全的身份认同使人能做出稳定选择；不安全的身份认同导致漂移和决策困难 |
| 自体幻想（Fantasies about the Self） | 关于自我的意识和无意识幻想（如成为被赞美的学生、诺贝尔奖得主） | 与现实能力匹配的幻想有益；不匹配的幻想损害自尊。无意识幻想可通过梦境和行为推断 |

**自尊调节（Self-Esteem Regulation）**

| 子变量 | 定义 | 临床表现 |
|--------|------|----------|
| 对自尊威胁的脆弱性（Vulnerability） | 对外界伤害的敏感程度 | 脆弱型（fragile self-esteem）：微小的批评/拒绝就全面崩溃 vs 坚韧型：重大打击仍能维持 |
| 内部反应（Internal Response） | 遭受威胁时使用的机制 | **较少适应性**：夸大（grandiosity）/自我贬低（self-deprecation）/ 自虐（masochism）；**中等适应性**：竞争性增强/回避竞争；**更适应性**：幽默/升华（sublimation）/利他（altruism） |
| 利用他人调节自尊（Use of Others） | 是否需要他人的持续赞赏来维持自尊 | 过度利用：不断要求关注和认可、缺乏共情；适应良好：能内化赞赏、自我认可 |

### 关键区分

- **嫉妒（Envy）vs 羡慕（Jealousy）**：Envy 是想要摧毁别人拥有的；Jealousy 是想与别人平起平坐
- **自恋型（Narcissistic）人格**：依赖夸大来保护自尊，外化失败、贬低他人、易怒

### 临床评估问题

- 自我感知："你认为自己对优势和困难的感知准确吗？别人会怎么描述你是一个'知道自己是谁'的人？"
- 自尊脆弱性："当你处在一群看起来更有钱/更有成就/教育程度更高的人中间时，你感觉如何？"
- 内部反应："你觉得自己倾向于认为周围人无能吗？别人会说你是一个有竞争性的人吗？"
- 利用他人："你做得好时会自己知道而不需要别人称赞吗？你能在没有他人意见的情况下做决定吗？"

## 第 5 章：关系（Relationships）

### 章节主线

关系模式是理解一个人功能的核心。描述关系使用五个变量：信任（trust）、自我与他人的感知（sense of self and other）、安全感（security）、亲密（intimacy）、互惠性（mutuality）。关系的变异性和不均一性很重要——同一人可能在不同类型的关系中表现不同。

### 变量与描述框架

**信任（Trust）**
- 是建立有意义、相互满足关系的基础
- 缺乏信任导致持续的恐惧、被忽视感和孤独感
- 形成于最早期，取决于气质和早期关系
- 过度信任也可能有问题

**自我与他人的感知（Sense of Self and Other）**
- 三维（three-dimensional）感知：能看到自己和他人的好坏两面、独立且独特的心智、前后一致的感受
- **分裂（Splitting）**：将人视为全好或全坏——正常于幼儿，持续至成人期则高度不适应
- **心智化（Mentalization）**：理解他人有与自己不同的想法和感受。参见第 7 章和第 10 章

**安全感（Security）**
- 关系中感到安全——即使有分离、分歧和负面感受，关系仍能持续
- 在发展中也称安全依恋（secure attachment，参见第 10、18 章）
- 安全型：能耐受矛盾情感、有多种持久关系、缓慢建立关系
- 不安全型：快速进入/退出关系、关系中充满焦虑

**亲密（Intimacy）**
- 分享私人想法、感受、愿望和失望
- 不等同于性——有性行为不等于真正的亲密
- 极端：要么过度分享太快，要么过度保留从不敞开

**互惠性（Mutuality）**
- 双向的给予和接受
- 缺乏互惠的两种形态：付出过多（缺乏边界）vs 索取过多（缺乏共情 empathy）

### 临床评估问题

- 信任："这个世界上有你真正信任的人吗？你一般觉得人们会关照你吗？"
- 自我与他人感知："告诉我一个对你重要的人。他/她是什么样的？他/她总是那样吗？"
- 安全感："你独处时感觉如何？你倾向于慢慢还是快速建立关系？"
- 亲密："你能舒适地告诉朋友你不那么自豪的事情吗？有你可以倾诉几乎一切的人吗？"
- 互惠性："你觉得你的伴侣/朋友给你你需要的吗？他/她得到了他/她需要的吗？"

## 第 6 章：适应（Adapting）

### 章节主线

适应（adapting）指调节内部和外部刺激的能力。每个人有其独特的刺激容忍阈值和应对模式。使用四个变量描述：防御机制（defense mechanisms）、情绪管理（managing emotions）、冲动控制（impulse control）、感觉调节（sensory regulation）。

### 变量与描述框架

**防御机制（Defense Mechanisms）**

防御是无意识地应对压力以维持功能的方式。像平衡感一样自动且持续运作。按适应性从低到高分类：

| 层级 | 防御机制 | 简要定义 |
|------|----------|----------|
| **较少适应性** | 分裂（Splitting） | 将好与坏的感受分离，以保护好的部分 |
| | 投射（Projection） | 将不可接受的品质或感受感知为来自外部 |
| | 投射性认同（Projective Identification） | 将感受投射到他人身上，然后与其互动使之体验该感受 |
| | 病态理想化与贬低（Pathological Idealization and Devaluation） | 对他人赋予过度正面或负面的情感 |
| | 否认（Denial） | 否认不可接受的感受和想法 |
| | 解离（Dissociation） | 将不可接受的想法和感受与当前现实断开 |
| | 见诸行动（Acting Out） | 通过行动表达不可接受的想法或感受 |
| | 退行（Regression） | 使用早期发展阶段的应对策略 |
| **中等适应性** | 情感隔离（Isolation of Affect） | 忘记感受但保持对相关想法的意识 |
| | 理智化（Intellectualization） | 用过度思考替代痛苦感受 |
| | 合理化（Rationalization） | 逻辑化解释不可接受的行为或感受 |
| | 置换（Displacement） | 将感受或冲动转向他人或活动 |
| | 躯体化（Somatization） | 将不适感体验为身体症状 |
| | 抵消（Undoing） | 通过相反行为"修复"不可接受的想法 |
| | 反向形成（Reaction Formation） | 将不可接受的感受转化为其对立面 |
| | 压抑（Repression） | 将不适想法和感受逐出意识 |
| **最适应** | 幽默（Humor） | 以笑话表达不适的想法或感受 |
| | 利他（Altruism） | 将痛苦感受转化为帮助他人 |
| | 升华（Sublimation） | 将不可接受的冲动转化为有益/社会接受的形式 |
| | 抑制（Suppression） | 有意识地决定不关注困难的感受 |

- **防御灵活性**：能否根据情境使用不同防御，而不是僵化使用同一策略
- **防御与思维/感受的关联**：有些防御隔离感受（如理性化→强迫型防御风格，obsessive style）；有些隔离想法（如躯体化→癔症型防御风格，hysterical style）

**情绪管理（Managing Emotions）**
- 能够体验、容忍和表达**广泛的情感**
- 情绪失控vs情绪调节良好
- 注意：管理困难可能来自先天低容忍度，也可能来自慢性焦虑/心境症状

**冲动控制（Impulse Control）**
- 涉及：食欲管理（物质、食物、性）、赌博、攻击/暴力控制、偷窃
- 过度控制也可能成问题——无法享受自发性
- 要同时描述控制策略（如12步项目、宗教信仰）和困难

**感觉调节（Sensory Regulation）**
- 对噪音、气味、质地等感觉刺激的耐受和调节
- 极端感觉敏感可能严重损害功能（如因嗅觉敏感无法乘坐公共交通）

### 临床评估问题

- "你对自己如何应对焦虑和强烈感受有什么了解？你觉得你总是用同一种方式应对吗？"
- "别人会描述你为一个冒险者还是一个谨慎者？你发现很难克制自己不做某些你认为不该做的事吗？"
- "当你生气或焦虑时会发生什么？你能'坐住'那种感受吗？"

## 第 7 章：认知（Cognition）

### 章节主线

思维的方式反映在几乎所有行为中。描述认知功能使用五个变量：一般认知能力、决策与问题解决、自我反思/现实检验、心智化、判断力。认知功能的发展变化可以为发展中出现问题的时间提供线索。

### 变量与描述框架

**一般认知能力（General Cognitive Abilities）**
- 智力（intelligence）、记忆（memory）、注意（attention）、言语/语言（speech and language）
- 正式评估有时需要标准化测试（如MMSE、神经心理学测试）
- 尤其要注意认知功能是明显优势还是障碍——这将影响所有其他领域

**决策与问题解决（Decision Making and Problem Solving）**
- 决策风格差异很大：细节导向vs印象主义；研究导向vs直觉导向；计划者vs随机应变者
- 包含：组织思维、计划能力、创造性思维（creative thinking）
- 这些能力对自尊和人际关系有重大影响

**自我反思（Self-Reflection）与心理-mindedness**
- 自我反思（self-reflection）：审视自己的思想和行为的能力
- 心理-mindedness（psychological mindedness）：思考自己行为背后可能的无意识动机
- 通过**试探性解释（trial interpretation）**评估："你想过你可能因为不想去才睡过头吗？"
- **现实检验（Reality Testing）**：区分现实与幻想的能力

**心智化（Mentalization）**
- 理解他人有与自己不同的想法和感受的能力
- 是共情的基础
- 评估：问来访者"你认为对方可能有不同的看法吗？"

**判断力（Judgment）**
- 考虑行为后果的能力——不仅知道对错，还据此行动
- 在心理动力学中属于超我功能（superego function）
- 判断力可能在不同情境下波动（如喝酒时判断力下降）
- 关键区分：判断力差（不知对错）vs冲动控制差（知道对错但控制不住）

### 临床评估问题

- "你觉得自己是一个容易还是难做决定的人？你通常会研究选择还是'相信直觉'？"
- "你觉得自己通常是准时的人还是经常迟到？"
- "你倾向于遵守规则吗？你觉得认识你的人会说你判断力好吗？"

## 第 8 章：工作与玩乐（Work and Play）

### 章节主线

Freud 说人需要能"爱与工作"，后人加上"玩乐"。描述工作和玩乐模式，不是为了评判其"应该做什么"，而是评估这些活动是否适合其个人生活和社会环境。使用四个变量：与发展水平的匹配度、满意度/舒适度、对自我和依赖者的照顾是否充分、文化适应性。

### 变量与描述框架

**工作类型多样性**
- 有偿/无偿、持续/间断、家庭内/外、有技能/无技能
- 学生的"工作"是上学；全职父母的"工作"是照顾孩子

**玩乐的关键维度**
- 占用多少时间、独自还是与他人、投入的深度和广度
- 性作为玩乐的重要方面
- 玩乐缺失（absence of play）是重要临床信号

**评估变量**

| 变量 | 评估要点 |
|------|----------|
| 与发展阶段/才能/限制匹配 | 是否与训练水平一致？是否与发展阶段匹配？（如：博士做入门级工作=不匹配） |
| 舒适与满意度 | 是否享受工作/玩乐？是满足还是必须？是否变成强迫性行为？ |
| 对自我和依赖者的照顾 | 经济是否维持基本生活？若依赖他人是否双方同意并满意？是否能定期锻炼？ |
| 文化适应性（Culturally Sanctioned） | 是否合法？在文化背景下是否合理？需要考虑跨文化差异 |

### 临床评估问题

- 工作："你做这项工作多久了？你觉得有乐趣吗？还是只是为了谋生？"
- 玩乐："你做什么来放松？有让你期待去做的事吗？你玩得开心吗？"
- 自我照顾："你能维持生计吗？对家人呢？你接受他人的经济资助吗？对这个感觉如何？"

## Putting it Together：完整 DESCRIBE 示例（Mr A）

**个案简介**：64岁同性恋男性，母亲4个月前去世后感到"漫无目的"和孤独。与伴侣B同居15年，关系稳定但各自生活较独立。即将面临退休。

**DESCRIBE 摘要**：
- **自体**：身份认同未完全建立（退休后不知所措）；对自尊威胁较脆弱（嫉妒伴侣）；内部反应偏向自我贬低
- **关系**：15年关系是优势；但近期出现信任困难（莫名担忧被抛弃）；理想化伴侣（影响三维感知）；亲密能力中等；互惠性偏单向（付出多于接受）
- **适应**：一般适应良好；使用合理化和理想化防御；在压力下防御灵活性下降；无冲动控制问题
- **认知**：主要为优势领域；一般认知功能良好；自我反思能力一般（能理解与丧母的关联，但无法反思嫉妒感受）；心智化可能有一些困难
- **工作/玩乐**：工作匹配才能、有满足感；玩乐能力有限（没有工作外的兴趣）
