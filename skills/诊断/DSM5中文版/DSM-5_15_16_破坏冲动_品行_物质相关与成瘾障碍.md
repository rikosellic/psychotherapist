# DSM-5_15_16_破坏冲动_品行_物质相关与成瘾障碍

来源：`skills/DSM5中文版/book/DSM5中文版.md`
抽取范围：破坏性、冲动控制及品行障碍（第453-471页）、物质相关及成瘾障碍（第473-582页），**Disruptive, Impulse-Control, and Conduct Disorders, Substance-Related and Addictive Disorders**。
用途：供 agent 在个案概念化中查询冲动控制问题和物质使用障碍的诊断标准。

## 破坏性、冲动控制及品行障碍

### 对立违抗障碍（ODD, F91.3）
- 愤怒/易激惹心境、好争辩/反抗行为或报复心，持续≥6个月，与≥4项症状
- 患病率约3.3%（儿童）；**严重程度**：轻度（仅1种场景）、中度（2种场景）、重度（3种场景）

### 间歇性暴怒障碍（IED, F63.81）
- 行为上失控（言语/躯体攻击），平均每周≥2次，持续3个月；或3个月内≥3次造成财物损坏/人身伤害的爆发
- 冲动性/攻击性是核心特征

### 品行障碍（Conduct Disorder, CD）
- 侵犯他人基本权利或违反与年龄相符的主要社会规范，持续≥12个月
- 四组症状中≥3项：攻击人或动物、毁坏财物、欺骗或偷窃、严重违反规定
- **亚型**：儿童期发生型（10岁前）、青少年期发生型（10岁后）、未特定发生型
- **标注**：伴有限的亲社会情感；患病率约2%-10%

### 纵火狂（Pyromania, F63.1）
- 反复故意纵火，伴紧张/情感唤起；纵火非出于金钱、意识形态、报复或妄想

### 偷窃狂（Kleptomania, F63.2）
- 反复无法抵抗偷窃冲动，偷取非个人所用或无价值的物品

## 物质相关及成瘾障碍

### 物质使用障碍的通用诊断标准（12个月内出现≥2项）
1. 使用量/时间超过预期
2. 持续渴望或无法减少使用
3. 大量时间用于获取/使用/恢复
4. 渴求（强烈欲望）
5. 因使用未履行工作/学业/家庭责任
6. 尽管造成社交/人际问题仍继续使用
7. 因使用放弃/减少重要活动
8. 在危险情况下使用
9. 明知问题仍继续使用
10. 耐受（需要明显增加剂量或效果明显下降）
11. 戒断（典型戒断综合征或使用该物质以缓解/避免戒断）

**严重程度**：轻度（2-3项）、中度（4-5项）、重度（≥6项）

### 各类物质的中毒与戒断特征

| 物质 | 中毒表现 | 戒断表现 |
|-----|---------|---------|
| **酒精** | 言语不清、协调障碍、步态不稳、眼球震颤、昏迷 | 自主神经活动亢进、手抖、失眠、恶心呕吐、幻觉、焦虑、惊厥发作（DTs） |
| **咖啡因** | 坐立不安、神经过敏、兴奋、失眠、心悸 | 头痛、疲劳、情绪低落、注意困难 |
| **大麻** | 损害运动协调、时间感变慢、判断受损、社交退缩 | 易激惹、愤怒、焦虑、头痛、食欲下降、睡眠困难 |
| **阿片类** | 瞳孔缩小（或针尖样）、嗜睡/昏迷、言语不清 | 烦躁心境、流涕流泪、肌肉疼痛、腹泻、发热、失眠 |
| **兴奋剂**（可卡因/苯丙胺） | 欣快或情感迟钝、警觉性增高、心动过速、瞳孔散大 | 疲劳、失眠或嗜睡、食欲增加、精神运动迟滞、抑郁心境 |
| **烟草** | N/A | 易激惹、焦虑、注意困难、食欲增加、坐立不安、抑郁心境 |
| **镇静剂/催眠药/抗焦虑药** | 言语不清、协调障碍、步态不稳、昏迷 | 自主神经活动亢进、手抖、失眠、焦虑、惊厥发作 |

### 非物质相关障碍：赌博障碍（Gambling Disorder, F63.0）
- 持续和反复的有问题的赌博行为，12个月内出现≥4项症状
- **严重程度**：轻度（4-5项）、中度（6-7项）、重度（8-9项）

## Agent 使用建议

- **适用场景**：当 agent 需要查询冲动控制障碍、物质使用障碍或赌博障碍的诊断标准时使用。
- **可直接引用**：物质使用障碍的11项通用诊断标准及严重程度分级、各类物质的戒断综合征特征表。
- **需要结合**：`DSM-5_17_22_神经认知_人格_性欲倒错与其他临床状况.md`（反社会型人格障碍、物质所致的 NCD）；`DSM-5_02_04_精神分裂症谱系_双相_抑郁障碍.md`（物质/药物所致的精神病性/双相/抑郁障碍）。
- **注意**：DSM-5 将"药物滥用"和"药物依赖"合并为"物质使用障碍"，"依赖"不再等同于"成瘾"——耐受和戒断是处方药的正常生理反应。
