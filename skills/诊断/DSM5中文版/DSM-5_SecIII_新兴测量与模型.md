# DSM-5_SecIII_新兴测量与模型

来源：`skills/DSM5中文版/book/DSM5中文版.md`
抽取范围：第三部分（第721-790页），**Emerging Measures and Models**。
用途：供 agent 了解 DSM-5 第三部分的评估量表、文化概念化工具、人格障碍替代模式和需要进一步研究的状况。

## 评估量表

### 跨领域症状量表（Cross-Cutting Symptom Measures）
- 覆盖13个心理领域，用于所有诊断组中测量不同严重程度的症状水平
- 可在线获取（www.psychiatry.org/dsm5）

### WHO 残疾评估量表（WHODAS 2.0）
- 基于 ICF（国际功能、残疾和健康分类）
- 取代 GAF（功能大体评定量表），提供精神障碍的整体残疾水平的标准评估
- 涵盖6个领域：认知、活动能力、自我照顾、与人相处、生活活动、社会参与

### 精神病性症状严重程度维度临床工作者评估
（Clinician-Rated Dimensions of Psychosis Symptom Severity）
- 评估幻觉、妄想、言语紊乱、异常精神运动行为、阴性症状、抑郁和躁狂的维度

## 文化概念化

### 文化概念化访谈（Cultural Formulation Interview, CFI）
- 核心访谈工具，包含4个领域：
  1. 文化身份定义
  2. 疾病的文化概念化
  3. 心理社会应激源和文化特征的支持因素
  4. 文化因素对治疗寻求和治疗关系的影响

### 补充工具
- CFI-知情者版本（CFI-Informant Version）
- 补充模块（如移民和身份、精神评估等）

## 人格障碍的 DSM-5 替代模式

### 混合型模式（保留在第三部分供研究使用）
- 替代传统的10种人格障碍的分类诊断
- 包含两个核心要素：
  1. **人格功能损害**（LPFS, Level of Personality Functioning Scale）：自我功能（身份、自我导向）和人际功能（共情、亲密关系）的损害
  2. **病态人格特质**（Pathological Personality Traits）：5个领域25个特质面
     - 负性情感、疏离、对抗、脱抑制、精神病性

### 纳入的6种特定人格障碍
- 反社会型、回避型、边缘型、自恋型、强迫型、分裂型
- 加上"特定特质型人格障碍"（PD-Trait Specified, PD-TS）

## 需要进一步研究的状况

这些状况的科学证据尚不能支持其广泛的临床使用，但被纳入以刺激进一步研究：

- **衰减的精神病综合征**（Attenuated Psychosis Syndrome）
- **抑郁发作伴短暂轻躁狂**（Depressive Episodes with Short-Duration Hypomania）
- **持续性复杂丧痛障碍**（Persistent Complex Bereavement Disorder）
- **咖啡因使用障碍**（Caffeine Use Disorder）
- **非自杀性自伤**（Non-Suicidal Self-Injury, NSSI）
- **自杀行为障碍**（Suicidal Behavior Disorder）
- **网络游戏障碍**（Internet Gaming Disorder）

## Agent 使用建议

- **适用场景**：当 agent 需要了解 DSM-5 的评估工具、文化概念化方法、人格障碍的前沿研究模式，或查询尚在研究阶段的诊断提案时使用。
- **可直接引用**：CFI 的4个核心领域、人格障碍替代模型的核心构成（LPFS + 特质维度）、WHODAS 的6个领域。
- **需要结合**：`DSM-5_SecI_基础与使用指南.md`（文化概念化的三种概念框架）；`DSM-5_17_22_神经认知_人格_性欲倒错与其他临床状况.md`（人格障碍的类别诊断标准）。
- **注意**：第三部分的内容尚在研究阶段，不应作为临床诊断的正式依据。替代人格模型可能在未来取代或补充现有的人格障碍分类。
